# 页面设置说明

页面设置用于调整网站显示、上传页默认选项、背景图和管理端显示效果。

## 全局设置

| 选项 | 功能 |
| --- | --- |
| 网站标题 | 浏览器标签页显示的标题。 |
| 网站图标 | 浏览器标签页的小图标地址。 |
| 图床名称 | 前台页面显示的图床名称。 |
| 图床Logo | 前台页面显示的 Logo 图片地址。 |
| Logo跳转链接 | 点击 Logo 或头像后打开的链接。 |
| 背景切换间隔 | 多张背景轮播的切换时间，单位为毫秒；`60000` 表示 60 秒。 |
| 背景图透明度 | 背景图显示透明度，范围 `0` 到 `1`；数值越小越淡。 |
| 默认URL前缀 | 生成图片链接时使用的前缀；留空时使用当前站点域名。 |

## 客户端设置

| 选项 | 功能 |
| --- | --- |
| 公告 | 上传页顶部公告内容，支持填写 HTML。 |
| 默认上传渠道 | 上传页默认选中的上传渠道，也可以选择 Smart Dispatch 自动调度。 |
| 默认上传目录 | 上传页默认目录；例如 `/user/`，留空或 `/` 表示根目录。 |
| 默认命名方式 | 上传后文件名的默认生成方式，见下方说明。 |
| 默认转换WebP | 上传前将图片转换为 WebP。 |
| 默认开启压缩 | 上传前在浏览器本地压缩图片。 |
| 默认压缩阈值 | 图片超过该大小时自动压缩，单位 MB。 |
| 默认期望大小 | 压缩后希望接近的文件大小，单位 MB。 |
| 登录页背景图 | 用户登录页背景图。 |
| 上传页背景图 | 上传页背景图。 |
| 页脚传送门链接 | 页脚传送门按钮打开的链接。 |
| 隐藏页脚 | 开启后隐藏前台页脚。 |

## 管理端设置

| 选项 | 功能 |
| --- | --- |
| 管理端登录页背景图 | 管理端登录页背景图。 |
| 管理端背景图 | 管理端后台页面背景图；可填单张图片 URL，也可填多个 URL。 |
| 图像加载模式 | 管理端图片列表的预览加载方式，可选择原图或智能加载；智能加载会优先给公开图片使用缩略图，受限图片使用原图。 |
| 缩略图图源 | 选择缩略图由哪个服务生成，可选 wsrv.nl、Cloudflare 图片转换或 WordPress Photon；选择 Cloudflare 图片转换时，需要先在 Cloudflare 开启图片转换功能。 |
| Live2D 看板娘 | 开启后在管理端显示 Live2D 看板娘。 |
| 烟花点击特效 | 开启后点击页面时显示烟花效果。 |
| 星星拖尾特效 | 开启后鼠标移动时显示星星拖尾效果。 |

## 背景图填写方式

登录页背景图、上传页背景图、管理端登录页背景图支持以下写法：

| 写法 | 效果 |
| --- | --- |
| `bing` | 使用必应壁纸轮播。 |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | 多张图片轮播。 |
| `["https://example.com/1.jpg"]` | 单张背景图。 |
| `["https://your-domain.com/random?..."]` | 随机图接口链接；也可以先在“其他设置”里配置自己的随机图 API，再把调用演示的随机图链接按单张背景图格式填到这里。 |

管理端背景图支持填写图片 URL；多个 URL 可按页面提示用英文逗号分隔。留空时使用默认背景。

## 默认命名方式

| 方式 | 生成效果 |
| --- | --- |
| 默认 | 时间随机前缀 + 原文件名，例如 `1760000000000_cat.png`。 |
| 仅前缀 | 只使用时间随机前缀和扩展名，例如 `1760000000000.png`。 |
| 仅原名 | 保留原文件名，例如 `cat.png`；重名时自动加 `(1)`、`(2)`。 |
| 短链接 | 使用 8 位短 ID 和扩展名，例如 `a1b2c3d4.png`。 |
