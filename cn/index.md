# ImgBed

基于 Cloudflare 生态的全能个人内容托管工具。

从图片上传与链接管理起步，ImgBed 现已成长为一套完整的个人内容托管解决方案。你可以用它：

- 上传文件到任意你偏好的存储渠道
- 公开分享图片与任意格式文件
- 一分钟快速搭建个人博客
- 通过磁力链接一键转存文件
- 加入 ImgBed 节点联盟，共享开放文件索引

我们的初衷，是希望每个人都能拥有属于自己的图片托管和文件分享站点。无论你来自哪个国家，使用哪种语言；无论你只是想简单保存图片、整理相册、分享文件给朋友，还是想为博客、创作内容或公共资源搭建一个稳定的托管空间，ImgBed 都希望尽量降低门槛，让这件事变得更自由、更清晰、更容易开始。

## 开箱即用的强大能力

- 🌍 支持 28 种界面语言，全球用户无障碍使用
- 📦 内置 13 种主流上传渠道：Telegram、Cloudflare R2、S3、WebDAV、Discord、Hugging Face、GitHub Releases、GitLab Packages、OneDrive、Google Drive、Dropbox、Yandex、pCloud
- 🧩 纯模块化设计，所有功能按需开启，不浪费资源
- 🔐 支持 API Token、访问模式和名单规则，便于精细化管理内容权限

这个项目仍在持续完善。我们能力有限，但会尽全力把每一个功能都做清楚、做稳定、做好用。如果 ImgBed 能帮你更轻松地拥有自己的图床、博客或文件分享空间，我们会非常开心。
