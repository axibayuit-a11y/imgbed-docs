# ImgBed 用户文档

ImgBed 是一个基于 Cloudflare 生态构建的图床与文件分享项目。它最初从图片上传和链接管理开始，现在已经扩展为一个更完整的个人内容托管工具：你可以把文件上传到多个存储渠道，公开展示图片，搭建自己的博客，通过磁力链接转存文件，也可以和其他 ImgBed 节点组成联盟，分享彼此开放的文件索引。

我们希望 ImgBed 不只是一个“上传图片拿链接”的工具，而是让每个人都能更容易拥有自己的图床、博客和文件分享方式。无论你只是想保存图片、管理相册、写一点文章，还是想搭建一个轻量的公开资源页，它都尽量给你保留选择空间。

目前 ImgBed 支持 28 种本地化语言，并提供 Telegram、Cloudflare R2、S3、WebDAV、Discord、Hugging Face、GitHub Releases、GitLab Packages、OneDrive、Google Drive、Dropbox、Yandex、pCloud 等多种上传渠道。你可以按自己的地区、账号和使用习惯选择合适的存储方式。

这个项目仍在持续完善。我们的能力有限，但会尽力把每个功能做得清楚、稳定、好用。如果 ImgBed 能帮你更轻松地拥有自己的图床、博客或文件分享空间，我们会非常开心。
