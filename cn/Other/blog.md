# 博客使用说明

博客功能可以给你的图床站点增加一个独立博客页面。

开启后，访客可以通过：

```text
https://你的域名/blog/
```

访问博客。

![博客首页](../../image/other/博客/博客首页.png)

博客页面基于 [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki) 开源博客网站改造而来。感谢原项目的开源，我们使用 Vue 重写并适配到当前图床项目里。

## 入口位置

博客设置在：

系统配置 -> 其他设置 -> 博客

![博客设置](../../image/other/博客/QQ20260611-221702.png)

## 第一次使用

1. 打开“启用”开关。
2. 选择用于保存博客配置的 GitHub 账号。
3. 点击“更新博客”。
4. 等系统提示博客更新成功。
5. 访问 `https://你的域名/blog/` 查看博客。

如果是第一次使用，系统会在你选择的 GitHub 账号下准备一个私有仓库：

```text
imgbed-blog-config
```

这个仓库用于保存博客配置和文章内容。

## 怎么写文章

博客文章在你自己的 GitHub 私有仓库里修改：

```text
imgbed-blog-config
```

常见做法是：

1. 打开自己的 GitHub。
2. 进入 `imgbed-blog-config` 私有仓库。
3. 修改或新增文章文件。
4. 保存提交。
5. 回到图床后台，点击“更新博客”刷新博客内容或者到博客首页左上角logo点击三次触发更新博客。

“更新博客”不会覆盖你已经写好的内容，它主要用于初始化仓库或刷新博客缓存。

## 支持的功能

博客支持文章列表、分类、标签、归档、搜索、深色模式、多语言切换等常见博客功能。

也支持留言和访问统计。

![博客留言](../../image/other/博客/支持留言.png)

留言会显示在文章页面下方。访客可以填写头像、昵称、邮箱和评论内容后提交。

访问统计会显示文章访问量、站点访问次数等信息，方便你了解博客的浏览情况。

## 访问地址

博客固定在 `/blog/` 路径下。

例如你的图床域名是：

```text
https://image.example.com
```

博客地址就是：

```text
https://image.example.com/blog/
```

关闭博客后，访客将无法访问博客页面。
