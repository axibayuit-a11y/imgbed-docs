# 联盟分布索引使用说明

联盟分布索引用来让多个图床站点互相共享文件列表。

简单理解：

- 你可以把自己的某些文件夹共享给别人。
- 你也可以加入别人的节点，把对方共享的文件列表同步到自己的后台里查看。
- 联盟文件主要用于浏览、搜索和访问，不是在本机重新上传一份文件。

## 入口位置

联盟分布索引在：

系统配置 -> 其他设置 -> 联盟分布索引

![联盟分布索引本地节点](../../image/other/联盟图/联盟分布式索引本地节点.png)

页面分成三个页签：

| 页签 | 用途 |
| --- | --- |
| 本地节点 | 设置自己的联盟开关、对外域名、共享文件夹，并更新索引 |
| 我加入的节点 | 管理自己加入的其他图床节点 |
| 加入我的节点 | 管理别人申请加入你的节点 |

## 第一次使用怎么设置

1. 打开“本地节点”。
2. 打开“启用”开关。
3. 在“同步文件夹”里选择要共享出去的文件夹。
4. 点击“更新出站索引”。
5. 如果系统提示检测到域名变化，确认当前域名没问题后继续。

同步文件夹可以选多个。

如果同步文件夹清空，表示共享全部文件夹。

## 本地节点怎么用

### 对外域名

对外域名是别人访问你这个节点时使用的站点地址。

这个地址由系统自动识别，不需要手动填写。第一次更新索引时，系统会让你确认当前访问地址是不是正式域名。

如果以后换了域名，再点更新索引时也会提示确认。

### 同步文件夹

同步文件夹决定哪些文件会共享给联盟节点。

例如你只选择：

```text
/1/
/2/
```

别人同步你的节点时，就只能看到这两个目录里的文件。

### 更新出站索引

这个按钮用于更新“我共享给别人看的文件列表”。

适合在这些时候点击：

- 第一次启用联盟。
- 新上传了一批想共享的文件。
- 改了同步文件夹。
- 换了对外域名后需要确认新域名。

## 我加入的节点怎么用

“我加入的节点”是你订阅别人的地方。

![我加入的节点](../../image/other/联盟图/我加入的节点.png)

### 申请加入别人

1. 找对方要一条邀请链接。
2. 粘贴到输入框。
3. 点击“申请加入”。
4. 等对方在自己的后台同意。

对方同意后，这条节点会变成“已同意”。

### 更新入站索引

更新入站索引用来把你已加入节点的文件列表同步到本机。

适合在这些时候点击：

- 对方刚同意你的申请后。
- 对方告诉你他更新了共享内容。
- 你想刷新全部已加入节点的联盟文件列表。

如果只想更新某一个节点，可以在对应节点卡片上点“更新索引”。

![更新索引](../../image/other/联盟图/更新索引.png)

### 取消订阅

如果不想继续同步某个节点，可以点击“取消订阅”。

取消后，这个节点的联盟索引也会从本机清理掉。

## 加入我的节点怎么用

“加入我的节点”是别人申请加入你的地方。

![加入我的节点](../../image/other/联盟图/加入我的节点.png)

### 生成邀请链接

1. 先确认本地节点已经启用。
2. 先点过“更新出站索引”，让系统确认过对外域名。
3. 打开“加入我的节点”。
4. 点击“重置邀请链接”。
5. 复制邀请链接发给对方。

如果邀请链接为空，通常是还没有确认对外域名。回到“本地节点”点一次“更新出站索引”即可。

### 处理加入申请

别人提交申请后，会出现在“加入我的节点”列表里。

| 操作 | 作用 |
| --- | --- |
| 同意 | 允许对方同步你共享的文件列表 |
| 拒绝 | 不允许对方加入 |
| 删除 | 删除已经结束的记录 |
| 检查状态 | 检查对方是否还保留这段关系 |

同意之后，对方还需要在自己那边点“更新入站索引”，才能看到你共享的文件列表。

![邀请节点同意](../../image/other/联盟图/邀请节点同意.png)

## 留言功能

关系已经同意后，可以在节点卡片里点“留言”。

留言只用于联盟关系沟通，不会影响文件、标签、目录和访问权限。

![留言功能](../../image/other/联盟图/留言功能.png)

## 在哪里看联盟文件

同步完成后，回到管理后台文件列表。

页面顶部可以在本地文件和联盟文件之间切换。切到联盟文件后，就能浏览已同步的联盟内容。

联盟文件主要用于查看、搜索、预览和复制链接。它不是本机文件，所以不能在本机直接移动、删除、改标签或备份。

![联盟管理显示效果图](../../image/other/联盟图/联盟管理显示效果图.png)

## 常见问题

### 为什么有无关系记录重新申请？

一般是对方把你删除并删除记录，导致查询不到你的这段记录显示无关系记录，需要你重新申请关系


![无关系记录重新申请](../../image/other/联盟图/无关系记录重新申请.png)

### 为什么加入后看不到文件？

先确认这几件事：

1. 对方已经同意你的申请。
2. 对方已经点过“更新出站索引”。
3. 你自己已经点过“更新入站索引”。
4. 对方的同步文件夹里确实包含要共享的目录。

### 提示域名变化怎么办？

如果你现在打开后台用的就是正式域名，可以确认继续。

如果你只是临时用其他地址访问后台，建议取消，换成正式域名重新打开后台再操作。

### 同步文件夹清空是什么意思？

同步文件夹清空表示共享全部文件夹。

如果只想共享一部分目录，就手动选择对应文件夹。

### 更新出站和更新入站有什么区别？

| 按钮 | 简单理解 |
| --- | --- |
| 更新出站索引 | 更新“别人能从我这里同步到什么” |
| 更新入站索引 | 更新“我从别人那里同步到了什么” |
