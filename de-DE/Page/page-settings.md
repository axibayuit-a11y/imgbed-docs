# Seiteneinstellungen

Die Seiteneinstellungen steuern Darstellung, öffentliche Seiten und Upload-Erlebnis in ImgBed.

## Wo einstellen

```text
Systemeinstellungen -> Seiteneinstellungen
```

## Wichtige Optionen

| Option | Beschreibung |
| --- | --- |
| Seitenname | Wird im Browser-Titel und in der Oberfläche verwendet |
| Seitenbeschreibung | Beschreibungstext der Website |
| Logo / Symbol | Bilder für Seite und Browser |
| Hintergrundbild | Hintergrund der Startseite |
| Fußbereich | Copyright, externe Links oder weitere Informationen |
| Upload-Text | Hinweistext im Upload-Bereich |

## Öffentliche Seiten

Wenn du Besucher-Seiten nutzt, prüfe auch öffentliche Ordner, Zugriffsmodus und Bildlademodus.

Besonders bei öffentlicher Galerie oder Zufallsbild-API sollten Seiteneinstellungen und Veröffentlichungsrechte unter Weitere Einstellungen zusammenpassen.

## Nach dem Speichern

1. Speichere die Änderungen.
2. Lade die Startseite neu.
3. Prüfe Desktop- und Mobilansicht.
4. Kontrolliere Logo, Hintergrund, Beschreibung und Links im Fußbereich.

Wenn ein Bild nicht aktualisiert wird, prüfe, ob die URL direkt erreichbar ist und ob der Browser noch alten Cache nutzt.
