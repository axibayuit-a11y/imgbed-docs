# আপলোড সেটিংস

আপলোড সেটিংস ImgBed-কে আপনার নিজস্ব সংরক্ষণ চ্যানেলের সঙ্গে যুক্ত করে। কোনো চ্যানেল সেট করার পর আপলোড করা ছবি ও ফাইল আপনার বেছে নেওয়া পরিষেবায় সংরক্ষিত হয়। ImgBed প্রবেশ লিংক, ফাইল রেকর্ড, প্রিভিউ, পাবলিক গ্যালারি, র‍্যান্ডম ইমেজ API, WebDAV প্রবেশাধিকার এবং সংশ্লিষ্ট কাজগুলো পরিচালনা করে।

ব্যবহারের ধরন অনুযায়ী উপযুক্ত চ্যানেল আলাদা হতে পারে। সহজভাবে শুরু করতে চাইলে Telegram, Discord বা GitHub Releases বিবেচনা করুন। ধারণক্ষমতা, গতি এবং দীর্ঘমেয়াদি স্থিতিশীলতা বেশি গুরুত্বপূর্ণ হলে Cloudflare R2, S3, OneDrive, Google Drive, Dropbox, Yandex, pCloud অথবা আপনার নিজস্ব WebDAV পরিষেবা ব্যবহার করুন।

## শুরু করার আগে

- যে সংরক্ষণ অ্যাকাউন্ট বা API পরিচয়পত্র ব্যবহার করবেন, তা প্রস্তুত রাখুন।
- আপনার ImgBed ডোমেইন ঠিকভাবে কাজ করছে কি না নিশ্চিত করুন, কারণ OAuth চ্যানেলগুলোর জন্য কলব্যাক URL দরকার।
- চ্যানেল যোগ করার পর আগে একটি পরীক্ষামূলক ছবি আপলোড করে নিশ্চিত করুন যে ফাইলটি সংরক্ষিত হচ্ছে এবং ঠিকভাবে খোলা যাচ্ছে।
