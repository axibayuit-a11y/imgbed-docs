# Paramètres de page

Les paramètres de page contrôlent les éléments de présentation, les pages publiques et l’expérience d’upload dans ImgBed.

## Où configurer

```text
Paramètres système -> Paramètres de page
```

## Options principales

| Option | Description |
| --- | --- |
| Nom du site | Utilisé dans le titre du navigateur et l’interface |
| Description du site | Texte de présentation du site |
| Logo / icône | Images utilisées dans la page et le navigateur |
| Image de fond | Fond de la page d’accueil |
| Pied de page | Copyright, liens externes ou autres informations |
| Texte d’upload | Message affiché dans la zone de téléversement |

## Pages publiques

Si vous utilisez des pages visiteurs, vérifiez aussi les dossiers publics, le mode d’accès et le mode de chargement des images.

Avec la galerie publique ou l’API d’image aléatoire, les paramètres de page doivent être cohérents avec les autorisations de publication dans Autres paramètres.

## Après enregistrement

1. Enregistrez les modifications.
2. Rechargez la page d’accueil.
3. Vérifiez l’affichage sur ordinateur et mobile.
4. Assurez-vous que le logo, le fond, la description et les liens du pied de page s’affichent comme prévu.

Si une image ne change pas, vérifiez que son URL s’ouvre directement et que le navigateur n’utilise pas un ancien cache.
