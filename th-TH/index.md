# เอกสารผู้ใช้ ImgBed

ImgBed เป็นโปรเจกต์โฮสต์รูปภาพและแชร์ไฟล์ที่สร้างอยู่บนระบบนิเวศของ Cloudflare เดิมทีเริ่มจากการอัปโหลดรูปและจัดการลิงก์ แต่ตอนนี้ขยายเป็นเครื่องมือสำหรับโฮสต์คอนเทนต์ส่วนตัวที่ครบขึ้น: คุณสามารถอัปโหลดไฟล์ไปยังหลายช่องทาง จัดแสดงแกลเลอรีสาธารณะ ทำบล็อกของตัวเอง โอนไฟล์จากลิงก์ magnet และเชื่อมกับโหนด ImgBed อื่นผ่าน federated file index ได้

เราอยากให้ ImgBed เป็นมากกว่าเครื่องมือที่อัปโหลดรูปแล้วได้ลิงก์กลับมา เป้าหมายคือช่วยให้ทุกคนมีพื้นที่โฮสต์รูป บล็อก และวิธีแชร์ไฟล์ของตัวเองได้ง่ายขึ้น ไม่ว่าคุณต้องการเก็บรูป จัดการอัลบั้ม เขียนบทความ หรือทำหน้ารวมทรัพยากรแบบเบา ๆ ImgBed พยายามเปิดพื้นที่ให้คุณเลือกใช้งานในแบบของคุณ

ปัจจุบัน ImgBed รองรับภาษาแบบ localize 28 ภาษา และมีช่องทางอัปโหลดหลายแบบ เช่น Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex และ pCloud คุณสามารถเลือกพื้นที่จัดเก็บตามภูมิภาค บัญชี และพฤติกรรมการใช้งานของคุณ

โปรเจกต์นี้ยังพัฒนาอย่างต่อเนื่อง ความสามารถของเรามีจำกัด แต่เราจะพยายามทำให้ทุกฟีเจอร์ชัดเจน เสถียร และใช้งานได้ดี หาก ImgBed ช่วยให้คุณมีพื้นที่โฮสต์รูป บล็อก หรือแชร์ไฟล์ของตัวเองได้ เราจะดีใจมาก
