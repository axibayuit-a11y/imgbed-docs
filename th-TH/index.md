# ImgBed

เครื่องมือโฮสต์คอนเทนต์ส่วนตัวแบบครบในตัวเดียว บนระบบนิเวศของ Cloudflare

ImgBed เริ่มต้นจากการอัปโหลดรูปภาพและจัดการลิงก์ ตอนนี้เติบโตเป็นโซลูชันโฮสต์คอนเทนต์ส่วนตัวที่ครบขึ้นแล้ว คุณสามารถใช้มันเพื่อ:

- อัปโหลดไฟล์ไปยังช่องทางจัดเก็บใดก็ได้ที่คุณชอบ
- แชร์รูปภาพและไฟล์ทุกฟอร์แมตแบบสาธารณะ
- สร้างบล็อกส่วนตัวได้อย่างรวดเร็วในหนึ่งนาที
- บันทึกไฟล์จาก magnet links ได้ในคลิกเดียว
- เข้าร่วม federation ของ ImgBed nodes และแชร์ open file indexes

เราไม่เชื่อในโซลูชันแบบ "one-size-fits-all" ไม่ว่าคุณแค่อยากเก็บรูป จัดการอัลบั้มจำนวนมาก เขียนโน้ตสั้น ๆ หรือทำเว็บทรัพยากรสาธารณะแบบเบา ๆ ImgBed เคารพตัวเลือกของคุณ และเปิดพื้นที่ให้ยืดหยุ่นที่สุดเท่าที่ทำได้

## ความสามารถทรงพลังที่พร้อมใช้ทันที

- 🌍 รองรับ 28 ภาษาอินเทอร์เฟซ ให้ผู้ใช้ทั่วโลกใช้งานได้ง่าย
- 📦 มี 13 ช่องทางอัปโหลดหลักในตัว: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud
- 🧩 ออกแบบแบบ modular ล้วน เปิดใช้แต่ละฟีเจอร์เมื่อจำเป็น ไม่เปลืองทรัพยากร
- 🔐 รองรับ API Token, โหมดการเข้าถึง และกฎรายการ เพื่อจัดการสิทธิ์ของคอนเทนต์ได้ละเอียดขึ้น

โปรเจกต์นี้ยังพัฒนาอย่างต่อเนื่อง ความสามารถของเรามีจำกัด แต่เราจะพยายามเต็มที่ให้ทุกฟีเจอร์ชัดเจน เสถียร และใช้งานดี หาก ImgBed ช่วยให้คุณมีพื้นที่โฮสต์รูป บล็อก หรือแชร์ไฟล์ของตัวเองได้ง่ายขึ้น เราจะดีใจมาก
