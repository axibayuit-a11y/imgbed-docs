# சீரற்ற பட API மற்றும் பொது காட்சியகம்

இரு அம்சங்களும் இங்கு அமைக்கப்படுகின்றன:

```text
System Settings -> Other Settings
```

## சீரற்ற பட API

சீரற்ற பட API தேர்ந்தெடுத்த அடைவுகளிலிருந்து ஒரு சீரற்ற கோப்பை திரும்ப அளிக்கும். தள பின்னணிகள், அவதார் சுழற்சி அல்லது வெளிப்புற பக்கங்களில் சீரற்ற படம் அழைப்புகளுக்கு இது பயனுள்ளது.

இதை இயக்கிய பிறகு பயன்படுத்தவும்:

```text
https://your-domain.com/random
```

## சீரற்ற பட API அமைப்புகள்

| விருப்பம் | நோக்கம் |
| --- | --- |
| இயக்கு | `/random` இறுதிப் புள்ளியை இயக்கும் அல்லது அணைக்கும். அணைக்கப்பட்டால் அணுகல் தடை செய்யப்படும். |
| அடைவுகள் | சீரற்ற API பயன்படுத்தக்கூடிய அடைவுகளை வரையறுக்கும். இங்கு சேர்க்கப்படாத அடைவுகளை API பயன்படுத்த முடியாது. |
| அழைப்பு மாதிரி | நேரடியாக நகலெடுக்கக்கூடிய சீரற்ற API இணைப்புகளை உருவாக்கும். |

பல அடைவுகளை தேர்வு செய்யலாம். உதாரணமாக `/landscape/` மற்றும் `/portrait/` மட்டும் அனுமதிக்கப்பட்டால், சீரற்ற API அந்த அடைவுகளிலும் அவற்றின் துணை அடைவுகளிலும் உள்ள கோப்புகளை மட்டுமே தேர்வு செய்யும்.

## சீரற்ற பட API அளவுருக்கள்

| அளவுரு | உதாரணம் | நோக்கம் |
| --- | --- | --- |
| `dir` | `/landscape/` | சீரற்ற அடைவை குறிப்பிடுகிறது. |
| `content` | `image` | ஊடக வகையை குறிப்பிடுகிறது. `image`, `video`, `audio` அல்லது கமாவால் பிரிக்கப்பட்ட சேர்க்கைகளை பயன்படுத்தவும். |
| `orientation` | `auto` | பட திசையை வடிகட்டுகிறது. `portrait`, `landscape` அல்லது `auto` பயன்படுத்தவும். |
| `type` | `url` | திரும்பும் வடிவம். காலியாக இருந்தால் திருப்பிவிடல், `url` எளிய உரை URL தரும், `json` JSON தரும். |
| `origin` | `1` | `type=url` உடன் முழு URL திரும்ப அளிக்க பயன்படுத்தப்படுகிறது. |
| `age` | `all-ages,r12` | வயது மதிப்பீட்டின் அடிப்படையில் வடிகட்டுகிறது. |
| `tag` | `wallpaper,sky` | இந்த குறிச்சொற்கள் உள்ள கோப்புகளை மட்டும் திரும்ப அளிக்கும். |
| `ex` | `private` | இந்த குறிச்சொற்கள் உள்ள கோப்புகளை விலக்கும். |

## திரும்பும் வடிவங்கள்

`type` இல்லாமல், API நேரடியாக சீரற்ற கோப்பு URL-க்கு திருப்பிவிடும்.

`type=url` உடன், அது உரை URL-ஐ திரும்ப அளிக்கும்.

`type=json` உடன், கோப்பு URL, கோப்பு ID, கோப்பு பெயர், கோப்பு வகை, குறிச்சொற்கள், மதிப்பீடு மற்றும் தொடர்புடைய மேட்டாடேட்டா உட்பட கோப்பு தகவலை திரும்ப அளிக்கும்.

## அணுகல் விதிகள்

சீரற்ற பட API பொது அணுகல் விதிகளைப் பின்பற்றும்:

| விதி | விளைவு |
| --- | --- |
| அடைவு கட்டுப்பாடு | அனுமதிக்கப்பட்ட அடைவுகளிலுள்ள கோப்புகள் மட்டுமே தேர்வு செய்யப்படும். |
| தடைப் பட்டியல் | தடைப் பட்டியலில் உள்ள கோப்புகள் சீரற்ற தொகுப்பிலிருந்து விலக்கப்படும். |
| அனுமதி பட்டியல் முறை | இயக்கப்பட்டால், பொது அணுகலுக்கு அனுமதிக்கப்பட்ட கோப்புகள் மட்டுமே திரும்ப அளிக்கப்படும். |
| வயது மதிப்பீடு | R12, R16, R18 மற்றும் இதற்கு ஒத்த உள்ளடக்கம் தற்போதைய அணுகல் முறையின்படி வடிகட்டப்படும். |

வடிகட்டலுக்குப் பிறகு எந்த கோப்பும் பொருந்தவில்லை என்றால், API பொருந்தும் முடிவை திரும்ப அளிக்காது.

## கேஷ்

வேகத்தை மேம்படுத்த, சீரற்ற பட API அடைவு வேட்பாளர் தொகுப்புகளை கேஷ் செய்கிறது.

கோப்புகள் மாறிய பிறகு, ImgBed அடைவு கேஷ் பதிப்பை புதுப்பிக்கும்; பின்னர் வரும் கோரிக்கைகள் வேட்பாளர் தொகுப்பை மீண்டும் உருவாக்கும். மீண்டும் மீண்டும் வினவப்படுவதைத் தவிர்க்க காலியான அடைவுகள் குறுகிய நேரம் கேஷ் செய்யப்படும்.

## பொது காட்சியகம்

பொது காட்சியகம், நீங்கள் பார்வையாளர்களுக்கு பார்க்க அனுமதிக்கும் அடைவுகளுக்காக வாசிப்பு-மட்டும் கொண்ட பொது உலாவல் பக்கத்தை வழங்குகிறது.

இதை இயக்கிய பிறகு, பார்வையாளர்கள் திறக்கலாம்:

```text
https://your-domain.com/browse/directory-name
```

## பொது காட்சியக அமைப்புகள்

| விருப்பம் | நோக்கம் |
| --- | --- |

| பட ஏற்றும் முறை | முன்னோட்டங்கள் மூலப் படங்களையா அல்லது சிறுபடங்களையா பயன்படுத்தும் என்பதை கட்டுப்படுத்தும். |
| திறந்த அடைவுகள் | பார்வையாளர்கள் அணுகக்கூடிய அடைவுகளை அமைக்கும். |

| சிறுபடம் | வேகமான ஏற்றத்திற்காக பார்வையாளர் பக்கம் சிறுபடங்களுக்கு முன்னுரிமை அளிக்கும். |

## திறந்த அடைவுகள்

திறந்த அடைவுகள் பார்வையாளர்கள் என்ன பார்க்க முடியும் என்பதை தீர்மானிக்கும்.

உதாரணம்:

```text
/1/,/2/,/landscape/,/portrait/
```

பின்னர் பார்வையாளர்கள் இவற்றை அணுகலாம்:

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

`/2026/lucky/` போன்ற துணை அடைவுகளையும் திறக்கலாம். திறக்கப்படாத அடைவுகளுக்கு பார்வையாளர்கள் தடுக்கப்படுவார்கள்.
