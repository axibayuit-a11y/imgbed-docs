# பட மிதப்படுத்தலும் அணுகல் முறையும்

பட மிதப்படுத்தல் பதிவேற்றப்பட்ட படங்களுக்கு வயது மதிப்பீடுகளை வழங்கும். அணுகல் முறை, பொதுப் பயன்பாட்டில் எந்த மதிப்பீடுகள் காட்டப்படலாம் என்பதை கட்டுப்படுத்தும்.

இது பொதுக் காட்சியகம், பொதுக் கோப்பு URL-கள் மற்றும் சீரற்ற படம் API ஆகியவற்றை பாதிக்கும். நிர்வாகப் பலகைக்கு இது கட்டுப்பாடாகாது. நிர்வாகிகள் எல்லா கோப்புகளையும் தொடர்ந்து பார்க்கவும் நிர்வகிக்கவும் முடியும்.

## எங்கே அமைப்பது

நிர்வாகப் பலகையைத் திறந்து இங்கே செல்லவும்:

```text
System Settings -> Security Settings -> Upload Management -> Image Moderation
```

முக்கிய அமைப்புகள்:

- அணுகல் முறை
- மிதப்படுத்தலை இயக்கு
- மிதப்படுத்தல் வழங்குநர்

## அணுகல் முறை என்ன செய்கிறது

அணுகல் முறை, எந்த வயது மதிப்பீடுகள் பொதுவாக காட்டப்படலாம் என்பதை தீர்மானிக்கும்.

தற்போதைய முறைகள்:

| அணுகல் முறை | பொதுவாகத் தெரியக்கூடிய மதிப்பீடுகள் |
| --- | --- |
| பெரியவர்கள் முறை | General, R12, R16, R18 |
| இளைஞர் முறை | General, R12, R16 |
| பதின்ம வயது முறை | General, R12 |
| குழந்தை முறை | General மட்டும் |

இயல்புநிலை பெரியவர்கள் முறை.

தனிப்பட்ட தளங்கள் அல்லது முதிர்ந்த உள்ளடக்கம் கொண்ட தளங்களுக்கு பெரியவர்கள் முறை பொருத்தமாக இருக்கலாம். அதிக கட்டுப்பாடு கொண்ட பொதுக் காட்சியகத்திற்காக இளைஞர், பதின்ம வயது அல்லது குழந்தை முறையைத் தேர்ந்தெடுக்கவும்.

## மிதப்படுத்தலை இயக்கினால் என்ன ஆகும்

மிதப்படுத்தல் இயக்கப்பட்டால், பதிவேற்றத்தின் போது ImgBed தேர்ந்தெடுக்கப்பட்ட மிதப்படுத்தல் வழங்குநரை அழைத்து கண்டறியப்பட்ட வயது மதிப்பீட்டைச் சேமிக்கும்.

முக்கிய மதிப்பீடுகள்:

| மதிப்பீடு | பொருள் |
| --- | --- |
| General | பொதுப் பயன்பாட்டுக்கு பாதுகாப்பான உள்ளடக்கம் |
| R12 | லேசான உணர்வுப்பூர்வ உள்ளடக்கம் |
| R16 | அதிகமாக உணர்வுப்பூர்வமான உள்ளடக்கம் |
| R18 | பெரியவர்களுக்கான உள்ளடக்கம் |

பொது அணுகலைத் தீர்மானிக்க மிதப்படுத்தல் முடிவு பயன்படுத்தப்படும்.

மிதப்படுத்தல் இயக்கப்படவில்லை என்றாலோ, பழைய கோப்புகளுக்கு மதிப்பீடு இல்லையெனில், அவை மதிப்பிடப்படாத கோப்புகளாக கருதப்படும். மதிப்பீடு இல்லை என்பதற்காக மட்டும் அவை பொதுக் காட்சியகத்திலிருந்தோ சீரற்ற படம் API-யிலிருந்தோ தானாக நீக்கப்படாது.

## மிதப்படுத்தல் வழங்குநரைத் தேர்வு செய்தல்

கிடைக்கும் வழங்குநர்கள்:

- moderatecontent.com
- nsfwjs
- Sightengine

ஒவ்வொரு வழங்குநருக்கும் தேவைகள் வேறுபடும்:

- moderatecontent.com பொதுவாக API Key தேவைப்படும்.
- nsfwjs பொதுவாக API முனைய URL தேவைப்படும்.
- Sightengine-க்கு API user மற்றும் API secret தேவைப்படும்.

உங்கள் கணக்கு, கிடைப்புத் தன்மை மற்றும் கண்டறிதல் தரத்தை அடிப்படையாகக் கொண்டு தேர்வு செய்யவும். மிதப்படுத்தல் இயக்கப்பட்டு சரியாக அமைக்கப்பட்டிருந்தால், பதிவேற்றத்தின் போது பட மதிப்பீட்டை எழுத ImgBed முயலும்.

## பொதுக் காட்சியகத்தில் ஏற்படும் விளைவு

பொதுக் காட்சியகம் அணுகல் முறையை அடிப்படையாகக் கொண்டு கோப்புகளை வடிகட்டும்.

எடுத்துக்காட்டுகள்:

- பெரியவர்கள் முறை: R18 படங்கள் தோன்றலாம்.
- இளைஞர் முறை: R18 படங்கள் மறைக்கப்படும்.
- பதின்ம வயது முறை: R16 மற்றும் R18 மறைக்கப்படும்.
- குழந்தை முறை: General மதிப்பீடு கொண்ட படங்கள் மட்டும் காட்டப்படும்.

இது வழக்கமான பொதுப் பயன்பாட்டை மட்டுமே பாதிக்கும். நிர்வாகப் பலகை எல்லா கோப்புகளையும் தொடர்ந்து காட்டும்.

## பொதுக் கோப்பு URL-களில் ஏற்படும் விளைவு

பொதுக் கோப்பு URL-கள் என்பது பார்வையாளர்கள் திறக்கும் நேரடி படச் சுட்டிகள்.

கோப்பின் மதிப்பீடு தற்போதைய அணுகல் முறையில் அனுமதிக்கப்பட்டிருந்தால், ImgBed அசல் படத்தைத் திருப்பி வழங்கும்.

மதிப்பீடு அனுமதிக்கப்பட்ட நிலையை விட அதிகமாக இருந்தால், வழக்கமான பொதுப் பயன்பாடு அசல் படத்தைத் திருப்பி வழங்காது. அதற்கு பதிலாக அமைக்கப்பட்ட தடுப்பு முடிவையோ இடம்பிடி படத்தையோ ImgBed வழங்கும்.

எடுத்துக்காட்டு:

- தற்போதைய முறை குழந்தை முறை.
- ஒரு படத்திற்கு R18 மதிப்பீடு உள்ளது.
- பார்வையாளர் பொதுப் URL-ஐ நேரடியாகத் திறக்கிறார்.
- அந்த பார்வையாளருக்கு ImgBed அசல் R18 படத்தை வழங்காது.

![கட்டுப்படுத்தப்பட்ட கோப்பு படம்](../../image/Safety/文件受限图.png)

நிர்வாகப் பலகையில் கோப்புகளைப் பார்க்கும் நிர்வாகிகள் இந்த கட்டுப்பாட்டால் பாதிக்கப்படமாட்டார்கள்.

## சீரற்ற படம் API-யில் ஏற்படும் விளைவு

சீரற்ற படம் API-யும் அணுகல் முறையை அடிப்படையாகக் கொண்டு தனது வேட்பாளர் கோப்புக் குளத்தை வடிகட்டும்.

குழந்தை முறையில், சீரற்ற படங்கள் General மதிப்பீடு கொண்ட கோப்புகளிலிருந்தே தேர்ந்தெடுக்கப்படும்.

இளைஞர் முறையில், சீரற்ற படங்கள் General, R12 மற்றும் R16 கோப்புகளிலிருந்து வரலாம்; ஆனால் R18 கோப்புகளிலிருந்து வராது.

இதனால் சீரற்ற படம் API பொதுக் காட்சியகக் கட்டுப்பாடுகளை மீற முடியாது.

## பட்டியல் விதிகளுடன் உள்ள தொடர்பு

அணுகல் முறை மட்டுமே பொதுப் பயன்பாட்டு விதி அல்ல. அது அனுமதி/தடை பட்டியல் விதிகளுடன் சேர்ந்து செயல்படும்.

எளிமையாகச் சொன்னால்:

- அனுமதி பட்டியலில் உள்ள உள்ளடக்கம் முதலில் பொதுவாக கருதப்படும்.
- தடை பட்டியலில் உள்ள உள்ளடக்கத்தை சாதாரண பார்வையாளர்கள் நேரடியாகப் பார்க்க முடியாது.
- இரு பட்டியலிலும் இல்லாத உள்ளடக்கம் பின்னர் அணுகல் முறையுடன் ஒப்பிடப்படும்.

ஒரு படம் வயது மதிப்பீட்டாலும் பட்டியல் விதிகளாலும் கட்டுப்படுத்தப்பட்டிருந்தால், சாதாரண பார்வையாளர்கள் அசல் கோப்பை இன்னும் நேரடியாகப் பார்க்க முடியாது.

## பரிந்துரைக்கப்பட்ட அமைப்புகள்

பொது தளங்களுக்கு:

- மிதப்படுத்தலை இயக்கவும்.
- தளத்தின் பார்வையாளர்களுக்கு பொருந்தும் அணுகல் முறையைத் தேர்ந்தெடுக்கவும்.
- எல்லா வயதினருக்கும் திறந்த தளங்களுக்கு குழந்தை முறை அல்லது பதின்ம வயது முறையைப் பயன்படுத்தவும்.
- பெரியவர்களுக்கான உள்ளடக்கத்தை பொதுவாகக் காட்ட விரும்பவில்லை என்றால் பெரியவர்கள் முறையைத் தவிர்க்கவும்.
- நிர்வாகப் பலகையில் கோப்பு மதிப்பீடுகளைப் பார்த்து தேவையானபோது கைமுறையாக மாற்றவும்.

தனிப்பட்ட அல்லது சொந்தப் பயன்பாட்டு தளங்களுக்கு:

- பெரியவர்கள் முறை பொதுவாகப் பொருத்தமானது.
- பயனுள்ளதாக இருந்தால் மிதப்படுத்தலை இயக்கவும்.
- தேவைக்கேற்ப நிர்வாகப் பலகையில் மதிப்பீடுகளைப் பார்த்து திருத்தவும்.

### பழைய மதிப்பிடப்படாத படங்களுக்கு என்ன ஆகும்?

மிதப்படுத்தல் முடிவுகள் இல்லாததற்காக மட்டும் மதிப்பிடப்படாத படங்கள் தானாக மறைக்கப்படாது. அவற்றின் மதிப்பீடுகளை பின்னர் நிர்வாகப் பலகையில் மாற்றலாம்.
