# ImgBed 사용자 문서

ImgBed는 Cloudflare 생태계를 기반으로 만든 이미지 호스팅 및 파일 공유 프로젝트입니다. 처음에는 이미지 업로드와 링크 관리에서 출발했지만, 지금은 개인 콘텐츠를 더 넓게 보관하고 공개할 수 있는 도구로 확장되었습니다. 여러 저장소 채널에 파일을 올리고, 공개 갤러리를 운영하고, 나만의 블로그를 만들고, 마그넷 링크로 파일을 전송하고, 다른 ImgBed 노드와 연합해 공개 파일 인덱스를 공유할 수 있습니다.

우리는 ImgBed가 단순히 "이미지를 올리고 링크를 받는 도구"에 머무르지 않기를 바랍니다. 누구나 자기만의 이미지 저장소, 블로그, 파일 공유 공간을 더 쉽게 가질 수 있게 하는 것이 목표입니다. 이미지를 보관하거나, 앨범을 정리하거나, 글을 쓰거나, 가벼운 공개 자료 페이지를 만들 때도 ImgBed가 선택지를 남겨 두는 도구가 되기를 바랍니다.

현재 ImgBed는 28개 현지화 언어를 지원하며 Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud 등 다양한 업로드 채널을 제공합니다. 지역, 계정, 사용 습관에 맞춰 저장 방식을 고를 수 있습니다.

이 프로젝트는 계속 다듬어 가는 중입니다. 우리의 역량은 제한적이지만, 각 기능을 명확하고 안정적이며 쓰기 좋게 만들기 위해 노력하겠습니다. ImgBed가 여러분의 이미지 호스트, 블로그, 파일 공유 공간을 만드는 데 도움이 된다면 정말 기쁩니다.
