# 인증 관리와 로그인 기기 관리

인증 관리에서는 관리자 화면과 사용자 페이지의 로그인 방식, 로그인된 기기, 접근 상태를 확인할 수 있습니다.

## 관리자 로그인

관리자는 관리자 로그인 화면을 통해 ImgBed 설정과 파일 관리로 들어갑니다.

![관리자 로그인](../../image/Safety/管理端登录界面.png)

관리자 계정은 강한 권한을 가지므로 비밀번호와 인증 정보는 안전하게 관리해야 합니다.

## 사용자 로그인

사용자용 로그인 화면도 제공됩니다.

![사용자 로그인](../../image/Safety/用户端登录界面.png)

공개 업로드나 제한 공개 기능을 운영한다면 사용자 로그인을 함께 사용해 누가 작업했는지 추적하기 쉬워집니다.

## 인증 관리 화면

인증 관리 화면에서는 로그인 상태와 기기 정보를 확인할 수 있습니다.

![인증 관리](../../image/Safety/认证管理界面.png)

## 로그인 기기에서 확인할 항목

| 항목 | 설명 |
| --- | --- |
| 로그인 기기 | 현재 로그인된 기기 확인 |
| IP / 위치 | 어디에서 로그인했는지 참고 |
| 마지막 접근 | 최근에 사용된 기기인지 확인 |
| 해제 | 불필요하거나 의심스러운 로그인 세션 제거 |

## 의심스러운 로그인을 발견하면

1. 해당 기기의 로그인을 해제합니다.
2. 관리자 비밀번호와 관련 토큰을 변경합니다.
3. Cloudflare, GitHub, 저장 채널의 인증 정보도 확인합니다.
4. 필요하면 API Token을 재발급합니다.

## 운영 팁

- 공용 PC에서는 로그인 상태를 유지하지 마세요.
- 관리자 권한 계정은 최소한으로 유지합니다.
- 로그인 기기 목록을 주기적으로 확인합니다.
- 오래된 세션이나 쓰지 않는 기기는 삭제합니다.
