# WebDAV 설정과 접근

ImgBed는 WebDAV를 통해 파일에 접근할 수 있습니다. PC의 파일 탐색기나 WebDAV 클라이언트에서 연결하면 저장된 파일을 드라이브처럼 확인할 수 있습니다.

## 사용 사례

- Windows나 macOS에서 파일을 직접 확인하고 싶을 때.
- WebDAV 지원 클라이언트로 파일을 정리하고 싶을 때.
- 관리자 화면 외부에서 저장된 파일에 접근해야 할 때.

## Windows 11에서 연결하기

1. 파일 탐색기를 엽니다.
2. `내 PC`를 오른쪽 클릭합니다.
3. `네트워크 위치 추가`를 선택합니다.
4. WebDAV URL을 입력합니다.
5. 사용자 이름과 비밀번호를 입력합니다.
6. 연결 후 폴더처럼 열리는지 확인합니다.

![Windows 11 WebDAV 설정](../../image/Safety/webdav在win11配置.png)

연결에 성공하면 탐색기에서 WebDAV 내용을 볼 수 있습니다.

![Windows 표시](../../image/Safety/webdav在win显示效果.png)

## 인증 정보

WebDAV 사용자 이름과 비밀번호는 ImgBed에 설정된 인증 정보를 사용합니다. 공개용 계정보다는 필요한 범위의 권한만 가진 계정을 쓰는 편이 안전합니다.

## 연결이 안 될 때

| 확인 항목 | 내용 |
| --- | --- |
| URL | `https://`를 포함한 올바른 WebDAV 주소인지 |
| 인증 정보 | 사용자 이름과 비밀번호가 맞는지 |
| 권한 | 대상 디렉터리에 읽기/쓰기 권한이 있는지 |
| 클라이언트 | Windows 또는 클라이언트가 WebDAV 접속을 제한하지 않는지 |

용량 조회만 실패하고 파일 업로드는 되는 경우도 있습니다. 이때는 WebDAV 서버가 용량 정보를 제공하지 않는 것일 수 있습니다.
