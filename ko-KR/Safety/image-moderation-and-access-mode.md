# 이미지 심사와 접근 모드

이미지 심사와 접근 모드는 공개 범위와 연령 등급에 따라 방문자가 볼 수 있는 파일을 제어하는 기능입니다.

공개 갤러리, 랜덤 이미지 API, 외부 파일 접근과 함께 사용됩니다.

## 할 수 있는 일

| 기능 | 설명 |
| --- | --- |
| 이미지 심사 | 파일 내용이나 등급 정보를 기록 |
| 블랙리스트 | 공개하지 않을 파일 제외 |
| 화이트리스트 모드 | 허용한 파일만 공개 |
| 연령 등급 제어 | R12, R16, R18 등의 표시 여부 조정 |
| 접근 모드 | 방문자가 볼 수 있는 범위를 한 번에 전환 |

## 접근 모드

접근 모드를 사용하면 공개 페이지와 랜덤 API가 반환하는 내용을 제한할 수 있습니다.

예를 들어 전체 이용가 중심으로 운영하고 싶다면 일반 파일만 반환되도록 설정합니다. 내부용이나 제한 공개 사이트라면 운영 목적에 맞게 범위를 넓힐 수 있습니다.

## 파일이 제한되는 경우

제한 대상 파일은 방문자에게 원래 내용 대신 접근할 수 없다는 안내가 표시됩니다.

## 운영 방식 선택

| 방식 | 적합한 상황 |
| --- | --- |
| 블랙리스트 중심 | 기본은 공개하고 문제 파일만 제외 |
| 화이트리스트 중심 | 허용한 파일만 확실히 공개 |
| 연령 등급 중심 | 콘텐츠 등급에 따라 공개 범위 조정 |

공개 사이트라면 화이트리스트나 연령 등급을 함께 쓰는 편이 의도치 않은 공개를 줄이기 좋습니다.

## 확인할 점

1. 공개 갤러리에서 대상 파일이 보이는지 확인합니다.
2. 랜덤 이미지 API가 제한 대상 파일을 반환하지 않는지 확인합니다.
3. 블랙리스트나 화이트리스트를 바꾼 뒤 캐시 영향이 없는지 확인합니다.
4. 자동 태그의 등급 결과를 사용할 때는 오판 가능성을 고려해 중요한 파일은 직접 확인합니다.
