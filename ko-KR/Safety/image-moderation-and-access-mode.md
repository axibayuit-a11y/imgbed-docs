# 이미지 모더레이션 및 액세스 모드

이미지 모더레이션은 업로드된 이미지에 연령 등급을 부여합니다. 액세스 모드는 공개 액세스에서 어떤 등급을 볼 수 있는지 제어합니다.

이는 공개 갤러리, 공개 파일 URL, 랜덤 이미지 API에 영향을 줍니다. 관리자 패널은 제한하지 않습니다. 관리자는 계속 모든 파일을 보고 관리할 수 있습니다.

## 설정 위치

관리자 패널을 열고 다음 위치로 이동합니다.

```text
System Settings -> Security Settings -> Upload Management -> Image Moderation
```

주요 설정:

- 액세스 모드
- 모더레이션 활성화
- 모더레이션 제공자

## 액세스 모드의 역할

액세스 모드는 공개적으로 표시할 수 있는 연령 등급을 결정합니다.

현재 모드:

| 액세스 모드 | 공개 표시 가능한 등급 |
| --- | --- |
| 성인 모드 | General, R12, R16, R18 |
| 청소년 모드 | General, R12, R16 |
| 10대 모드 | General, R12 |
| 아동 모드 | General만 |

기본값은 성인 모드입니다.

개인 사이트나 성인 콘텐츠가 있는 사이트에는 성인 모드가 적합할 수 있습니다. 더 보수적인 공개 갤러리가 필요하다면 청소년, 10대 또는 아동 모드를 선택하세요.

## 모더레이션을 활성화하면 일어나는 일

모더레이션이 활성화되면 ImgBed는 업로드 시 선택된 모더레이션 제공자를 호출하고 감지된 연령 등급을 저장합니다.

주요 등급:

| 등급 | 의미 |
| --- | --- |
| General | 안전한 공개 콘텐츠 |
| R12 | 약간 민감한 콘텐츠 |
| R16 | 비교적 민감한 콘텐츠 |
| R18 | 성인 콘텐츠 |

모더레이션 결과는 공개 액세스를 결정할 때 사용됩니다.

모더레이션이 비활성화되어 있거나 오래된 파일에 등급이 없으면 해당 파일은 미등급으로 간주됩니다. 미등급 파일은 등급이 없다는 이유만으로 공개 갤러리나 랜덤 이미지 API에서 자동 제거되지 않습니다.

## 모더레이션 제공자 선택

사용 가능한 제공자는 다음과 같습니다.

- moderatecontent.com
- nsfwjs
- Sightengine

제공자마다 요구 사항이 다릅니다.

- moderatecontent.com은 일반적으로 API Key가 필요합니다.
- nsfwjs는 일반적으로 API 엔드포인트 URL이 필요합니다.
- Sightengine은 API user와 API secret이 필요합니다.

계정, 가용성, 감지 품질에 따라 선택하세요. 모더레이션이 활성화되고 올바르게 구성되어 있으면 ImgBed는 업로드 시 이미지 등급을 기록하려고 시도합니다.

## 공개 갤러리에 미치는 영향

공개 갤러리는 액세스 모드에 따라 파일을 필터링합니다.

예:

- 성인 모드: R18 이미지가 표시될 수 있습니다.
- 청소년 모드: R18 이미지는 숨겨집니다.
- 10대 모드: R16 및 R18 이미지는 숨겨집니다.
- 아동 모드: General 이미지 만 표시됩니다.

이는 일반 공개 액세스에만 영향을 줍니다. 관리자 패널은 계속 모든 파일을 표시합니다.

## 공개 파일 URL에 미치는 영향

공개 파일 URL은 방문자가 여는 직접 이미지 링크입니다.

파일 등급이 현재 액세스 모드에서 허용되면 ImgBed는 원본 이미지를 반환합니다.

등급이 허용 수준보다 높으면 일반 공개 액세스는 원본 이미지를 반환하지 않습니다. 대신 ImgBed는 구성된 차단 결과 또는 차단용 대체 이미지를 반환합니다.

예:

- 현재 모드는 아동 모드입니다.
- 어떤 이미지의 등급이 R18입니다.
- 방문자가 공개 URL을 직접 엽니다.
- ImgBed는 해당 방문자에게 R18 원본 이미지를 반환하지 않습니다.

![제한된 파일 이미지](../../image/Safety/文件受限图.png)

관리자 패널에서 파일을 보는 관리자는 이 제한의 영향을 받지 않습니다.

## 랜덤 이미지 API에 미치는 영향

랜덤 이미지 API도 액세스 모드에 따라 후보 풀을 필터링합니다.

아동 모드에서는 랜덤 이미지가 General 등급 파일에서만 선택됩니다.

청소년 모드에서는 랜덤 이미지가 General, R12, R16 파일에서 선택될 수 있지만 R18 파일은 선택되지 않습니다.

이렇게 하면 랜덤 이미지 API가 공개 갤러리 제한을 우회하지 못합니다.

## 목록 규칙과의 관계

액세스 모드가 유일한 공개 액세스 규칙은 아닙니다. 허용 목록 및 차단 목록 규칙과 함께 작동합니다.

간단히 말하면:

- 허용 목록의 콘텐츠는 공개 콘텐츠로 우선 처리됩니다.
- 차단 목록의 콘텐츠는 일반 방문자가 직접 볼 수 없습니다.
- 어떤 목록에도 없는 콘텐츠는 이후 액세스 모드에 따라 검사됩니다.

이미지가 연령 등급과 목록 규칙 모두에 의해 제한되면 일반 방문자는 여전히 원본 파일을 직접 볼 수 없습니다.

## 권장 설정

공개 사이트의 경우:

- 모더레이션을 활성화합니다.
- 사이트 대상 사용자에게 맞는 액세스 모드를 선택합니다.
- 모든 연령대의 방문자를 대상으로 한다면 아동 모드 또는 10대 모드를 사용합니다.
- 성인 콘텐츠를 공개적으로 표시하고 싶지 않다면 성인 모드를 피합니다.
- 관리자 패널에서 파일 등급을 검토하고 필요하면 수동으로 조정합니다.

비공개 또는 개인 사이트의 경우:

- 일반적으로 성인 모드로 충분합니다.
- 유용하다면 모더레이션을 활성화합니다.
- 필요에 따라 관리자 패널에서 등급을 검토하고 조정합니다.

## FAQ

### 액세스 모드를 변경하면 파일이 관리자 패널에서 사라지나요?

아니요.

액세스 모드는 일반 공개 액세스에만 영향을 줍니다. 관리자 패널에는 영향을 주지 않습니다.

### 아동 모드로 전환하면 공개 갤러리에 이미지가 더 적게 표시되는 이유는 무엇인가요?

아동 모드는 General 등급 파일만 공개적으로 표시합니다. R12, R16, R18 파일은 필터링됩니다.

### 공개 URL로 성인 이미지를 계속 열 수 있나요?

현재 액세스 모드가 해당 등급을 허용하지 않으면 일반 공개 URL은 원본 이미지를 반환하지 않습니다.

### 랜덤 이미지 API가 제한된 이미지를 반환할 수 있나요?

아니요.

랜덤 이미지 API는 현재 액세스 모드에 따라 후보를 필터링합니다.

### 오래된 미등급 이미지는 어떻게 되나요?

미등급 이미지는 모더레이션 결과가 없다는 이유만으로 자동 숨김 처리되지 않습니다. 나중에 관리자 패널에서 등급을 조정할 수 있습니다.
