# 페이지 설정

페이지 설정은 사이트 표시, 업로드 페이지 기본값, 배경 이미지, 관리 패널의 표시 방식을 제어합니다.

## 전역 설정

| 옵션 | 목적 |
| --- | --- |
| 사이트 제목 | 브라우저 탭에 표시되는 제목입니다. |
| 사이트 아이콘 | 브라우저 탭에 표시되는 작은 아이콘입니다. |
| ImgBed 이름 | 프런트엔드 페이지에 표시되는 이름입니다. |
| ImgBed 로고 | 프런트엔드 페이지에 표시되는 로고 이미지입니다. |
| 로고 링크 | 로고 또는 아바타를 클릭했을 때 열리는 URL입니다. |
| 배경 전환 간격 | 여러 배경을 순환하는 간격이며 단위는 밀리초입니다. `60000`은 60초를 의미합니다. |
| 배경 불투명도 | 배경 이미지의 불투명도이며 범위는 `0`부터 `1`까지입니다. 값이 낮을수록 더 옅게 표시됩니다. |
| 기본 URL 접두사 | 이미지 링크를 생성할 때 사용하는 접두사입니다. 비워 두면 현재 사이트 도메인을 사용합니다. |

## 클라이언트 설정

| 옵션 | 목적 |
| --- | --- |
| 공지 | 업로드 페이지 상단에 표시되는 공지입니다. HTML을 지원합니다. |
| 기본 업로드 채널 | 업로드 페이지에서 기본으로 선택되는 업로드 채널입니다. Smart Dispatch도 선택할 수 있습니다. |
| 기본 업로드 디렉터리 | 기본 업로드 디렉터리입니다. 예: `/user/`. 비어 있거나 `/`이면 루트를 의미합니다. |
| 기본 이름 지정 방식 | 업로드 후 파일 이름을 생성하는 기본 전략입니다. 아래 설명을 참고하세요. |
| 기본적으로 WebP로 변환 | 업로드 전에 이미지를 WebP로 변환합니다. |
| 기본적으로 압축 활성화 | 업로드 전에 브라우저에서 이미지를 로컬로 압축합니다. |
| 기본 압축 임계값 | 이미지가 이 크기를 초과하면 자동으로 압축합니다. 단위는 MB입니다. |
| 기본 목표 크기 | 압축 후 목표 파일 크기이며 단위는 MB입니다. |
| 로그인 페이지 배경 | 사용자 로그인 페이지의 배경 이미지입니다. |
| 업로드 페이지 배경 | 업로드 페이지의 배경 이미지입니다. |
| 푸터 포털 링크 | 푸터 포털 버튼이 여는 URL입니다. |
| 푸터 숨기기 | 활성화하면 프런트엔드 푸터를 숨깁니다. |

## 관리 설정

| 옵션 | 목적 |
| --- | --- |
| 관리 로그인 배경 | 관리 로그인 페이지의 배경 이미지입니다. |
| 관리 배경 | 관리 페이지의 배경 이미지입니다. 이미지 URL 하나 또는 여러 URL을 사용할 수 있습니다. |
| 이미지 로딩 모드 | 관리 파일 목록의 미리보기 로딩 모드입니다. 원본 이미지 모드는 원본 이미지를 로드합니다. 스마트 로딩은 공개 이미지는 썸네일을, 제한된 이미지는 원본 이미지를 우선 사용합니다. |
| 썸네일 소스 | 썸네일 생성에 사용하는 서비스: wsrv.nl, Cloudflare Image Resizing, WordPress Photon. Cloudflare Image Resizing을 선택하기 전에 Cloudflare에서 활성화해야 합니다. |
| Live2D 위젯 | 관리 패널에 Live2D 캐릭터를 표시합니다. |
| 클릭 시 불꽃 효과 | 페이지를 클릭할 때 불꽃 효과를 표시합니다. |
| 커서 별 꼬리 효과 | 마우스를 움직일 때 별 꼬리 효과를 표시합니다. |

## 배경 이미지 형식

로그인 페이지 배경, 업로드 페이지 배경, 관리 로그인 배경은 다음 형식을 지원합니다.

| 값 | 효과 |
| --- | --- |
| `bing` | Bing 배경화면 순환을 사용합니다. |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | 여러 이미지를 순환합니다. |
| `["https://example.com/1.jpg"]` | 단일 배경 이미지를 사용합니다. |
| `["https://your-domain.com/random?..."]` | 랜덤 이미지 API 링크를 사용합니다. 기타 설정에서 자체 랜덤 이미지 API를 구성한 뒤 생성된 랜덤 이미지 링크를 단일 배경 항목으로 여기에 붙여 넣을 수 있습니다. |

관리 배경은 이미지 URL을 지원합니다. 여러 URL은 페이지 안내에 따라 영문 쉼표로 구분할 수 있습니다. 비워 두면 기본 배경을 사용합니다.

## 기본 이름 지정 방식

| 방식 | 결과 |
| --- | --- |
| 기본 | 시간 기반 랜덤 접두사 + 원래 파일 이름, 예: `1760000000000_cat.png`. |
| 접두사만 | 시간 기반 랜덤 접두사와 확장자만 사용, 예: `1760000000000.png`. |
| 원래 이름만 | 원래 파일 이름을 유지합니다. 예: `cat.png`. 중복되면 ImgBed가 `(1)`, `(2)` 등을 추가합니다. |
| 짧은 링크 | 확장자를 포함한 8자 짧은 ID를 사용합니다. 예: `a1b2c3d4.png`. |
