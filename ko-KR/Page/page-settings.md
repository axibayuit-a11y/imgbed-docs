# 페이지 설정

페이지 설정에서는 ImgBed의 표시 방식, 공개 페이지, 업로드 화면에 관련된 기본 항목을 조정합니다.

## 설정 위치

```text
시스템 설정 -> 페이지 설정
```

## 주요 항목

| 항목 | 설명 |
| --- | --- |
| 사이트 이름 | 브라우저 제목과 페이지 표시 이름에 사용 |
| 사이트 설명 | 사이트 소개 문구로 표시 |
| 로고 / 아이콘 | 화면과 브라우저에서 사용할 이미지 |
| 배경 이미지 | 홈 화면 배경으로 사용할 이미지 |
| 푸터 정보 | 저작권 문구나 외부 링크 표시 |
| 업로드 안내 | 업로드 영역에 보여 줄 안내 문구 |

## 공개 페이지와 함께 확인할 것

방문자용 공개 페이지를 운영한다면 공개 디렉터리, 접근 모드, 이미지 로딩 방식도 함께 확인해야 합니다.

특히 공개 갤러리나 랜덤 이미지 API를 사용하는 경우, 페이지 설정뿐 아니라 기타 설정의 공개 범위도 같이 맞춰야 합니다.

## 변경 후 확인

1. 설정을 저장합니다.
2. 홈 화면을 새로고침합니다.
3. PC 화면과 모바일 폭에서 모두 확인합니다.
4. 로고, 배경, 설명 문구, 푸터 링크가 의도한 대로 보이는지 확인합니다.

이미지가 반영되지 않으면 이미지 URL이 직접 열리는지, 브라우저 캐시가 남아 있지 않은지 확인하세요.
