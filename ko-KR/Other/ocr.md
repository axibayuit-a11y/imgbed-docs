# OCR 문자 인식

OCR 문자 인식은 이미지, PDF, Word 파일 안의 문자를 읽어 검색과 내용 확인에 활용할 수 있게 하는 기능입니다.

## 설정 위치

```text
시스템 설정 -> 기타 설정 -> OCR 문자 인식
```

![OCR 설정](../../image/other/ip定位和ocr文字识别.png)

## 할 수 있는 일

| 기능 | 설명 |
| --- | --- |
| 이미지 OCR | 스크린샷이나 사진 속 문자를 읽습니다 |
| PDF OCR | PDF 안의 문자 내용을 추출합니다 |
| Word OCR | Word 문서의 텍스트를 읽습니다 |
| 검색 보조 | 인식한 문자를 파일 검색에 활용합니다 |
| 일괄 처리 | 기존 파일을 한 번에 인식할 수 있습니다 |

![OCR 결과](../../image/other/ocr识别截图.png)

## 사전 준비

사용할 OCR 서비스에 따라 API 키나 토큰을 준비해야 합니다. 화면 안내에 맞춰 필요한 인증 정보를 입력합니다.

PaddleOCR 같은 외부 서비스를 사용하는 경우, 인증 정보가 정확한지와 무료 사용량 또는 호출 제한에 여유가 있는지 확인해 두는 것이 좋습니다.

![토큰 가져오기](../../image/other/获取飞浆令牌.png)

## 사용 방법

1. OCR 기능을 켭니다.
2. 사용할 인식 서비스의 키를 입력합니다.
3. 설정을 저장합니다.
4. 새 파일을 업로드하거나 기존 파일에 OCR을 실행합니다.
5. 파일 상세 또는 검색에서 인식 결과를 확인합니다.

![Word 인식 결과](../../image/other/word识别结果.png)

![PDF 인식](../../image/other/pdf识别截图.png)

## 운영 팁

- 이미지가 너무 작거나 흐리거나 기울어져 있으면 인식률이 떨어집니다.
- PDF와 Word 파일은 크기가 클수록 처리 시간이 길어질 수 있습니다.
- API 사용량 제한에 도달하면 인식이 실패할 수 있습니다.
- OCR 결과는 보조 정보로 보고, 중요한 내용은 원본 파일도 함께 확인하세요.
