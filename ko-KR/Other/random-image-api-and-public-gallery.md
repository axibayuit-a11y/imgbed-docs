# 랜덤 이미지와 공개 갤러리

두 기능은 모두 다음 위치에서 설정합니다.

```text
System Settings -> Other Settings
```

## 랜덤 이미지 API

랜덤 이미지 API는 선택한 디렉터리에서 무작위 파일 하나를 반환합니다. 사이트 배경, 아바타 로테이션, 외부 페이지의 랜덤 이미지 호출에 유용합니다.

활성화한 뒤에는 다음 URL을 사용할 수 있습니다.

```text
https://your-domain.com/random
```

## 랜덤 이미지 API 설정

| 옵션 | 목적 |
| --- | --- |
| 활성화 | `/random` 엔드포인트를 켜거나 끕니다. 비활성화되면 접근이 금지됩니다. |
| 디렉터리 | 랜덤 이미지 API가 사용할 수 있는 디렉터리를 제한합니다. 여기에 포함되지 않은 디렉터리는 API에서 사용할 수 없습니다. |
| 호출 데모 | 바로 복사할 수 있는 랜덤 이미지 API 링크를 생성합니다. |

여러 디렉터리를 선택할 수 있습니다. 예를 들어 `/landscape/`와 `/portrait/`만 허용하면 랜덤 이미지 API는 해당 디렉터리와 그 하위 디렉터리에서만 파일을 선택할 수 있습니다.

## 랜덤 이미지 API 파라미터

| 파라미터 | 예 | 목적 |
| --- | --- | --- |
| `dir` | `/landscape/` | 랜덤 디렉터리를 지정합니다. |
| `content` | `image` | 미디어 유형을 지정합니다. `image`, `video`, `audio` 또는 쉼표로 구분한 조합을 사용합니다. |
| `orientation` | `auto` | 이미지 방향을 필터링합니다. `portrait`, `landscape`, `auto`를 사용합니다. |
| `type` | `url` | 반환 형식입니다. 비어 있으면 리다이렉트, `url`은 일반 텍스트 URL, `json`은 JSON을 반환합니다. |
| `origin` | `1` | `type=url`과 함께 사용하여 전체 URL을 반환합니다. |
| `age` | `all-ages,r12` | 연령 등급으로 필터링합니다. |
| `tag` | `wallpaper,sky` | 이 태그를 포함한 파일만 반환합니다. |
| `ex` | `private` | 이 태그를 포함한 파일을 제외합니다. |

## 반환 형식

`type`이 없으면 API는 랜덤 파일 URL로 바로 리다이렉트합니다.

`type=url`이면 텍스트 URL을 반환합니다.

`type=json`이면 파일 URL, 파일 ID, 파일명, 파일 유형, 태그, 등급 및 관련 메타데이터를 포함한 파일 정보를 반환합니다.

## 접근 규칙

랜덤 이미지 API는 공개 접근 규칙을 따릅니다.

| 규칙 | 효과 |
| --- | --- |
| 디렉터리 제한 | 허용된 디렉터리 안의 파일만 선택될 수 있습니다. |
| 차단 목록 | 차단 목록에 있는 파일은 랜덤 후보에서 제외됩니다. |
| 허용 목록 모드 | 활성화되면 공개 접근이 허용된 파일만 반환됩니다. |
| 연령 등급 | R12, R16, R18 및 유사 콘텐츠는 현재 접근 모드에 따라 필터링됩니다. |

필터링 후 일치하는 파일이 없으면 API는 일치 결과 없음을 반환합니다.

## 캐시

랜덤 이미지 API는 속도 향상을 위해 디렉터리 후보 풀을 캐시합니다.

파일이 변경되면 ImgBed는 디렉터리 캐시 버전을 업데이트하고, 이후 요청에서 후보 풀을 다시 만듭니다. 빈 디렉터리는 반복 쿼리를 피하기 위해 짧게 캐시됩니다.

## 공개 갤러리

공개 갤러리는 방문자에게 표시하도록 허용한 디렉터리를 읽기 전용으로 탐색할 수 있는 공개 페이지를 제공합니다.

활성화하면 방문자는 다음 URL을 열 수 있습니다.

```text
https://your-domain.com/browse/directory-name
```

## 공개 갤러리 설정

| 옵션 | 목적 |
| --- | --- |
| 활성화 | 공개 갤러리를 켜거나 끕니다. 비활성화되면 방문자는 탐색할 수 없습니다. |
| 이미지 로딩 모드 | 미리보기에 원본 이미지를 사용할지 썸네일을 사용할지 제어합니다. |
| 공개 디렉터리 | 방문자가 접근할 수 있는 디렉터리를 설정합니다. |

## 이미지 로딩 모드

| 모드 | 목적 |
| --- | --- |
| 원본 | 방문자 페이지가 원본 파일을 직접 로드합니다. |
| 썸네일 | 방문자 페이지가 더 빠른 로딩을 위해 썸네일을 우선 사용합니다. |

## 공개 디렉터리

공개 디렉터리는 방문자가 볼 수 있는 내용을 결정합니다.

예:

```text
/1/,/2/,/landscape/,/portrait/
```

그러면 방문자는 다음에 접근할 수 있습니다.

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

`/2026/lucky/` 같은 하위 디렉터리도 공개할 수 있습니다. 공개되지 않은 디렉터리에 대한 방문자 접근은 차단됩니다.

## 공개 갤러리 기능

| 기능 | 설명 |
| --- | --- |
| 디렉터리 탐색 | 공개 디렉터리의 파일과 하위 디렉터리를 봅니다. |
| 검색 | 파일명, 파일 ID 또는 태그로 검색합니다. |
| 유형 필터 | 이미지, 동영상, 오디오 또는 기타 파일을 필터링합니다. |
| 태그 필터 | 선택한 태그를 포함하거나 제외합니다. |
| 방향 필터 | 가로 또는 세로 이미지를 필터링합니다. |
| 시간 필터 | 업로드 시간 범위로 필터링합니다. |
| 확장자 필터 | 파일 확장자로 필터링합니다. |
| 링크 복사 | 파일 접근 링크를 복사합니다. |
| 미디어 미리보기 | 방문자 페이지에서 이미지, 동영상, 오디오를 보거나 재생합니다. |

## 공개 갤러리 접근 규칙

공개 갤러리도 공개 접근 규칙을 따릅니다.

| 규칙 | 효과 |
| --- | --- |
| 공개 디렉터리 | 허용된 디렉터리만 표시됩니다. |
| 접근 모드 | 콘텐츠는 현재 연령 등급 접근 모드에 따라 필터링됩니다. |
| 허용 목록 모드 | 활성화되면 공개 접근이 허용된 파일만 표시됩니다. |
| 차단 목록 | 차단 목록에 있는 파일은 숨겨집니다. |
