# IP 위치 조회와 사용자 관리

IP 위치 조회는 업로드 기록, 로그인 기기, 접속 로그 등에 남은 IP 주소를 대략적인 지역 정보로 변환합니다.

설정해 두면 관리자 화면에서 업로드 출처와 접속 위치를 더 쉽게 파악할 수 있습니다. 사용자 관리에서는 의심스러운 IP의 업로드를 차단하거나 다시 허용할 수도 있습니다.

## 설정 위치

```text
시스템 설정 -> 기타 설정 -> IP 위치 조회
```

![IP 위치 조회](../../image/other/ip定位/ip定位.png)

## 설정 항목

| 항목 | 설명 |
| --- | --- |
| IP 위치 표시 언어 | 한국어, 영어, 중국어 등 표시 언어 선택 |
| MaxMind Account ID | MaxMind GeoLite Web Service 계정 ID |
| MaxMind License Key | MaxMind 라이선스 키 |
| Tencent Map Key | 중국 지역 주소 표시에 강한 Tencent 위치 서비스 키 |
| ipapi Key | 다국어 IP 위치 조회를 지원하는 APILayer ipapi 키 |

필요한 서비스만 설정하면 됩니다. 아무 키를 넣지 않아도 내장 무료 소스를 시도하지만, 안정성, 언어 지원, 정확도는 직접 설정한 서비스가 더 좋습니다.

## 한국어 표시로 쓰려면

표시 언어를 한국어로 두고 MaxMind 또는 ipapi를 설정하는 방식이 무난합니다. 중국 본토 IP를 더 자세히 보고 싶다면 Tencent Map Key를 추가로 설정합니다.

## MaxMind 설정

MaxMind에는 다음 두 값이 필요합니다.

```text
MaxMind Account ID
MaxMind License Key
```

대시보드에서 Account ID를 확인하고, License Keys 페이지에서 키를 만든 뒤 ImgBed에 붙여 넣습니다.

![MaxMind 키 설정](../../image/other/ip定位/maxmind的key配置.png)

## ipapi 설정

ipapi 콘솔에서 API Key를 복사해 ImgBed의 `ipapi Key` 항목에 붙여 넣습니다.

![ipapi 설정](../../image/other/ip定位/ipapi配置.png)

## 사용자 관리에서 확인할 내용

사용자 관리는 관리자 화면 상단에서 열 수 있습니다.

![사용자 관리](../../image/other/用户管理显示.png)

IP별로 다음 정보를 확인합니다.

| 항목 | 설명 |
| --- | --- |
| IP | 업로드 출처 IP |
| 위치 | IP로 추정한 지역 |
| 총 업로드 용량 | 해당 IP가 업로드한 전체 용량 |
| 업로드 횟수 | 업로드한 파일 수 |
| 업로드 허용 | 끄면 이후 해당 IP의 업로드를 차단 |

왼쪽 화살표를 열면 해당 IP가 업로드한 파일 목록을 볼 수 있습니다. 파일명, 미리보기, 파일 크기, 심사 결과, 상태, 업로드 시각을 확인한 뒤 차단 여부를 판단합니다.

![고급 필터](../../image/other/用户管理高级筛选.png)

## 운영 팁

- 의심스러운 IP는 먼저 파일 내용을 확인한 뒤 업로드 허용을 끕니다.
- 최근 업로드, 고빈도 업로드, 대용량 업로드는 검색과 정렬로 빠르게 찾을 수 있습니다.
- 위치 정보는 추정값입니다. 최종 판단 기준이 아니라 참고 자료로 활용하세요.
