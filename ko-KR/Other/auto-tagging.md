# 자동 태그

자동 태그 기능은 다음 위치에서 설정합니다.

```text
시스템 설정 -> 기타 설정 -> 자동 태그
```

이미지에 태그를 자동으로 붙여 검색, 랜덤 이미지 필터, 공개 갤러리 필터, 연령 등급 기반 접근 제어에 활용할 수 있게 합니다.

## 할 수 있는 일

| 기능 | 설명 |
| --- | --- |
| 콘텐츠 태그 생성 | 인물, 배경, 사물, 화풍 등 시각적 태그를 추가 |
| 캐릭터 태그 생성 | 애니메이션 이미지나 일러스트 캐릭터 판별에 유용 |
| 방향 태그 보완 | `landscape`, `portrait`, `square` 추가 |
| 이미지 등급 저장 | `G/S/Q/E` 등급 결과 저장 |
| 업로드 시 자동 태그 | 새 이미지 업로드 후 자동으로 태그 처리 |
| 일괄 태그 | 기존 이미지에 한 번에 태그 추가 |

## 먼저 준비할 것

접근 가능한 Hugging Face Space URL이 최소 1개 필요합니다.

권장 방식은 SmilingWolf의 `wd-tagger` Space를 본인의 Hugging Face 계정으로 복제해 사용하는 것입니다.

```text
https://huggingface.co/spaces/SmilingWolf/wd-tagger
```

공개 Space를 임시로 사용할 수도 있지만, 여러 사용자가 함께 쓰기 때문에 대기열이 길어지거나 느려지거나 일시적으로 사용할 수 없을 수 있습니다. 장기적으로는 본인 계정에 복제한 Space가 더 안정적입니다.

## Space 복제하기

1. Hugging Face에 로그인합니다.
2. `https://huggingface.co/spaces/SmilingWolf/wd-tagger`를 엽니다.

![SmilingWolf 공개 Space](../../image/other/微笑狼的公开仓库.png)

3. 오른쪽 위 메뉴에서 `Duplicate this Space`를 선택합니다.
4. Space 이름은 기본값을 쓰거나 `wd-tagger`처럼 알아보기 쉽게 정합니다.
5. 공개 범위는 `Public`으로 두는 편이 ImgBed에서 호출하기 쉽습니다.
6. 처음에는 무료 하드웨어로 시작합니다.
7. 빌드가 끝날 때까지 기다립니다.

빌드가 완료되면 브라우저 주소를 복사해 ImgBed의 `Space URLs`에 붙여 넣습니다.

## 설정 항목

| 항목 | 권장값 |
| --- | --- |
| `Space URLs` | 준비한 Space URL을 한 줄에 하나씩 입력 |
| 대상 폴더 | 전체 처리면 비워 두고, 특정 폴더만 처리하려면 선택 |
| 인식 모델 | 기본은 `wd-swinv2-tagger-v3` |
| 일반 태그 임계값 | 보통 `0.35` 전후로 시작 |
| 캐릭터 태그 임계값 | 오인식을 줄이려면 `0.85` 전후 |
| `MCut` | 처음에는 끄고 테스트 |
| 업로드 시 자동 태그 | 새 이미지에도 적용하려면 켜기 |

태그가 너무 많으면 일반 태그 임계값을 조금 올리고, 너무 적으면 조금 낮춥니다.

## 일괄 태그

1. `Space URLs`를 입력합니다.
2. 대상 폴더를 선택합니다.
3. 태그 작업을 시작합니다.
4. 진행률이 끝날 때까지 기다립니다.

대상 폴더가 비어 있으면 전체 폴더를 처리합니다.

## 업로드 시 자동 태그

이 옵션을 켜면 새로 업로드한 이미지가 자동으로 태그 처리 흐름에 들어갑니다.

Space가 대기 중이어도 업로드 자체는 먼저 끝날 수 있고, 태그 처리는 뒤에서 계속 진행됩니다.

## 자주 묻는 점

### 왜 내 Space를 복제해서 쓰나요?

공개 Space는 많은 사용자가 함께 사용합니다. 본인 Space는 주로 내 ImgBed 사이트에서만 호출하므로 더 빠르고 안정적입니다.

### 태그가 영어로 나오는 것이 정상인가요?

정상입니다. SmilingWolf 계열 모델은 영어 태그를 반환합니다. 이 태그는 검색, 필터, 랜덤 이미지 API, 공개 갤러리에서 사용됩니다.

### 등급 태그는 어디에 쓰나요?

보안 설정의 접근 모드와 함께 사용합니다. 방문자 접근을 연령 등급으로 제한할 때 공개 갤러리와 랜덤 이미지 기능이 해당 규칙을 따릅니다.

## 빠른 절차

```text
Hugging Face 로그인
-> SmilingWolf/wd-tagger 열기
-> Duplicate this Space
-> 빌드 완료 대기
-> Space URL 복사
-> ImgBed의 Space URLs에 입력
-> 모델과 임계값 설정
-> 일괄 태그 시작 또는 업로드 시 자동 태그 켜기
```
