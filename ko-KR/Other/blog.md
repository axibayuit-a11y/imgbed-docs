# 블로그 사용 안내

블로그 기능을 켜면 ImgBed 사이트에 별도의 블로그 페이지를 추가할 수 있습니다.

```text
https://내도메인/blog/
```

![블로그 홈](../../image/other/博客/博客首页.png)

이 블로그 페이지는 오픈소스 프로젝트 [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki)를 바탕으로 ImgBed에 맞게 Vue로 다시 구성한 기능입니다.

## 설정 위치

```text
시스템 설정 -> 기타 설정 -> 블로그
```

![블로그 설정](../../image/other/博客/QQ20260611-221702.png)

## 처음 사용할 때

1. `사용` 스위치를 켭니다.
2. 블로그 설정을 저장할 GitHub 계정을 선택합니다.
3. `블로그 업데이트`를 누릅니다.
4. 업데이트 성공 안내가 나올 때까지 기다립니다.
5. `https://내도메인/blog/` 주소로 접속해 확인합니다.

처음 사용할 때는 선택한 GitHub 계정 아래에 비공개 저장소가 준비됩니다.

```text
imgbed-blog-config
```

이 저장소에는 블로그 설정과 글 내용이 저장됩니다.

## 글 작성 방식

블로그 글은 본인의 GitHub 비공개 저장소 `imgbed-blog-config`에서 수정합니다.

일반적인 작업 흐름은 다음과 같습니다.

1. GitHub를 엽니다.
2. `imgbed-blog-config` 저장소로 들어갑니다.
3. 글 파일을 새로 만들거나 수정합니다.
4. 변경 내용을 커밋합니다.
5. ImgBed 관리자 화면으로 돌아와 `블로그 업데이트`를 누릅니다. 블로그 홈 왼쪽 위 로고를 세 번 눌러 업데이트를 트리거할 수도 있습니다.

`블로그 업데이트`는 이미 작성한 글을 덮어쓰는 기능이 아닙니다. 저장소를 처음 준비하거나 블로그 캐시를 새로고침할 때 사용합니다.

## 지원 기능

글 목록, 카테고리, 태그, 아카이브, 검색, 다크 모드, 다국어 전환 같은 기본 블로그 기능을 지원합니다.

댓글과 방문 통계도 사용할 수 있습니다.

![블로그 댓글](../../image/other/博客/支持留言.png)

댓글은 글 아래에 표시됩니다. 방문자는 아바타, 닉네임, 이메일, 댓글 내용을 입력해 남길 수 있습니다.

방문 통계에서는 글 조회수와 사이트 방문 수를 확인할 수 있어 블로그 이용 현황을 파악하기 좋습니다.

## 접속 주소

블로그는 항상 `/blog/` 경로에서 열립니다.

예를 들어 ImgBed 도메인이 다음과 같다면:

```text
https://image.example.com
```

블로그 주소는 다음과 같습니다.

```text
https://image.example.com/blog/
```

블로그를 끄면 방문자는 블로그 페이지에 접속할 수 없습니다.
