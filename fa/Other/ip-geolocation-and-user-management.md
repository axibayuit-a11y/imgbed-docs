# مکان‌یابی IP و مدیریت کاربران

مکان‌یابی IP، نشانی‌های IP موجود در رکوردهای بارگذاری، دستگاه‌های ورود و گزارش‌های مشابه را به مکان‌های تقریبی تبدیل می‌کند.

پس از پیکربندی، پنل مدیریت می‌تواند مبدأ بارگذاری و دسترسی را واضح‌تر نمایش دهد. مدیریت کاربران همچنین اجازه می‌دهد دسترسی بارگذاری برای نشانی‌های IP مشکوک مسدود یا دوباره فعال شود.

## محل پیکربندی

باز کنید:

```text
System Settings -> Other Settings -> IP Geolocation
```

![مکان‌یابی IP](../../image/other/ip定位/ip定位.png)

## تنظیمات موجود

روند جدیدتر مکان‌یابی IP به‌جای تکیه بر یک سرویس نقشه، از چند منبع پشتیبانی می‌کند.

| تنظیم | هدف |
| --- | --- |
| زبان مکان‌یابی IP | زبان نمایش را انتخاب می‌کند؛ مانند انگلیسی، چینی ساده‌شده، ژاپنی، فرانسوی و موارد دیگر. |
| MaxMind Account ID | شناسه حساب MaxMind برای MaxMind GeoLite Web Service. |
| MaxMind License Key | کلید مجوز MaxMind. |
| Tencent Map Key | کلید Tencent Location Service. برای نشانی‌ها به زبان چینی و IPهای سرزمین اصلی چین مفید است. |
| ipapi Key | کلید APILayer ipapi. از مکان‌یابی IP چندزبانه پشتیبانی می‌کند. |

فقط سرویس‌هایی را پر کنید که نیاز دارید. لازم نیست همه فیلدها را پیکربندی کنید.

اگر هیچ کلیدی وارد نشود، ImgBed همچنان تلاش می‌کند از منابع رایگان داخلی استفاده کند، اما پایداری، پشتیبانی زبانی و دقت ممکن است کمتر از سرویسی باشد که خودتان پیکربندی کرده‌اید.

## گزینه‌های پیشنهادی

اگر بیشتر به نشانی‌ها به زبان چینی نیاز دارید:

1. زبان مکان‌یابی IP را روی چینی ساده‌شده بگذارید.
2. Tencent Map Key را پیکربندی کنید.
3. در صورت نیاز MaxMind یا ipapi را به‌عنوان منبع پشتیبان اضافه کنید.

اگر بیشتر به نشانی‌های انگلیسی یا چندزبانه نیاز دارید:

1. زبان موردنیاز را انتخاب کنید.
2. MaxMind Account ID و License Key را پیکربندی کنید.
3. اگر نتایج چندزبانه بهتر می‌خواهید، ipapi Key اضافه کنید.

## راه‌اندازی MaxMind

MaxMind به این موارد نیاز دارد:

```text
MaxMind Account ID
MaxMind License Key
```

Account ID را در داشبورد MaxMind پیدا کنید و از صفحه License Keys یک License Key بسازید.

![پیکربندی کلید MaxMind](../../image/other/ip定位/maxmind的key配置.png)

پس از ساخت، Account ID و License Key را در ImgBed جای‌گذاری و ذخیره کنید.

طرح رایگان MaxMind برای استفاده روزمره مناسب است، اما محدودیت درخواست دارد. اگر سهمیه تمام شود، ImgBed منابع موجود دیگر را امتحان می‌کند.

## راه‌اندازی ipapi

ipapi از APILayer API Key استفاده می‌کند.

کنسول ipapi را باز کنید و API Key نمایش‌داده‌شده را کپی کنید.

![پیکربندی ipapi](../../image/other/ip定位/ipapi配置.png)

آن را در فیلد `ipapi Key` داخل ImgBed جای‌گذاری و ذخیره کنید.

ipapi از مکان‌یابی IP چندزبانه پشتیبانی می‌کند و وقتی می‌خواهید نشانی‌ها با زبان انتخاب‌شده نمایش داده شوند مفید است. طرح رایگان آن نیز محدودیت درخواست دارد. اگر سهمیه تمام شود، ImgBed منابع موجود دیگر را امتحان می‌کند.

## راه‌اندازی Tencent Map Key

Tencent Map Key برای نشانی‌ها به زبان چینی، به‌خصوص IPهای سرزمین اصلی چین، مفید است.

هنگام ساخت کلید در Tencent Location Service، این مورد را فعال کنید:

```text
WebServiceAPI
```

پس از ساخت، کلید را در `Tencent Map Key` جای‌گذاری و ذخیره کنید.

اگر فقط مکان‌یابی پایه IP به زبان چینی می‌خواهید، Tencent Map Key برای شروع کافی است.

## در مدیریت کاربران چه چیزهایی را بررسی کنیم

مدیریت کاربران از بالای پنل مدیریت در دسترس است.

![مدیریت کاربران](../../image/other/用户管理显示.png)

مدیریت کاربران فعالیت بارگذاری را بر اساس IP نشان می‌دهد:

| فیلد | توضیح |
| --- | --- |
| منبع IP | IP مبدأِ کاربر بارگذاری‌کننده. |
| نشانی | مکان تقریبی به‌دست‌آمده از IP. |
| مجموع اندازه بارگذاری | مجموع اندازه فایل‌هایی که این IP بارگذاری کرده است. |
| تعداد بارگذاری | تعداد بارگذاری‌ها از این IP. |
| بارگذاری مجاز | روشن یعنی بارگذاری مجاز است. خاموش یعنی بارگذاری مسدود است. |

برای باز کردن فهرست فایل‌های بارگذاری‌شده توسط آن IP، روی فلش سمت چپ کلیک کنید.

فهرست فایل‌ها نام فایل، پیش‌نمایش، اندازه فایل، نتیجه بررسی، وضعیت فایل و زمان بارگذاری را نشان می‌دهد. اگر بارگذاری‌ها مشکوک به نظر می‌رسند، ابتدا IP را باز کنید، فایل‌ها را بررسی کنید و سپس تصمیم بگیرید بارگذاری‌های بعدی مسدود شوند یا نه.

اگر IP مشکوک است، `Upload allowed` را خاموش کنید. بارگذاری‌های آینده از آن IP مسدود خواهند شد.

## جست‌وجو، مرتب‌سازی و فیلترهای پیشرفته

در بالای مدیریت کاربران می‌توانید بر اساس منبع IP یا نشانی جست‌وجو کنید.

برای یافتن کاربرانی که اخیراً بارگذاری کرده‌اند، کاربران پرتکرار یا IPهای پرمصرف، بر اساس زمان، تعداد بارگذاری یا مجموع اندازه بارگذاری مرتب کنید.

برای بررسی دقیق‌تر، فیلترهای پیشرفته را باز کنید.

![فیلترهای پیشرفته](../../image/other/用户管理高级筛选.png)

فیلترهای پیشرفته از این موارد پشتیبانی می‌کنند:

| فیلتر | کاربرد |
| --- | --- |
| بازه زمانی | IPهایی را نشان می‌دهد که در دوره انتخاب‌شده فایل بارگذاری کرده‌اند. |
| وضعیت دسترسی | بر اساس حالت عادی، مسدود و وضعیت‌های مشابه فیلتر می‌کند. |
| فهرست مجاز/مسدود | بر اساس فهرست مجاز، فهرست مسدود یا تنظیم‌نشده فیلتر می‌کند. |
| نوع فایل | IPهایی را نشان می‌دهد که تصویر، ویدئو، صدا، سند، کد یا فایل‌های دیگر بارگذاری کرده‌اند. |
| اندازه فایل | بر اساس بازه اندازه فایل بارگذاری‌شده فیلتر می‌کند. |
| رده‌بندی سنی | بر اساس تنظیم‌نشده، General، R12+، R16+، R18 و رده‌های مشابه فیلتر می‌کند. |
| وضعیت فایل | برای بررسی فایل‌های غیرعادی، بر اساس وضعیت فعلی فایل فیلتر می‌کند. |

برای اعمال، روی `Apply Filters` کلیک کنید. برای بازگشت به همه داده‌ها از `Reset` استفاده کنید.

## نمایش موبایل

در موبایل، مدیریت کاربران به چیدمان کارت تبدیل می‌شود.

![مدیریت کاربران در موبایل](../../image/other/手机端显示用户管理效果.png)

هر کارت IP، نشانی، مجموع اندازه بارگذاری، تعداد بارگذاری و کلید بارگذاری مجاز را نشان می‌دهد. می‌توانید بدون اسکرول افقی جدول، کاربران را مدیریت کنید.

## اگر مکان اشتباه به نظر می‌رسد

مکان‌یابی IP تقریبی است. این یک نشانی دقیق خیابانی نیست.

اگر کاربر پشت پراکسی، مرکز داده، سرور ابری یا شبکه فرامرزی باشد، مکان نمایش‌داده‌شده ممکن است با مکان واقعی متفاوت باشد.

از این قابلیت برای درک مبدأ تقریبی، یافتن بارگذاری‌های غیرعادی و کمک به تصمیم‌های مسدودسازی استفاده کنید. آن را ردیابی دقیق در نظر نگیرید.

## موارد رایج

| مورد | معنی |
| --- | --- |
| نشانی خالی است | IP شاید هنوز مکان‌یابی نشده باشد، یا منبع فعلی موقتاً در دسترس نباشد. |
| زبان نشانی اشتباه است | زبان مکان‌یابی IP و وجود منبعی را که از آن زبان پشتیبانی می‌کند بررسی کنید. |
| نشانی مرکز داده نشان می‌دهد | بسیاری از پراکسی‌ها، سرورهای ابری و خزنده‌ها به‌صورت نشانی مرکز داده یا ISP دیده می‌شوند. |
| تعداد بارگذاری زیاد است | این IP را با دقت بررسی کنید و در صورت نیاز بارگذاری‌ها را مسدود کنید. |
| مجموع اندازه بارگذاری زیاد است | مرتب یا فیلتر کنید، IP را باز کنید و فایل‌های مشخص را بررسی کنید. |
| نیاز به بازگردانی پس از مسدودسازی | `Upload allowed` را دوباره روشن کنید. |

## روند سریع

```text
Open IP Geolocation in Other Settings
-> Choose IP geolocation language
-> Fill MaxMind, Tencent Map, or ipapi credentials as needed
-> Save settings
-> Open User Management
-> Review IP source, address, total upload size, and upload count
-> Use search, sort, or advanced filters to find abnormal IPs
-> Allow or block uploads as needed
```
