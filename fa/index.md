# مستندات کاربری ImgBed

ImgBed پروژه‌ای برای میزبانی تصویر و اشتراک‌گذاری فایل است که بر پایه اکوسیستم Cloudflare ساخته شده است. این پروژه ابتدا از آپلود تصویر و مدیریت لینک شروع شد، اما اکنون به ابزاری کامل‌تر برای میزبانی محتوای شخصی تبدیل شده است: می‌توانید فایل‌ها را در چندین کانال ذخیره‌سازی آپلود کنید، گالری عمومی منتشر کنید، وبلاگ خودتان را داشته باشید، فایل‌ها را از لینک‌های magnet منتقل کنید و با نودهای دیگر ImgBed از طریق یک فهرست فایل فدرال‌شده همکاری کنید.

ما می‌خواهیم ImgBed فقط ابزاری برای «آپلود تصویر و گرفتن لینک» نباشد. هدف این است که هر فرد راحت‌تر بتواند میزبان تصویر، وبلاگ و فضای اشتراک‌گذاری فایل خودش را داشته باشد. چه بخواهید تصاویر را نگه دارید، آلبوم‌ها را مدیریت کنید، چند مطلب بنویسید یا یک صفحه سبک برای منابع عمومی بسازید، ImgBed تلاش می‌کند برای شیوه استفاده شما انعطاف باقی بگذارد.

در حال حاضر ImgBed از ۲۸ زبان بومی‌سازی‌شده و کانال‌های آپلود متعددی مانند Telegram، Cloudflare R2، S3، WebDAV، Discord، Hugging Face، GitHub Releases، GitLab Packages، OneDrive، Google Drive، Dropbox، Yandex و pCloud پشتیبانی می‌کند. می‌توانید براساس منطقه، حساب‌ها و عادت استفاده خودتان گزینه ذخیره‌سازی مناسب را انتخاب کنید.

این پروژه همچنان در حال بهبود است. توان ما محدود است، اما تلاش می‌کنیم هر قابلیت را روشن، پایدار و کاربردی بسازیم. اگر ImgBed به شما کمک کند میزبان تصویر، وبلاگ یا فضای اشتراک‌گذاری فایل خودتان را بسازید، بسیار خوشحال می‌شویم.
