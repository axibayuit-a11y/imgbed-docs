# تنظیمات بارگذاری

تنظیمات بارگذاری، ImgBed را به کانال‌های ذخیره‌سازی اختصاصی شما متصل می‌کند. پس از پیکربندی یک کانال، تصاویر و فایل‌های بارگذاری‌شده در سرویس انتخابی شما ذخیره می‌شوند. ImgBed رکوردهای فایل، لینک‌های دسترسی، پیش‌نمایش‌ها، گالری عمومی، API تصویر تصادفی، دسترسی WebDAV و جریان‌های مرتبط را مدیریت می‌کند.

هر کاربر ممکن است کانال‌های متفاوتی را ترجیح دهد. اگر راه‌اندازی ساده می‌خواهید، Telegram، Discord یا GitHub Releases می‌توانند نقطه شروع مناسبی باشند. اگر ظرفیت، سرعت و پایداری بلندمدت برایتان مهم‌تر است، Cloudflare R2، S3، OneDrive، Google Drive، Dropbox، Yandex، pCloud یا سرویس WebDAV خودتان را در نظر بگیرید.

## پیش از شروع

- حساب ذخیره‌سازی یا اعتبارنامه‌های API موردنیاز را آماده کنید.
- مطمئن شوید دامنه ImgBed شما در دسترس است، زیرا کانال‌های مبتنی بر OAuth به URL بازگشت نیاز دارند.
- پس از افزودن هر کانال، ابتدا یک تصویر آزمایشی بارگذاری کنید تا مطمئن شوید فایل‌ها درست ذخیره و باز می‌شوند.
