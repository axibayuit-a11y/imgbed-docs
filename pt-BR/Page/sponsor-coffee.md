# Pagar um café

Olá, eu sou Linjiang, o desenvolvedor do ImgBed.  

No começo eu só queria criar um host simples de imagens aleatórias para o meu próprio blog. Por acaso encontrei o projeto open source CloudFlare-ImgBed. Na época ele ainda era bem básico, então comecei enviando alguns PRs para completar funções que eu precisava.  
Conforme fui mexendo, surgiu uma ideia: será que daria para transformar isso em uma ferramenta completa de hospedagem de conteúdo pessoal, fácil para qualquer pessoa usar?  

Passei vários meses reconstruindo o projeto aos poucos e polindo cada recurso até chegar ao que ele é hoje. Sendo sincero, não sou desenvolvedor profissional; ainda estou aprendendo. Sem ferramentas de IA como Codex e Claude, eu não teria conseguido terminar uma arquitetura tão complexa.  

Se o ImgBed realmente te ajudou, espero de coração que você considere apoiar o desenvolvimento. Mesmo o valor de um café significa muito para mim.
