# Configurações da página

As configurações da página controlam elementos básicos de apresentação, páginas públicas e experiência de upload no ImgBed.

## Onde configurar

```text
Configurações do sistema -> Configurações da página
```

## Principais opções

| Opção | Descrição |
| --- | --- |
| Nome do site | Usado no título do navegador e na interface |
| Descrição do site | Texto de apresentação do site |
| Logotipo / ícone | Imagens usadas na página e no navegador |
| Imagem de fundo | Fundo da página inicial |
| Rodapé | Copyright, links externos ou outras informações |
| Texto de upload | Mensagem exibida na área de upload |

## Páginas públicas

Se você usa páginas para visitantes, confira também diretórios públicos, modo de acesso e modo de carregamento de imagens.

Especialmente com galeria pública ou API de imagem aleatória, as configurações da página precisam combinar com as permissões de publicação em Outras configurações.

## Depois de salvar

1. Salve as alterações.
2. Recarregue a página inicial.
3. Confira em desktop e em largura de celular.
4. Verifique se logotipo, fundo, descrição e links do rodapé aparecem como esperado.

Se uma imagem não atualizar, veja se a URL abre diretamente e se o navegador não está usando cache antigo.
