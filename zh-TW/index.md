# ImgBed 使用者文件

ImgBed 是一個基於 Cloudflare 生態打造的圖床與檔案分享專案。它最初從圖片上傳與連結管理開始，現在已經擴展成更完整的個人內容託管工具：你可以把檔案上傳到不同儲存渠道，公開展示圖片，建立自己的部落格，透過磁力連結轉存檔案，也可以和其他 ImgBed 節點組成聯盟，分享彼此開放的檔案索引。

我們希望 ImgBed 不只是「上傳圖片取得連結」的工具，而是讓每個人都能更容易擁有自己的圖床、部落格和檔案分享方式。無論你只是想保存圖片、整理相簿、寫一些文章，或是建立一個輕量的公開資源頁，ImgBed 都盡量保留彈性給你。

目前 ImgBed 支援 28 種在地化語言，並提供 Telegram、Cloudflare R2、S3、WebDAV、Discord、Hugging Face、GitHub Releases、GitLab Packages、OneDrive、Google Drive、Dropbox、Yandex、pCloud 等多種上傳渠道。你可以依照所在地區、帳號和使用習慣選擇合適的儲存方式。

這個專案仍在持續完善。我們能力有限，但會盡力把每個功能做得清楚、穩定、好用。如果 ImgBed 能幫你更輕鬆地擁有自己的圖床、部落格或檔案分享空間，我們會非常開心。
