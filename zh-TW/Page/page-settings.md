# 頁面設定說明

頁面設定用來調整網站顯示、上傳頁預設選項、背景圖和管理端顯示效果。

## 全域設定

| 選項 | 功能 |
| --- | --- |
| 網站標題 | 瀏覽器分頁顯示的標題 |
| 網站圖示 | 瀏覽器分頁的小圖示位址 |
| 圖床名稱 | 前台頁面顯示的圖床名稱 |
| 圖床 Logo | 前台頁面顯示的 Logo 圖片位址 |
| Logo 跳轉連結 | 點擊 Logo 或頭像後開啟的連結 |
| 背景切換間隔 | 多張背景輪播的切換時間，單位毫秒；`60000` 代表 60 秒 |
| 背景圖透明度 | 背景圖透明度，範圍 `0` 到 `1`；數值越小越淡 |
| 預設 URL 前綴 | 產生圖片連結時使用的前綴；留空時使用目前站台網域 |

## 使用者端設定

| 選項 | 功能 |
| --- | --- |
| 公告 | 上傳頁頂部公告內容，支援 HTML |
| 預設上傳渠道 | 上傳頁預設選中的上傳渠道，也可以選 Smart Dispatch 自動調度 |
| 預設上傳目錄 | 上傳頁預設目錄，例如 `/user/`；留空或 `/` 代表根目錄 |
| 預設命名方式 | 上傳後檔名的預設產生方式，見下方說明 |
| 預設轉換 WebP | 上傳前將圖片轉成 WebP |
| 預設啟用壓縮 | 上傳前在瀏覽器本機壓縮圖片 |
| 預設壓縮門檻 | 圖片超過該大小時自動壓縮，單位 MB |
| 預設目標大小 | 壓縮後希望接近的檔案大小，單位 MB |
| 登入頁背景圖 | 使用者登入頁背景圖 |
| 上傳頁背景圖 | 上傳頁背景圖 |
| 頁尾傳送門連結 | 頁尾傳送門按鈕開啟的連結 |
| 隱藏頁尾 | 開啟後隱藏前台頁尾 |

## 管理端設定

| 選項 | 功能 |
| --- | --- |
| 管理端登入頁背景圖 | 管理端登入頁背景圖 |
| 管理端背景圖 | 後台頁面背景圖；可填單張圖片 URL，也可填多個 URL |
| 圖片載入模式 | 管理端圖片列表的預覽載入方式，可選原圖或智慧載入；智慧載入會優先讓公開圖片使用縮圖，受限圖片使用原圖 |
| 縮圖圖源 | 選擇縮圖由哪個服務產生，可選 wsrv.nl、Cloudflare 圖片轉換或 WordPress Photon；選 Cloudflare 圖片轉換前，需要先在 Cloudflare 啟用圖片轉換 |
| Live2D 看板娘 | 開啟後在管理端顯示 Live2D 看板娘 |
| 煙火點擊特效 | 開啟後點擊頁面時顯示煙火效果 |
| 星星拖尾特效 | 開啟後滑鼠移動時顯示星星拖尾效果 |

## 背景圖填寫方式

登入頁背景圖、上傳頁背景圖、管理端登入頁背景圖支援：

| 寫法 | 效果 |
| --- | --- |
| `bing` | 使用 Bing 桌布輪播 |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | 多張圖片輪播 |
| `["https://example.com/1.jpg"]` | 單張背景圖 |
| `["https://your-domain.com/random?..."]` | 隨機圖 API 連結；也可以先在「其他設定」裡設定自己的隨機圖 API，再把呼叫示範產生的隨機圖連結用單張背景圖格式填到這裡 |

管理端背景圖支援圖片 URL；多個 URL 可依頁面提示用英文逗號分隔。留空時使用預設背景。

## 預設命名方式

| 方式 | 產生效果 |
| --- | --- |
| 預設 | 時間隨機前綴 + 原檔名，例如 `1760000000000_cat.png` |
| 僅前綴 | 只使用時間隨機前綴和副檔名，例如 `1760000000000.png` |
| 僅原名 | 保留原檔名，例如 `cat.png`；重名時自動加 `(1)`、`(2)` |
| 短連結 | 使用 8 位短 ID 和副檔名，例如 `a1b2c3d4.png` |
