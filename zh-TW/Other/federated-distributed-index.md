# 聯盟分散式索引使用說明

聯盟分散式索引用來讓多個 ImgBed 站台互相共享檔案列表。

簡單來說：

- 你可以把自己的某些資料夾分享給別人。
- 你也可以加入別人的節點，把對方分享的檔案列表同步到自己的後台查看。
- 聯盟檔案主要用來瀏覽、搜尋和開啟連結，不會在本機重新上傳一份檔案。

## 入口位置

```text
系統設定 -> 其他設定 -> 聯盟分散式索引
```

![聯盟分散式索引本機節點](../../image/other/联盟图/联盟分布式索引本地节点.png)

頁面分成三個分頁：

| 分頁 | 用途 |
| --- | --- |
| 本機節點 | 設定自己的聯盟開關、對外網域、共享資料夾，並更新索引 |
| 我加入的節點 | 管理自己加入的其他 ImgBed 節點 |
| 加入我的節點 | 管理別人申請加入你的節點 |

## 第一次使用怎麼設定

1. 打開「本機節點」。
2. 開啟「啟用」。
3. 在「同步資料夾」裡選擇要分享出去的資料夾。
4. 點「更新出站索引」。
5. 如果系統提示偵測到網域變更，確認目前網域沒問題後繼續。

同步資料夾可以選多個。

如果同步資料夾清空，代表分享全部資料夾。

## 本機節點怎麼用

### 對外網域

對外網域是別人訪問你這個節點時使用的站台網址。

這個網址由系統自動辨識，不需要手動填。第一次更新索引時，系統會讓你確認目前訪問網址是不是正式網域。

如果之後換網域，再點更新索引時也會再次提示確認。

### 同步資料夾

同步資料夾決定哪些檔案會分享給聯盟節點。

例如你只選：

```text
/1/
/2/
```

別人同步你的節點時，就只能看到這兩個目錄裡的檔案。

### 更新出站索引

這個按鈕用來更新「我分享給別人看的檔案列表」。

適合在這些時候點：

- 第一次啟用聯盟。
- 新上傳了一批想分享的檔案。
- 修改了同步資料夾。
- 更換對外網域後需要確認新網域。

## 我加入的節點怎麼用

「我加入的節點」是你訂閱別人節點的地方。

![我加入的節點](../../image/other/联盟图/我加入的节点.png)

### 申請加入別人

1. 跟對方要一條邀請連結。
2. 貼到輸入框。
3. 點「申請加入」。
4. 等對方在自己的後台同意。

對方同意後，節點狀態會變成「已同意」。

### 更新入站索引

更新入站索引用來把你已加入節點的檔案列表同步到本機。

適合在這些時候點：

- 對方剛同意你的申請後。
- 對方告訴你他更新了共享內容。
- 你想刷新全部已加入節點的聯盟檔案列表。

如果只想更新某一個節點，可以在該節點卡片上點「更新索引」。

![更新索引](../../image/other/联盟图/更新索引.png)

### 取消訂閱

如果不想繼續同步某個節點，可以點「取消訂閱」。

取消後，這個節點的聯盟索引也會從本機清掉。

## 加入我的節點怎麼用

「加入我的節點」是別人申請加入你的地方。

![加入我的節點](../../image/other/联盟图/加入我的节点.png)

### 產生邀請連結

1. 先確認本機節點已啟用。
2. 先點過「更新出站索引」，讓系統確認對外網域。
3. 打開「加入我的節點」。
4. 點「重設邀請連結」。
5. 複製邀請連結傳給對方。

如果邀請連結是空的，通常是還沒確認對外網域。回到「本機節點」點一次「更新出站索引」即可。

### 處理加入申請

別人送出申請後，會出現在「加入我的節點」列表。

| 操作 | 作用 |
| --- | --- |
| 同意 | 允許對方同步你分享的檔案列表 |
| 拒絕 | 不允許對方加入 |
| 刪除 | 刪除已結束的紀錄 |
| 檢查狀態 | 檢查對方是否仍保留這段關係 |

同意後，對方還需要在自己那邊點「更新入站索引」，才會看到你分享的檔案列表。

![邀請節點同意](../../image/other/联盟图/邀请节点同意.png)

## 留言功能

關係已同意後，可以在節點卡片裡點「留言」。

留言只用於聯盟關係溝通，不會影響檔案、標籤、目錄和存取權限。

![留言功能](../../image/other/联盟图/留言功能.png)

## 在哪裡看聯盟檔案

同步完成後，回到管理後台檔案列表。

頁面頂部可以在本機檔案和聯盟檔案之間切換。切到聯盟檔案後，就能瀏覽已同步的聯盟內容。

聯盟檔案主要用於查看、搜尋、預覽和複製連結。它不是本機檔案，所以不能在本機直接移動、刪除、改標籤或備份。

![聯盟管理顯示效果](../../image/other/联盟图/联盟管理显示效果图.png)

## 常見問題

### 為什麼提示沒有關係紀錄，需要重新申請？

通常是對方把你刪除並清掉紀錄，導致系統查不到這段關係。這時需要重新申請。

![無關係紀錄重新申請](../../image/other/联盟图/无关系记录重新申请.png)

### 為什麼加入後看不到檔案？

先確認：

1. 對方已同意你的申請。
2. 對方已點過「更新出站索引」。
3. 你已點過「更新入站索引」。
4. 對方的同步資料夾確實包含要分享的目錄。

### 提示網域變更怎麼辦？

如果你現在開後台用的就是正式網域，可以確認繼續。

如果只是臨時用其他網址進後台，建議取消，改用正式網域重新打開後台再操作。

### 同步資料夾清空是什麼意思？

同步資料夾清空代表分享全部資料夾。

如果只想分享部分目錄，請手動選擇對應資料夾。

### 更新出站和更新入站有什麼差別？

| 按鈕 | 簡單理解 |
| --- | --- |
| 更新出站索引 | 更新「別人能從我這裡同步到什麼」 |
| 更新入站索引 | 更新「我從別人那裡同步到了什麼」 |
