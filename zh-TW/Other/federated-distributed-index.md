# 聯盟分散式索引

聯盟分散式索引用來讓多個 ImgBed 網站彼此分享檔案清單。

簡單來說：

- 你可以將自己網站中的指定資料夾分享給他人。
- 你也可以加入其他節點，把對方分享的檔案清單同步到自己的管理面板。
- 聯盟檔案主要用於瀏覽、搜尋和開啟連結，不會重新上傳到你自己的儲存空間。

## 在哪裡設定

開啟：

```text
System Settings -> Other Settings -> Federated Distributed Index
```

![本機聯盟節點](../../image/other/联盟图/联盟分布式索引本地节点.png)

頁面有三個分頁：

| 分頁 | 用途 |
| --- | --- |
| 本機節點 | 啟用自己的節點、確認公開網域、選擇分享資料夾並更新出站索引 |
| 我加入的節點 | 管理你已加入的其他 ImgBed 節點 |
| 加入我的節點 | 管理其他人想加入你節點的申請 |

## 第一次設定

1. 開啟 `Local Node`。
2. 開啟 `Enable`。
3. 在 `Sync folders` 下選擇要分享的資料夾。
4. 點選 `Update Outbound Index`。
5. 如果 ImgBed 偵測到網域變更，請先確認目前網域正確再繼續。

你可以選擇多個同步資料夾。

如果同步資料夾清單為空，表示分享所有資料夾。

## 本機節點

### 公開網域

公開網域是其他節點用來存取你節點的網站 URL。

ImgBed 會自動偵測，不需要手動輸入。第一次更新索引時，ImgBed 會要求你確認目前存取 URL 是否為正式網域。

如果之後更換網域，再更新索引時也會再次要求確認。

### 同步資料夾

同步資料夾決定哪些檔案會分享給聯盟節點。

例如，如果你只選擇：

```text
/1/
/2/
```

其他節點只能看到這兩個目錄中的檔案。

### 更新出站索引

此操作會更新其他節點可從你這裡同步的檔案清單。

適合在以下情況使用：

- 第一次啟用聯盟功能。
- 上傳了想分享的檔案。
- 變更同步資料夾。
- 變更公開網域且需要確認。

## 我加入的節點

`Nodes I Joined` 是你訂閱其他節點的地方。

![我加入的節點](../../image/other/联盟图/我加入的节点.png)

### 申請加入其他節點

1. 向對方擁有者索取邀請連結。
2. 將連結貼到輸入框。
3. 點選 `Request to Join`。
4. 等待對方擁有者在其管理面板中核准。

核准後，節點狀態會變成已核准。

### 更新入站索引

`Update Inbound Index` 會同步你已加入節點的檔案清單。

適合在以下情況使用：

- 對方擁有者剛核准你的申請。
- 對方擁有者通知你分享內容已更新。
- 你想重新整理所有已加入節點的聯盟檔案清單。

若只想更新某一個節點，請在該節點卡片上點選 `Update Index`。

![更新索引](../../image/other/联盟图/更新索引.png)

### 取消訂閱

如果不想再同步某個節點，請點選 `Unsubscribe`。

取消訂閱後，該節點的聯盟索引會從你的本機網站移除。

## 加入我的節點

`Nodes Joining Me` 是你處理他人申請的地方。

![加入我的節點](../../image/other/联盟图/加入我的节点.png)

### 產生邀請連結

1. 確認本機節點已啟用。
2. 至少點選一次 `Update Outbound Index`，讓 ImgBed 確認公開網域。
3. 開啟 `Nodes Joining Me`。
4. 點選 `Reset Invitation Link`。
5. 複製邀請連結並傳給對方擁有者。

如果邀請連結為空，通常表示公開網域尚未確認。請回到 `Local Node`，點選 `Update Outbound Index`。

### 處理加入申請

當有人提交申請時，會出現在 `Nodes Joining Me` 清單中。

| 操作 | 意義 |
| --- | --- |
| 核准 | 允許對方節點同步你的分享檔案清單 |
| 拒絕 | 拒絕加入申請 |
| 刪除 | 移除已完成的記錄 |
| 檢查狀態 | 檢查對方是否仍保留這段關係 |

核准後，對方仍需要點選 `Update Inbound Index`，才能在那邊看到你的分享檔案。

![核准受邀節點](../../image/other/联盟图/邀请节点同意.png)

## 訊息

關係核准後，可在節點卡片上點選 `Message`。

訊息僅用於聯盟關係溝通，不會變更檔案、標籤、目錄或權限。

![訊息](../../image/other/联盟图/留言功能.png)

## 查看聯盟檔案

同步完成後，回到管理面板的檔案清單。

在頁面頂部可在本機檔案和聯盟檔案之間切換。切到聯盟檔案後，即可瀏覽已同步的內容。

聯盟檔案主要用於查看、搜尋、預覽和複製連結。它們不是本機檔案，因此無法在你自己的站點中移動、刪除、重新標籤或備份。

![管理面板中的聯盟檔案](../../image/other/联盟图/联盟管理显示效果图.png)

## 常見問題

### 為什麼提示沒有關係記錄，需要重新申請？

這通常表示對方已將你刪除並移除記錄，因此無法再找到這段關係。請提交新的加入申請。

![沒有關係記錄時重新申請](../../image/other/联盟图/无关系记录重新申请.png)

### 為什麼加入後看不到檔案？

請確認：

1. 對方擁有者已核准你的申請。
2. 對方擁有者已點選 `Update Outbound Index`。
3. 你已點選 `Update Inbound Index`。
4. 對方擁有者的同步資料夾包含其想分享的目錄。

### 偵測到網域變更時該怎麼辦？

如果你目前是透過正式網域開啟管理面板，請確認並繼續。

如果你使用臨時位址，請取消，改用正式網域重新開啟管理面板後再試一次。

### 同步資料夾清單為空代表什麼？

同步資料夾清單為空表示分享所有資料夾。

如果只想分享部分目錄，請手動選擇那些資料夾。

### 出站與入站索引更新有什麼不同？

| 按鈕 | 簡單說明 |
| --- | --- |
| 更新出站索引 | 更新其他人能從我這裡同步的內容 |
| 更新入站索引 | 更新我已從其他人同步的內容 |
