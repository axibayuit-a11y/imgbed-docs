# 磁力連結使用說明

磁力連結功能可以把磁力連結中的檔案下載下來，再自動上傳到你指定的雲端渠道。

它適合用來轉存動畫、影片、壓縮檔等內容。你只要貼上磁力連結，系統會在背景建立下載任務；下載完成後會上傳到 ImgBed，並在上傳列表顯示最終連結。

![磁力連結](../../image/other/磁力链接/磁力链接.png)

## 入口位置

磁力連結入口在首頁上傳區。

把磁力連結貼到輸入框，模式選「轉存」，再按上傳。

![上傳動畫](../../image/other/磁力链接/上传番剧.png)

## 第一次使用前

使用前需要先在後台設定磁力轉存。

通常要準備：

1. GitHub 帳號，用來執行下載任務。
2. 雲端上傳渠道，例如 Google Drive、OneDrive。
3. 要上傳到的目錄。
4. 任務逾時時間。

設定完成後，回首頁貼上磁力連結就可以開始轉存。

## 怎麼上傳磁力連結

1. 在首頁上傳框貼上磁力連結。
2. 確認模式選「轉存」。
3. 點上傳。
4. 等系統建立磁力任務。
5. 任務開始後，可在右下角「磁力任務」浮窗查看進度。

下載和上傳可能需要一些時間，速度取決於資源熱度、GitHub 執行環境和選擇的雲端渠道。

![磁力連結下載中](../../image/other/磁力链接/磁力链接下载中.png)

## 完成後

任務完成後，上傳列表會顯示檔名和連結。

如果是影片會顯示影片預覽；圖片會顯示圖片預覽；其他檔案會顯示一般檔案圖示。

![下載完成的影片](../../image/other/磁力链接/下载好后的视频.png)

可以直接複製：

| 連結類型 | 用途 |
| --- | --- |
| 原始連結 | 直接開啟檔案 |
| Markdown | 貼到 Markdown 文章或說明 |
| HTML | 貼到網頁程式碼 |
| BBCode | 貼到支援 BBCode 的論壇 |

## 磁力任務浮窗

右下角浮窗會顯示目前任務數量、任務名稱、進度和完成狀態。

| 狀態 | 說明 |
| --- | --- |
| 等待中 | 任務已建立，等待執行 |
| 下載中 | 正在下載磁力資源 |
| 上傳中 | 檔案已下載好，正在上傳雲端 |
| 完成 | 檔案已上傳成功，可以複製連結 |
| 失敗 | 任務沒有成功完成，需要依提示排查 |

## 使用建議

- 磁力連結內如果有多個檔案，系統會優先顯示主要完成檔案。
- 大檔案需要較長時間，建議等任務完成後再重新整理頁面。
- 如果資源本身沒有速度，任務可能很慢或失敗。
- 如果雲端帳號額度不足、授權失效或目錄設定錯誤，任務也可能失敗。
- 影片上傳完成後，頁面預覽可能需要等幾秒載入。

## 常見問題

### 貼上磁力連結後沒有開始

先確認後台已啟用磁力轉存，並選了可用的 GitHub 帳號和雲端渠道。

### 一直下載很慢

磁力速度取決於資源本身。如果沒有可用節點，下載會很慢，甚至無法完成。

### 上傳完成後看不到預覽

先確認檔案連結可以開啟。影片有時需要瀏覽器載入一段時間，也可以直接點連結查看。

### 任務失敗怎麼辦

檢查磁力連結是否有效、雲端渠道是否可用、上傳目錄是否正確，然後重新提交任務。
