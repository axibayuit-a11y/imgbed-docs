# IP 定位與使用者管理

IP 定位會把上傳者、登入裝置等紀錄裡的 IP 顯示成大概地理位置。

啟用後，後台可以更直觀看到訪問來源、上傳來源，也可以在使用者管理裡針對異常 IP 單獨禁止或恢復上傳。

## 入口位置

```text
系統設定 -> 其他設定 -> IP 定位
```

![IP 定位](../../image/other/ip定位/ip定位.png)

## 可以設定什麼

新版 IP 定位支援多個來源，不再只依賴單一地圖服務。

| 設定項 | 作用 |
| --- | --- |
| IP 解析語言 | 選擇地址顯示語言，例如 English、繁體中文、日本語、Français |
| MaxMind Account ID | MaxMind 帳號 ID，用於 MaxMind GeoLite Web Service |
| MaxMind License Key | MaxMind License Key |
| 騰訊地圖 Key | 騰訊位置服務 Key，適合中文地址和中國大陸 IP |
| ipapi Key | APILayer ipapi Key，支援多語 IP 地址解析 |

這些設定可以依需求填，不需要全部都填。

如果什麼 Key 都不填，系統也會嘗試使用內建免費來源解析 IP，只是穩定性、語言支援和精準度可能不如自己設定的服務。

## 推薦怎麼選

如果你主要看中文地址：

1. IP 解析語言選「繁體中文」或需要的中文語系。
2. 有需要時設定騰訊地圖 Key。
3. 也可以再設定 MaxMind 或 ipapi 作為補充。

如果你主要看英文或多語地址：

1. IP 解析語言選需要的語言。
2. 設定 MaxMind Account ID 和 License Key。
3. 需要更好的多語結果時，再設定 ipapi Key。

## MaxMind 怎麼設定

MaxMind 需要填兩項：

```text
MaxMind Account ID
MaxMind License Key
```

在 MaxMind 後台可以看到 Account ID，並在 License Keys 頁面產生 License Key。

![MaxMind Key 設定](../../image/other/ip定位/maxmind的key配置.png)

產生後，把 Account ID 和 License Key 分別填回 ImgBed 後台儲存。

MaxMind 免費方案適合日常使用，但有呼叫次數限制。如果超過額度，系統會繼續嘗試其他可用來源。

## ipapi 怎麼設定

ipapi 使用 APILayer 的 API Key。

進入 ipapi 控制台後，複製頁面上的 API Key。

![ipapi 設定](../../image/other/ip定位/ipapi配置.png)

然後填入 ImgBed 後台的 `ipapi Key` 並儲存。

ipapi 支援多語 IP 地址解析，適合想讓後台地址依指定語言顯示的情境。免費方案也有次數限制，用完後系統會繼續嘗試其他可用來源。

## 騰訊地圖 Key 怎麼設定

騰訊地圖 Key 適合中文地址，尤其是中國大陸 IP。

在騰訊位置服務控制台建立 Key 時，建議啟用：

```text
WebServiceAPI
```

建立完成後，把 Key 填到「騰訊地圖 Key」裡儲存。

## 使用者管理看什麼

入口在後台頂部的使用者管理。

![使用者管理顯示](../../image/other/用户管理显示.png)

使用者管理主要看每個 IP 的上傳狀況：

| 欄位 | 說明 |
| --- | --- |
| IP 來源 | 上傳者來源 IP |
| 地址 | IP 定位解析出的大概位置 |
| 上傳總量 | 這個 IP 上傳檔案占用的總大小 |
| 上傳次數 | 這個 IP 上傳過幾次 |
| 允許上傳 | 開啟代表允許，關閉代表禁止上傳 |

點左側展開箭頭，可以查看這個 IP 上傳過的檔案列表。

檔案列表裡可以看到檔名、預覽、大小、審查結果、檔案狀態和上傳時間。遇到異常上傳時，可以先展開看具體檔案，再決定是否禁止這個 IP 繼續上傳。

## 搜尋、排序和進階篩選

使用者管理頂部可以搜尋 IP 來源或地址。

排序可依時間、上傳次數、上傳總量切換，方便先看最近上傳、上傳次數最多，或占用空間最大的 IP。

需要更精準排查時，可以打開進階篩選。

![使用者管理進階篩選](../../image/other/用户管理高级筛选.png)

進階篩選支援：

| 篩選項 | 用法 |
| --- | --- |
| 時間範圍 | 只看某段時間內上傳過檔案的 IP |
| 存取狀態 | 依正常、封鎖等狀態篩選 |
| 黑白名單 | 依白名單、黑名單、未設定篩選 |
| 檔案類型 | 只看上傳過圖片、影片、音訊、文件、程式碼或其他類型的 IP |
| 檔案大小 | 依上傳檔案大小範圍篩選 |
| 年齡分級 | 依未設定、全年齡、R12+、R16+、R18 等分級篩選 |
| 檔案狀態 | 依檔案目前狀態篩選，方便排查異常檔案 |

篩選完點「套用篩選」即可，需要恢復全部資料時點「重設」。

## 手機版顯示

手機版會把使用者管理改成卡片顯示。

![手機版使用者管理](../../image/other/手机端显示用户管理效果.png)

每張卡片會顯示 IP、地址、上傳總量、上傳次數和允許上傳開關，不需要橫向拖表格也能操作。

## 地址不準怎麼辦

IP 定位顯示的是大概位置，不是精準門牌地址。

如果使用者使用代理、機房、雲端伺服器或跨境網路，顯示位置可能和真實位置不同。

這個功能比較適合判斷大致來源、發現異常上傳、輔助封鎖，不建議當成精確定位工具。

## 常見情況

| 情況 | 說明 |
| --- | --- |
| 地址顯示空白 | 可能還沒解析成功，或目前 IP 來源暫時不可用 |
| 地址語言不對 | 檢查 IP 解析語言，並確認已設定支援該語言的來源 |
| 地址顯示成機房 | 很多代理、雲端伺服器、爬蟲工具都會顯示成機房或 ISP 地址 |
| 上傳次數很高 | 可以重點檢查這個 IP，必要時關閉允許上傳 |
| 上傳總量很大 | 可排序或篩選後展開查看具體檔案 |
| 禁止後想恢復 | 把「允許上傳」重新打開即可 |

## 快速流程

```text
打開其他設定裡的 IP 定位
-> 選 IP 解析語言
-> 依需求填 MaxMind、騰訊地圖或 ipapi
-> 儲存
-> 打開使用者管理查看 IP 來源、地址、上傳總量和次數
-> 用搜尋、排序或進階篩選排查異常 IP
-> 依需要允許或禁止上傳
```
