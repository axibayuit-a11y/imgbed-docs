# ImgBed

أداة شاملة لاستضافة المحتوى الشخصي مبنية على منظومة Cloudflare.

بدأ ImgBed من رفع الصور وإدارة الروابط، ثم أصبح الآن حلا متكاملا لاستضافة المحتوى الشخصي. يمكنك استخدامه من أجل:

- رفع الملفات إلى أي قناة تخزين تفضلها
- مشاركة الصور والملفات بأي صيغة بشكل عام
- إنشاء مدونة شخصية بسرعة خلال دقيقة
- حفظ الملفات من روابط magnet بنقرة واحدة
- الانضمام إلى اتحاد عقد ImgBed ومشاركة فهارس الملفات المفتوحة

كانت فكرتنا منذ البداية أن يمتلك كل شخص موقعه الخاص لاستضافة الصور ومشاركة الملفات. أيًا كان بلدك أو لغتك، وسواء كنت تريد فقط حفظ الصور، أو تنظيم الألبومات، أو مشاركة الملفات مع الأصدقاء، أو بناء مساحة استضافة مستقرة لمدونتك أو محتواك الإبداعي أو مواردك العامة، فإن ImgBed يسعى إلى خفض العتبة قدر الإمكان، وجعل هذه التجربة أكثر حرية ووضوحًا وأسهل في البدء.

## قدرات قوية جاهزة مباشرة

- 🌍 دعم 28 لغة للواجهة، حتى يتمكن المستخدمون حول العالم من استخدامه بسهولة
- 📦 13 قناة رفع رئيسية مدمجة: Telegram و Cloudflare R2 و S3 و WebDAV و Discord و Hugging Face و GitHub Releases و GitLab Packages و OneDrive و Google Drive و Dropbox و Yandex و pCloud
- 🧩 تصميم معياري بالكامل، يمكنك تفعيل كل ميزة عند الحاجة فقط دون إهدار الموارد
- 🔐 يدعم API Token وأوضاع الوصول وقواعد القوائم لإدارة أذونات المحتوى بدقة

ما زال هذا المشروع يتحسن باستمرار. قدراتنا محدودة، لكننا سنبذل كل ما نستطيع لجعل كل ميزة واضحة ومستقرة وسهلة الاستخدام. إذا ساعدك ImgBed على امتلاك مساحة صور أو مدونة أو مساحة مشاركة ملفات خاصة بك بسهولة أكبر، فسنكون سعداء جدا.
