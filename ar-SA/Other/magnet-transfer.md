# نقل Magnet

يقوم Magnet transfer بتنزيل الملفات من magnet link ثم رفعها تلقائيًا إلى قناة التخزين السحابي التي تختارها.

يفيد في نقل حلقات الأنمي، والفيديوهات، والأرشيفات، والملفات المشابهة. الصق magnet link، وسيُنشئ ImgBed مهمة تنزيل في الخلفية. بعد انتهاء التنزيل يُرفع الملف إلى ImgBed ويظهر الرابط النهائي في قائمة الرفع.

![Magnet transfer](../../image/other/磁力链接/磁力链接.png)

## أين تستخدمه؟

مدخل magnet transfer موجود في منطقة الرفع في الصفحة الرئيسية.

الصق magnet link في مربع الإدخال، واختر `Transfer`، ثم ارفع.

![رفع أنمي](../../image/other/磁力链接/上传番剧.png)

## قبل أول استخدام

اضبط magnet transfer من لوحة الإدارة أولًا.

عادة تحتاج إلى:

1. حساب GitHub لتشغيل مهمة التنزيل.
2. قناة رفع سحابية، مثل Google Drive أو OneDrive.
3. مجلد الرفع الهدف.
4. مهلة زمنية للمهمة.

بعد جاهزية الإعدادات، ارجع إلى الصفحة الرئيسية والصق magnet link لبدء النقل.

## رفع Magnet Link

1. الصق magnet link في مربع الرفع في الصفحة الرئيسية.
2. تأكد أن الوضع مضبوط على `Transfer`.
3. اضغط upload.
4. انتظر حتى ينشئ ImgBed مهمة magnet.
5. بعد بدء المهمة، استخدم لوحة `Magnet Tasks` العائمة أسفل اليمين لمتابعة التقدم.

قد يستغرق التنزيل والرفع وقتًا. السرعة تعتمد على مصدر magnet وبيئة GitHub runtime وقناة التخزين المختارة.

![تنزيل magnet](../../image/other/磁力链接/磁力链接下载中.png)

## بعد الاكتمال

بعد اكتمال المهمة تعرض قائمة الرفع اسم الملف والرابط.

الفيديوهات تعرض معاينة فيديو، والصور تعرض معاينة صورة، والملفات الأخرى تعرض أيقونة ملف عادية.

![فيديو تم تنزيله](../../image/other/磁力链接/下载好后的视频.png)

يمكنك نسخ:

| نوع الرابط | الاستخدام |
| --- | --- |
| Original link | وصول مباشر للملف |
| Markdown | مقالات أو ملاحظات Markdown |
| HTML | كود صفحة ويب |
| BBCode | المنتديات التي تدعم BBCode |

## لوحة Magnet Task

تعرض لوحة مهام magnet أسفل اليمين عدد المهام واسم المهمة والتقدم والحالة النهائية.

حالات شائعة:

| الحالة | المعنى |
| --- | --- |
| Waiting | تم إنشاء المهمة وتنتظر التشغيل. |
| Downloading | يتم تنزيل مورد magnet. |
| Uploading | اكتمل التنزيل ويتم رفع الملف إلى التخزين السحابي. |
| Completed | نجح الرفع ويمكن نسخ الرابط. |
| Failed | لم تكتمل المهمة بنجاح. راجع الرسالة وحاول مرة أخرى. |

## نصائح

- إذا كان magnet link يحتوي عدة ملفات، يعطي ImgBed الأولوية لعرض الملف الرئيسي المكتمل.
- الملفات الكبيرة تستغرق وقتًا أطول. انتظر اكتمال المهمة قبل تحديث الصفحة.
- إذا لم يكن لمورد magnet peers متاحون فقد يكون بطيئًا جدًا أو يفشل.
- إذا نفدت حصة الحساب السحابي أو انتهى التفويض أو كان مجلد الرفع خاطئًا، قد تفشل المهمة.
- قد تحتاج معاينة الفيديو إلى ثوانٍ بعد اكتمال الرفع.

## FAQ

### لا يبدأ شيء بعد لصق Magnet Link

تأكد أن magnet transfer مفعّل في لوحة الإدارة، وأن حساب GitHub وقناة السحابة المحددين صالحان.

### التنزيل بطيء دائمًا

سرعة magnet تعتمد على المورد نفسه. إذا لم توجد peers متاحة فقد يكون التنزيل بطيئًا جدًا أو غير ممكن.

### لا تظهر معاينة بعد الرفع

تأكد أولًا أن رابط الملف يفتح. قد تحتاج ملفات الفيديو إلى وقت قصير للتحميل في المتصفح، أو يمكنك فتح الرابط مباشرة.

### ماذا أفحص إذا فشلت المهمة؟

تحقق أن magnet link صالح، وأن القناة السحابية تعمل، وأن مجلد الرفع صحيح. ثم أرسل المهمة مرة أخرى.
