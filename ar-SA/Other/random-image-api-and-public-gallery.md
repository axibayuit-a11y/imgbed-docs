# واجهة API للصور العشوائية والمعرض العام

تُضبط الميزتان ضمن:

```text
System Settings -> Other Settings
```

## واجهة API للصور العشوائية

تعيد واجهة API للصور العشوائية ملفًا عشوائيًا واحدًا من الأدلة المحددة. تفيد لخلفيات المواقع، أو تدوير الصور الرمزية، أو استدعاءات الصور العشوائية من صفحات خارجية.

بعد تفعيلها، استخدم:

```text
https://your-domain.com/random
```

## إعدادات واجهة API للصور العشوائية

| الخيار | الغرض |
| --- | --- |
| تمكين | يشغّل نقطة النهاية `/random` أو يوقفها. عند تعطيلها، يُمنع الوصول. |
| الأدلة | يحدد الأدلة التي يمكن للواجهة العشوائية استخدامها. لا تستطيع الواجهة استخدام الأدلة غير المدرجة هنا. |
| عرض تجربة الاستدعاء | ينشئ روابط واجهة API للصور العشوائية التي يمكنك نسخها مباشرة. |

يمكنك اختيار عدة أدلة. على سبيل المثال، إذا كان المسموح فقط هو `/landscape/` و`/portrait/`، فلا تستطيع واجهة API للصور العشوائية اختيار الملفات إلا من هذين الدليلين وأدلتهما الفرعية.

## معاملات واجهة API للصور العشوائية

| المعامل | مثال | الغرض |
| --- | --- | --- |
| `dir` | `/landscape/` | يحدد الدليل العشوائي. |
| `content` | `image` | يحدد نوع الوسائط. استخدم `image` أو `video` أو `audio` أو تركيبات مفصولة بفواصل. |
| `orientation` | `auto` | يرشح اتجاه الصورة. استخدم `portrait` أو `landscape` أو `auto`. |
| `type` | `url` | صيغة الإرجاع. القيمة الفارغة تعني إعادة توجيه، و`url` يعيد عنوان URL نصيًا، و`json` يعيد JSON. |
| `origin` | `1` | يُستخدم مع `type=url` لإرجاع عنوان URL كامل. |
| `age` | `all-ages,r12` | يرشح حسب التصنيف العمري. |
| `tag` | `wallpaper,sky` | لا يعيد إلا الملفات التي تحتوي على هذه الوسوم. |
| `ex` | `private` | يستبعد الملفات التي تحتوي على هذه الوسوم. |

## صيغ الإرجاع

من دون `type`، تعيد API التوجيه مباشرة إلى عنوان URL للملف العشوائي.

مع `type=url`، تعيد عنوان URL نصيًا.

مع `type=json`، تعيد معلومات الملف، بما في ذلك عنوان URL للملف، ومعرّف الملف، واسم الملف، ونوع الملف، والوسوم، والتصنيف، والبيانات الوصفية ذات الصلة.

## قواعد الوصول

تتبع واجهة API للصور العشوائية قواعد الوصول العام:

| القاعدة | التأثير |
| --- | --- |
| تقييد الأدلة | لا يمكن اختيار إلا الملفات الموجودة في الأدلة المسموح بها. |
| قائمة الحظر | تُستبعد الملفات المحظورة من مجموعة الاختيار العشوائي. |
| وضع قائمة السماح | عند تفعيله، لا تُعاد إلا الملفات المسموح بها للوصول العام. |
| التصنيف العمري | تُرشح محتويات R12 وR16 وR18 وما شابهها حسب وضع الوصول الحالي. |

إذا لم يطابق أي ملف بعد التصفية، تعيد API نتيجة تفيد بعدم وجود تطابق.

## ذاكرة التخزين المؤقت

تخزن واجهة API للصور العشوائية مجموعات الملفات المرشحة للأدلة في ذاكرة التخزين المؤقت لتحسين السرعة.

بعد تغيّر الملفات، يحدّث ImgBed إصدار ذاكرة التخزين المؤقت للدليل، ثم تعيد الطلبات اللاحقة بناء مجموعة المرشحين. تُخزن الأدلة الفارغة مؤقتًا لفترة قصيرة لتجنب الاستعلامات المتكررة.

## المعرض العام

يوفر المعرض العام صفحة تصفح عامة للقراءة فقط للأدلة التي تسمح للزوار برؤيتها.

بعد تفعيله، يستطيع الزوار فتح:

```text
https://your-domain.com/browse/directory-name
```

## إعدادات المعرض العام

| الخيار | الغرض |
| --- | --- |
| تمكين | يشغّل المعرض العام أو يوقفه. عند تعطيله، لا يستطيع الزوار تصفحه. |
| وضع تحميل الصور | يتحكم فيما إذا كانت المعاينات تستخدم الصور الأصلية أو الصور المصغرة. |
| الأدلة المفتوحة | يحدد الأدلة التي يستطيع الزوار الوصول إليها. |

## وضع تحميل الصور

| الوضع | الغرض |
| --- | --- |
| الأصلي | تحمّل صفحة الزائر الملفات الأصلية مباشرة. |
| الصورة المصغرة | تفضّل صفحة الزائر الصور المصغرة للتحميل بشكل أسرع. |

## الأدلة المفتوحة

تحدد الأدلة المفتوحة ما يستطيع الزوار رؤيته.

على سبيل المثال:

```text
/1/,/2/,/landscape/,/portrait/
```

يمكن للزوار بعد ذلك الوصول إلى:

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

يمكن أيضًا فتح أدلة فرعية، مثل `/2026/lucky/`. ويُمنع الزوار من الوصول إلى الأدلة غير المفتوحة.

## ميزات المعرض العام

| الميزة | الوصف |
| --- | --- |
| تصفح الأدلة | عرض الملفات والأدلة الفرعية داخل الأدلة المفتوحة. |
| البحث | البحث حسب اسم الملف، أو معرّف الملف، أو الوسوم. |
| مرشح النوع | ترشيح الصور، أو مقاطع الفيديو، أو الصوت، أو الملفات الأخرى. |
| مرشح الوسوم | تضمين وسوم محددة أو استبعادها. |
| مرشح الاتجاه | ترشيح الصور الأفقية أو العمودية. |
| مرشح الوقت | التصفية حسب نطاق وقت الرفع. |
| مرشح الامتداد | التصفية حسب امتداد الملف. |
| نسخ الرابط | نسخ روابط الوصول إلى الملفات. |
| معاينة الوسائط | عرض الصور ومقاطع الفيديو والصوت أو تشغيلها على صفحة الزائر. |

## قواعد وصول المعرض العام

يتبع المعرض العام أيضًا قواعد الوصول العام:

| القاعدة | التأثير |
| --- | --- |
| الأدلة المفتوحة | لا تظهر إلا الأدلة المسموح بها. |
| وضع الوصول | يُرشح المحتوى حسب وضع الوصول الحالي للتصنيف العمري. |
| وضع قائمة السماح | عند تفعيله، لا تظهر إلا الملفات المسموح بها للوصول العام. |
| قائمة الحظر | تُخفى الملفات المحظورة. |
