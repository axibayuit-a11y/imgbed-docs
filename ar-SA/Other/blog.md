# المدونة

تضيف ميزة المدونة صفحة Blog مستقلة إلى موقع ImgBed.

بعد تفعيلها يستطيع الزوار فتح:

```text
https://your-domain.com/blog/
```

![الصفحة الرئيسية للمدونة](../../image/other/博客/博客首页.png)

المدونة مبنية بتكييف من مشروع [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki) المفتوح المصدر. يعيد ImgBed دمجها مع Vue لتعمل كجزء من موقع استضافة الصور.

## أين تضبطها؟

إعدادات المدونة موجودة ضمن:

```text
System Settings -> Other Settings -> Blog
```

![إعدادات المدونة](../../image/other/博客/QQ20260611-221702.png)

## الإعداد لأول مرة

1. فعّل `Enable`.
2. اختر حساب GitHub الذي سيحفظ إعدادات المدونة.
3. اضغط `Update Blog`.
4. انتظر رسالة النجاح.
5. افتح `https://your-domain.com/blog/` لمشاهدة المدونة.

في أول استخدام، يجهز ImgBed مستودع GitHub خاصًا تحت الحساب المختار:

```text
imgbed-blog-config
```

يحفظ هذا المستودع إعدادات المدونة ومحتوى المقالات.

## كتابة المقالات

عدّل مقالات المدونة داخل مستودع GitHub الخاص:

```text
imgbed-blog-config
```

المسار المعتاد:

1. افتح GitHub.
2. ادخل إلى مستودع `imgbed-blog-config` الخاص.
3. عدّل ملفات المقالات أو أضف ملفات جديدة.
4. اعمل commit للتغييرات.
5. ارجع إلى لوحة إدارة ImgBed واضغط `Update Blog`، أو اضغط شعار أعلى يسار صفحة المدونة الرئيسية ثلاث مرات لتشغيل تحديث المدونة.

`Update Blog` لا يكتب فوق المحتوى الذي كتبته. يهيئ المستودع عند الحاجة ويحدّث كاش المدونة.

## الميزات المدعومة

تدعم المدونة ميزات مألوفة مثل قائمة المقالات، والتصنيفات، والوسوم، والأرشيف، والبحث، والوضع الداكن، وتغيير اللغة.

كما تدعم التعليقات وإحصاءات الزيارات.

![تعليقات المدونة](../../image/other/博客/支持留言.png)

تظهر التعليقات أسفل المقالات. يمكن للزوار إدخال avatar و nickname و email ومحتوى التعليق.

تعرض إحصاءات الزيارات مشاهدات المقالات وزيارات الموقع، ما يساعدك على فهم حركة المدونة.

## الرابط

تُقدّم المدونة دائمًا تحت `/blog/`.

إذا كان نطاق ImgBed هو:

```text
https://image.example.com
```

فسيكون رابط المدونة:

```text
https://image.example.com/blog/
```

بعد تعطيل المدونة لن يستطيع الزوار الوصول إلى صفحة المدونة.
