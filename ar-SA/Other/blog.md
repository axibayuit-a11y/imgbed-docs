# المدونة

تضيف ميزة المدونة صفحة مدونة مستقلة إلى موقع ImgBed.

بعد تفعيلها، يستطيع الزوار فتح:

```text
https://your-domain.com/blog/
```

![الصفحة الرئيسية للمدونة](../../image/other/博客/博客首页.png)

المدونة مقتبسة ومكيّفة من مشروع [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki) المفتوح المصدر. يعيد ImgBed كتابتها ودمجها مع Vue حتى تعمل بوصفها جزءًا من موقع استضافة الصور.

## أين تضبطها؟

تقع إعدادات المدونة ضمن:

```text
System Settings -> Other Settings -> Blog
```

![إعدادات المدونة](../../image/other/博客/QQ20260611-221702.png)

## الإعداد لأول مرة

1. فعّل `Enable`.
2. اختر حساب GitHub المستخدم لتخزين إعدادات المدونة.
3. انقر `Update Blog`.
4. انتظر رسالة النجاح.
5. افتح `https://your-domain.com/blog/` لعرض المدونة.

عند الاستخدام لأول مرة، يجهز ImgBed مستودع GitHub خاصًا ضمن الحساب المحدد:

```text
imgbed-blog-config
```

يخزن هذا المستودع إعدادات المدونة ومحتوى المقالات.

## كتابة المقالات

عدّل مقالات المدونة في مستودع GitHub الخاص بك:

```text
imgbed-blog-config
```

سير العمل المعتاد:

1. افتح GitHub.
2. ادخل إلى مستودع `imgbed-blog-config` الخاص.
3. عدّل ملفات المقالات أو أضف ملفات جديدة.
4. ثبّت التغييرات في المستودع.
5. ارجع إلى لوحة إدارة ImgBed وانقر `Update Blog`، أو انقر الشعار في الزاوية العلوية اليسرى من الصفحة الرئيسية للمدونة ثلاث مرات لتشغيل تحديث المدونة.

لا يستبدل `Update Blog` المحتوى الذي كتبته. إنه يهيئ المستودع عند الحاجة ويحدث ذاكرة التخزين المؤقت للمدونة.

## الميزات المدعومة

تدعم المدونة ميزات المدونات الشائعة، مثل قوائم المقالات، والتصنيفات، والوسوم، والأرشيف، والبحث، والوضع الداكن، وتبديل اللغة.

كما تدعم التعليقات وإحصاءات الزيارات.

![تعليقات المدونة](../../image/other/博客/支持留言.png)

تظهر التعليقات أسفل المقالات. يستطيع الزوار إرسال صورة رمزية، واسم مستعار، وبريد إلكتروني، ومحتوى التعليق.

تعرض إحصاءات الزيارات مشاهدات المقالات وزيارات الموقع، ما يساعدك على فهم حركة المدونة.

## عنوان الوصول

تُقدَّم المدونة دائمًا ضمن المسار `/blog/`.

على سبيل المثال، إذا كان نطاق ImgBed لديك هو:

```text
https://image.example.com
```

فسيكون عنوان المدونة:

```text
https://image.example.com/blog/
```

بعد تعطيل المدونة، لن يستطيع الزوار الوصول إلى صفحة المدونة.
