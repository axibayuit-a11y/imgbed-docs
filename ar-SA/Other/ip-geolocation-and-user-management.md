# تحديد الموقع الجغرافي عبر IP وإدارة المستخدمين

يحوّل تحديد الموقع الجغرافي عبر IP عناوين IP الموجودة في سجلات الرفع، وأجهزة تسجيل الدخول، والسجلات المشابهة إلى مواقع تقريبية.

بعد ضبطه، تستطيع لوحة الإدارة عرض مصادر الرفع والوصول بوضوح أكبر. تتيح إدارة المستخدمين أيضًا حظر الوصول إلى الرفع أو استعادته لعناوين IP المشبوهة.

## أين تضبطه؟

افتح:

```text
System Settings -> Other Settings -> IP Geolocation
```

![تحديد الموقع الجغرافي عبر IP](../../image/other/ip定位/ip定位.png)

## الإعدادات المتاحة

يدعم مسار تحديد الموقع الجغرافي عبر IP الأحدث عدة مصادر، بدل الاعتماد على خدمة خرائط واحدة.

| الإعداد | الغرض |
| --- | --- |
| لغة تحديد الموقع الجغرافي عبر IP | تختار لغة العرض، مثل الإنجليزية، والصينية المبسطة، واليابانية، والفرنسية، وغيرها. |
| معرّف حساب MaxMind | معرّف حساب MaxMind لخدمة MaxMind GeoLite Web Service. |
| مفتاح ترخيص MaxMind | مفتاح ترخيص MaxMind. |
| مفتاح Tencent Map | مفتاح Tencent Location Service. يفيد للعناوين الصينية وعناوين IP في بر الصين الرئيسي. |
| مفتاح ipapi | مفتاح APILayer ipapi. يدعم تحديد الموقع الجغرافي عبر IP بعدة لغات. |

املأ الخدمات التي تحتاجها فقط. لا يلزم ضبط كل الحقول.

إذا لم يُقدَّم أي مفتاح، فسيظل ImgBed يحاول استخدام مصادر مجانية مدمجة، لكن الاستقرار ودعم اللغة والدقة قد تكون أقل من خدمة تضبطها بنفسك.

## الخيارات الموصى بها

إذا كنت تحتاج العناوين الصينية في المقام الأول:

1. اضبط لغة تحديد الموقع الجغرافي عبر IP على الصينية المبسطة.
2. اضبط مفتاح Tencent Map.
3. أضف MaxMind أو ipapi اختياريًا كمصادر احتياطية.

إذا كنت تحتاج العناوين الإنجليزية أو متعددة اللغات في المقام الأول:

1. اختر اللغة التي تحتاجها.
2. اضبط معرّف حساب MaxMind ومفتاح الترخيص.
3. أضف مفتاح ipapi إذا كنت تحتاج نتائج متعددة اللغات أفضل.

## إعداد MaxMind

يحتاج MaxMind إلى:

```text
MaxMind Account ID
MaxMind License Key
```

ابحث عن معرّف الحساب في لوحة تحكم MaxMind، وأنشئ مفتاح ترخيص من صفحة مفاتيح الترخيص.

![إعداد مفتاح MaxMind](../../image/other/ip定位/maxmind的key配置.png)

بعد إنشائه، الصق معرّف الحساب ومفتاح الترخيص في ImgBed واحفظ.

الخطة المجانية من MaxMind مناسبة للاستخدام اليومي، لكنها تتضمن حدودًا للطلبات. إذا تم تجاوز الحصة، يواصل ImgBed تجربة المصادر الأخرى المتاحة.

## إعداد ipapi

يستخدم ipapi مفتاح APILayer API.

افتح وحدة تحكم ipapi وانسخ مفتاح API المعروض هناك.

![إعداد ipapi](../../image/other/ip定位/ipapi配置.png)

الصقه في حقل `ipapi Key` داخل ImgBed واحفظ.

يدعم ipapi تحديد الموقع الجغرافي عبر IP بعدة لغات، ويفيد عندما تريد عرض العناوين باللغة المحددة. تتضمن خطته المجانية أيضًا حدودًا للطلبات. إذا نفدت الحصة، يواصل ImgBed تجربة المصادر الأخرى المتاحة.

## إعداد مفتاح Tencent Map

يفيد مفتاح Tencent Map للعناوين الصينية، خاصة عناوين IP في بر الصين الرئيسي.

عند إنشاء مفتاح في Tencent Location Service، فعّل:

```text
WebServiceAPI
```

بعد الإنشاء، الصق المفتاح في `Tencent Map Key` واحفظ.

إذا كنت تحتاج فقط إلى تحديد موقع أساسي لعناوين IP الصينية، يكفي مفتاح Tencent Map للبدء.

## ما الذي يجب التحقق منه في إدارة المستخدمين؟

تتوفر إدارة المستخدمين من أعلى لوحة الإدارة.

![إدارة المستخدمين](../../image/other/用户管理显示.png)

تعرض إدارة المستخدمين نشاط الرفع حسب عنوان IP:

| الحقل | الوصف |
| --- | --- |
| مصدر IP | عنوان IP المصدر للرافع. |
| العنوان | الموقع التقريبي المستنتج من عنوان IP. |
| إجمالي حجم الرفع | إجمالي حجم الملفات التي رفعها عنوان IP هذا. |
| عدد مرات الرفع | عدد عمليات الرفع من عنوان IP هذا. |
| السماح بالرفع | يعني التشغيل أن الرفع مسموح. ويعني الإيقاف أن الرفع محظور. |

انقر السهم الموجود على اليسار لتوسيع قائمة الملفات التي رفعها ذلك العنوان.

تعرض قائمة الملفات اسم الملف، والمعاينة، وحجم الملف، ونتيجة المراجعة، وحالة الملف، ووقت الرفع. عندما تبدو عمليات الرفع مشبوهة، وسّع عنوان IP أولًا، وراجع الملفات، ثم قرر ما إذا كنت ستحظر عمليات الرفع اللاحقة.

إذا كان عنوان IP مشبوهًا، فأوقف `Upload allowed`. ستُحظر عمليات الرفع المستقبلية من ذلك العنوان.

## البحث والفرز والمرشحات المتقدمة

في أعلى إدارة المستخدمين، يمكنك البحث حسب مصدر IP أو العنوان.

افرز حسب الوقت، أو عدد مرات الرفع، أو إجمالي حجم الرفع للعثور على الرافعين الأحدث، أو الرافعين ذوي التكرار العالي، أو عناوين IP الأعلى استخدامًا.

لفحص أعمق، افتح المرشحات المتقدمة.

![المرشحات المتقدمة](../../image/other/用户管理高级筛选.png)

تدعم المرشحات المتقدمة:

| المرشح | الاستخدام |
| --- | --- |
| النطاق الزمني | عرض عناوين IP التي رفعت ملفات خلال فترة محددة. |
| حالة الوصول | التصفية حسب الحالة الطبيعية، والمحظورة، والحالات المشابهة. |
| قائمة السماح/الحظر | التصفية حسب قائمة السماح، أو قائمة الحظر، أو غير المضبوط. |
| نوع الملف | عرض عناوين IP التي رفعت صورًا، أو مقاطع فيديو، أو صوتًا، أو مستندات، أو ملفات كود، أو ملفات أخرى. |
| حجم الملف | التصفية حسب نطاق حجم الملف المرفوع. |
| التصنيف العمري | التصفية حسب غير المضبوط، وGeneral، وR12+، وR16+، وR18، والتصنيفات المشابهة. |
| حالة الملف | التصفية حسب حالة الملف الحالية للتحقيق في الملفات غير الطبيعية. |

انقر `Apply Filters` للتطبيق. استخدم `Reset` للعودة إلى جميع البيانات.

## عرض الجوال

على الجوال، تتحول إدارة المستخدمين إلى تخطيط بطاقات.

![إدارة المستخدمين على الجوال](../../image/other/手机端显示用户管理效果.png)

تعرض كل بطاقة عنوان IP، والعنوان، وإجمالي حجم الرفع، وعدد مرات الرفع، ومفتاح السماح بالرفع. يمكنك إدارة المستخدمين من دون تمرير الجدول أفقيًا.

## إذا بدا الموقع غير صحيح

تحديد الموقع الجغرافي عبر IP تقريبي. وهو ليس عنوان شارع دقيقًا.

إذا كان المستخدم خلف وكيل، أو مركز بيانات، أو خادم سحابي، أو شبكة عابرة للحدود، فقد يختلف الموقع المعروض عن الموقع الحقيقي.

استخدم هذه الميزة لفهم المصدر التقريبي، والعثور على عمليات الرفع غير الطبيعية، والمساعدة في قرارات الحظر. لا تتعامل معها كتتبع دقيق.

## حالات شائعة

| الحالة | المعنى |
| --- | --- |
| العنوان فارغ | قد لا يكون عنوان IP قد حُلّل بعد، أو قد يكون المصدر الحالي غير متاح مؤقتًا. |
| لغة العنوان غير صحيحة | تحقق من لغة تحديد الموقع الجغرافي عبر IP، وما إذا كان هناك مصدر مضبوط يدعم تلك اللغة. |
| العنوان يظهر مركز بيانات | تظهر كثير من الوكلاء والخوادم السحابية وبرامج الزحف بوصفها عناوين مراكز بيانات أو مزودي خدمة إنترنت. |
| عدد مرات الرفع مرتفع | راجع عنوان IP هذا بعناية واحظر الرفع عند الحاجة. |
| إجمالي حجم الرفع كبير | افرز أو رشّح، ثم وسّع عنوان IP وافحص الملفات المحددة. |
| تحتاج إلى الاستعادة بعد الحظر | أعد تشغيل `Upload allowed`. |

## المسار السريع

```text
Open IP Geolocation in Other Settings
-> Choose IP geolocation language
-> Fill MaxMind, Tencent Map, or ipapi credentials as needed
-> Save settings
-> Open User Management
-> Review IP source, address, total upload size, and upload count
-> Use search, sort, or advanced filters to find abnormal IPs
-> Allow or block uploads as needed
```
