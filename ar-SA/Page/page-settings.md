# إعدادات الصفحة

تتحكم إعدادات الصفحة في مظهر الموقع، والقيم الافتراضية لصفحة الرفع، وصور الخلفية، ومظهر لوحة الإدارة.

## الإعدادات العامة

| الخيار | الغرض |
| --- | --- |
| عنوان الموقع | العنوان الظاهر في تبويب المتصفح. |
| أيقونة الموقع | الأيقونة الصغيرة الظاهرة في تبويب المتصفح. |
| اسم ImgBed | الاسم الظاهر في صفحات الواجهة الأمامية. |
| شعار ImgBed | صورة الشعار الظاهرة في صفحات الواجهة الأمامية. |
| رابط الشعار | الرابط الذي يفتح عند الضغط على الشعار أو الصورة الرمزية. |
| فترة تبديل الخلفية | فترة تدوير عدة خلفيات بالميلي ثانية. `60000` تعني 60 ثانية. |
| شفافية الخلفية | شفافية صورة الخلفية من `0` إلى `1`. القيم الأقل تجعلها أفتح. |
| بادئة URL الافتراضية | البادئة المستخدمة عند إنشاء روابط الصور. تعني القيمة الفارغة استخدام نطاق الموقع الحالي. |

## إعدادات العميل

| الخيار | الغرض |
| --- | --- |
| الإعلان | الإعلان الظاهر أعلى صفحة الرفع. يدعم HTML. |
| قناة الرفع الافتراضية | قناة الرفع المختارة افتراضيًا في صفحة الرفع. يمكن أيضًا اختيار Smart Dispatch. |
| دليل الرفع الافتراضي | دليل الرفع الافتراضي، مثل `/user/`. تعني القيمة الفارغة أو `/` الدليل الجذر. |
| طريقة التسمية الافتراضية | استراتيجية إنشاء اسم الملف افتراضيًا بعد الرفع. انظر أدناه. |
| التحويل إلى WebP افتراضيًا | يحوّل الصور إلى WebP قبل الرفع. |
| تفعيل الضغط افتراضيًا | يضغط الصور محليًا في المتصفح قبل الرفع. |
| عتبة الضغط الافتراضية | يبدأ الضغط تلقائيًا عندما يتجاوز حجم الصورة هذه القيمة، بوحدة MB. |
| الحجم المستهدف الافتراضي | حجم الملف المستهدف بعد الضغط، بوحدة MB. |
| خلفية صفحة تسجيل الدخول | صورة خلفية صفحة تسجيل دخول المستخدم. |
| خلفية صفحة الرفع | صورة خلفية صفحة الرفع. |
| رابط بوابة التذييل | الرابط الذي يفتحه زر بوابة التذييل. |
| إخفاء التذييل | يخفي تذييل الواجهة الأمامية عند تفعيله. |

## إعدادات الإدارة

| الخيار | الغرض |
| --- | --- |
| خلفية صفحة تسجيل دخول الإدارة | صورة خلفية صفحة تسجيل دخول الإدارة. |
| خلفية الإدارة | صورة خلفية صفحات الإدارة. يمكن استخدام عنوان URL لصورة واحدة أو عدة عناوين URL. |
| وضع تحميل الصور | وضع تحميل المعاينات في قائمة ملفات الإدارة. يحمّل وضع الصورة الأصلية الصور الأصلية. ويفضل وضع التحميل الذكي الصور المصغّرة للصور العامة، والصور الأصلية للصور المقيّدة. |
| مصدر الصور المصغّرة | الخدمة المستخدمة لإنشاء الصور المصغّرة: wsrv.nl أو Cloudflare Image Resizing أو WordPress Photon. يجب تفعيل Cloudflare Image Resizing في Cloudflare قبل اختياره. |
| عنصر واجهة Live2D | يعرض شخصية Live2D في لوحة الإدارة. |
| تأثير النقر بالألعاب النارية | يعرض تأثير ألعاب نارية عند النقر على الصفحة. |
| أثر المؤشر النجمي | يعرض أثرًا نجميًا عند تحريك الفأرة. |

## صيغ صور الخلفية

تدعم خلفية صفحة تسجيل الدخول وخلفية صفحة الرفع وخلفية صفحة تسجيل دخول الإدارة هذه الصيغ:

| القيمة | التأثير |
| --- | --- |
| `bing` | يستخدم تدوير خلفيات Bing. |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | يدوّر عدة صور. |
| `["https://example.com/1.jpg"]` | يستخدم صورة خلفية واحدة. |
| `["https://your-domain.com/random?..."]` | يستخدم رابط واجهة API للصور العشوائية. يمكنك ضبط واجهة API الخاصة بك للصور العشوائية في الإعدادات الأخرى، ثم لصق رابط الصورة العشوائية الناتج هنا كإدخال لخلفية واحدة. |

تدعم خلفية الإدارة عناوين URL للصور. يمكن فصل عدة عناوين URL بفواصل إنجليزية كما توضح الصفحة. تعني القيمة الفارغة استخدام الخلفية الافتراضية.

## طريقة التسمية الافتراضية

| الطريقة | النتيجة |
| --- | --- |
| افتراضي | بادئة زمنية عشوائية + اسم الملف الأصلي، مثل `1760000000000_cat.png`. |
| البادئة فقط | يستخدم البادئة الزمنية العشوائية والامتداد فقط، مثل `1760000000000.png`. |
| الاسم الأصلي فقط | يحتفظ باسم الملف الأصلي، مثل `cat.png`. عند التكرار يضيف ImgBed `(1)` و `(2)` وهكذا. |
| رابط قصير | يستخدم معرّفًا قصيرًا من 8 أحرف مع الامتداد، مثل `a1b2c3d4.png`. |
