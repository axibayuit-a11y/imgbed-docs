# وصول WebDAV للموقع (Beta)

يعرض إعداد WebDAV في Security Settings موقع ImgBed كنقطة WebDAV endpoint.

بعد تفعيله يمكنك استخدام Windows أو macOS أو مديري الملفات على الجوال أو أي عميل متوافق مع WebDAV لتصفح ملفات ImgBed ورفعها وحذفها وإدارتها مثل مجلد بعيد.

هذا هو مدخل WebDAV الخاص بالموقع. يختلف عن قناة تخزين WebDAV في إعدادات الرفع. قناة الرفع تحفظ الملفات في خدمة WebDAV خارجية، أما هذا الإعداد فيجعل موقع ImgBed نفسه يوفر وصول WebDAV للعملاء.

## أين تضبطه؟

افتح لوحة الإدارة، ثم انتقل إلى:

```text
System Settings -> Security Settings -> WebDAV
```

الإعدادات المتاحة:

- Enable
- Username
- Password
- Image loading mode
- Default channel

## ماذا تفعل هذه الميزة؟

بعد تفعيل WebDAV يوفر ImgBed رابط وصول ثابتًا:

```text
https://your-domain.com/dav
```

استخدم هذا الرابط للاتصال بمجلد ملفات ImgBed.

حالات استخدام جيدة:

- تصفح ملفات ImgBed مباشرة من مدير ملفات الكمبيوتر.
- سحب الصور إلى مجلد WebDAV لرفعها.
- تنظيم مجلدات ImgBed من مدير الملفات المحلي.
- استخدام برامج متوافقة مع WebDAV للمزامنة أو إدارة الصور.
- الوصول إلى محتوى ImgBed دون فتح لوحة الإدارة.

## الإعدادات

### Enable

يفعّل WebDAV endpoint.

عند تعطيله لا يستطيع العملاء الاتصال عبر WebDAV.

### Username و Password

تستخدم هذه البيانات عند اتصال عملاء WebDAV.

استخدم اسم مستخدم وكلمة مرور مخصصين لـ WebDAV. لا تعِد استخدام كلمة مرور الإدارة أو كلمة مرور الرفع.

إذا كان اسم المستخدم أو كلمة المرور فارغًا، فلن يتصل عملاء WebDAV بشكل صحيح.

### Image Loading Mode

يحدد Image loading mode رابط الصورة الذي يفضله عملاء WebDAV عند قراءة الصور.

خيارات شائعة:

| Mode | الوصف |
| --- | --- |
| Smart loading | يختار ImgBed حسب السياق. موصى به للاستخدام العادي. |
| Original | يفضل الصور الأصلية. |
| Thumbnail | يفضل المصغرات. مفيد للمعاينة السريعة. |

إذا لم تكن متأكدًا، اترك `Smart loading`.

### Default Channel

تُستخدم default channel لرفعات WebDAV.

عند نسخ ملفات إلى مجلد WebDAV من Windows أو عميل آخر، يرفعها ImgBed عبر قناة الرفع الافتراضية المحددة.

إذا لم تُحدد قناة افتراضية فقد يعمل التصفح، لكن الرفع قد يفشل.

## الوصول إلى WebDAV في Windows 11

يمكن لـ Windows 11 إضافة WebDAV كموقع شبكة.

1. افتح `This PC`.
2. اختر `Add a network location`.
3. أدخل `https://your-domain.com/dav`.
4. أدخل WebDAV username و password عند الطلب.
5. أكمل المعالج. بعدها يمكن فتح مجلد WebDAV في File Explorer.

![إضافة WebDAV في Windows 11](../../image/Safety/webdav在win11配置.png)

بعد إضافته يظهر مجلد WebDAV في Windows File Explorer. يمكنك فتح الملفات ونسخها وإدارتها مثل مجلد عادي.

![WebDAV في Windows](../../image/Safety/webdav在win显示效果.png)

## العمليات المدعومة

بعد اتصال WebDAV ناجح يمكنك غالبًا:

- عرض الملفات والمجلدات.
- رفع الملفات.
- إنشاء مجلدات.
- إعادة تسمية الملفات أو المجلدات.
- نقل الملفات.
- حذف الملفات.

WebDAV مناسب للوصول اليومي وإدارة الملفات الصغيرة. للعمليات الكبيرة أو الحذف الجماعي أو التنظيم المعقد، استخدم لوحة الإدارة.

## Login Device Management

تظهر اتصالات WebDAV الناجحة أيضًا ضمن تبويب WebDAV في Login Device Management.

يمكنك مراجعة عملاء WebDAV هناك وإجبار الأجهزة القديمة على الخروج عند الحاجة.

إذا غيّرت WebDAV username أو password فسيحتاج العملاء القدامى إلى تسجيل الدخول مرة أخرى.

## FAQ

### Windows يطلب اسم المستخدم وكلمة المرور باستمرار

تحقق من:

- الرابط هو `https://your-domain.com/dav`.
- اسم المستخدم وكلمة المرور يطابقان إعدادات WebDAV.
- WebDAV مفعّل.
- الموقع يمكن الوصول إليه عبر HTTPS.

### التصفح يعمل لكن الرفع يفشل

تحقق من `Default channel`.

تحتاج رفعات WebDAV إلى قناة رفع افتراضية. إذا كانت مفقودة أو معطلة أو مضبوطة بشكل خاطئ فقد يفشل الرفع.

### سرعة الوصول غير مستقرة

يعتمد أداء WebDAV على العميل والشبكة وعدد الملفات وقناة الرفع الافتراضية.

إذا كان في المجلد ملفات كثيرة، نظمها في مجلدات بدل إبقاء عدد كبير في مجلد واحد.

## توصيات الأمان

- استخدم HTTPS لوصول WebDAV.
- عيّن كلمة مرور قوية.
- لا تشارك كلمة مرور WebDAV مع أشخاص غير موثوقين.
- عطّل WebDAV عندما لا تستخدمه.
- نظّف أجهزة WebDAV غير المستخدمة دوريًا من Login Device Management.
