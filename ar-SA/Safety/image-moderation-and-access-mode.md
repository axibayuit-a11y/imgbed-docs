# مراجعة الصور ووضع الوصول

تمنح مراجعة الصور تقييمات عمرية للصور المرفوعة. يحدد وضع الوصول أي التقييمات تظهر عبر الوصول العام.

يؤثر ذلك على public gallery وروابط الملفات العامة و random image API. لا يقيّد لوحة الإدارة، فالمسؤولون يستطيعون عرض كل الملفات وإدارتها.

## أين تضبطه؟

افتح لوحة الإدارة، ثم انتقل إلى:

```text
System Settings -> Security Settings -> Upload Management -> Image Moderation
```

الإعدادات الرئيسية:

- Access mode
- Enable moderation
- Moderation provider

## ماذا يفعل Access Mode؟

يحدد Access mode التقييمات العمرية المسموح بعرضها للعامة.

الأوضاع الحالية:

| Access Mode | التقييمات الظاهرة للعامة |
| --- | --- |
| Adult mode | General, R12, R16, R18 |
| Youth mode | General, R12, R16 |
| Teen mode | General, R12 |
| Child mode | General فقط |

الوضع الافتراضي هو Adult mode.

للمواقع الخاصة أو التي تحتوي محتوى مخصصًا للبالغين قد يكون Adult mode مناسبًا. أما للمعارض العامة المحافظة فاختر Youth أو Teen أو Child mode.

## ماذا يفعل تفعيل المراجعة؟

عند تفعيل المراجعة، يستدعي ImgBed مزود المراجعة المختار أثناء الرفع ويحفظ التقييم العمري المكتشف.

التقييمات الرئيسية:

| التقييم | المعنى |
| --- | --- |
| General | محتوى عام آمن |
| R12 | محتوى حساس بدرجة خفيفة |
| R16 | محتوى حساس بدرجة متوسطة |
| R18 | محتوى للبالغين |

تُستخدم نتيجة المراجعة عند تحديد الوصول العام.

إذا لم تكن المراجعة مفعلة، أو كانت الملفات القديمة بلا تقييم، تُعامل تلك الملفات كملفات unrated. عدم وجود تقييم لا يخفيها تلقائيًا من public gallery أو random image API.

## اختيار Moderation Provider

المزودون المتاحون:

- moderatecontent.com
- nsfwjs
- Sightengine

لكل مزود متطلبات مختلفة:

- moderatecontent.com يتطلب غالبًا API Key.
- nsfwjs يتطلب غالبًا API endpoint URL.
- Sightengine يتطلب API user و API secret.

اختر حسب حسابك، والتوفر، وجودة الكشف. ما دام التفعيل والإعداد صحيحين، يحاول ImgBed كتابة تقييم الصورة أثناء الرفع.

## التأثير على Public Gallery

تفلتر public gallery الملفات حسب access mode.

أمثلة:

- Adult mode: يمكن أن تظهر صور R18.
- Youth mode: تُخفى صور R18.
- Teen mode: تُخفى صور R16 و R18.
- Child mode: تظهر صور General فقط.

هذا يؤثر على الوصول العام العادي فقط. لوحة الإدارة تعرض كل الملفات.

## التأثير على روابط الملفات العامة

روابط الملفات العامة هي روابط الصور المباشرة التي يفتحها الزوار.

إذا كان تقييم الملف مسموحًا في access mode الحالي، يعيد ImgBed الصورة الأصلية.

إذا كان التقييم أعلى من المستوى المسموح، لا يعيد الوصول العام العادي الصورة الأصلية. بدلًا من ذلك يعيد ImgBed نتيجة الحجب المضبوطة أو صورة بديلة.

مثال:

- الوضع الحالي Child mode.
- الصورة مصنفة R18.
- زائر يفتح الرابط العام مباشرة.
- لا يعيد ImgBed الصورة الأصلية R18 لذلك الزائر.

![صورة ملف مقيّدة](../../image/Safety/文件受限图.png)

المسؤولون الذين يعرضون الملفات من لوحة الإدارة لا يتأثرون بهذا القيد.

## التأثير على Random Image API

يفلتر random image API أيضًا مجموعة المرشحين حسب access mode.

في Child mode تُختار الصور العشوائية فقط من ملفات General.

في Youth mode قد تأتي الصور من General و R12 و R16، وليس من R18.

هذا يمنع random image API من تجاوز قيود public gallery.

## العلاقة مع قواعد القوائم

Access mode ليس قاعدة الوصول العام الوحيدة. يعمل مع قواعد allow/block list.

ببساطة:

- محتوى allowlist يُعامل أولًا كمحتوى عام.
- محتوى blocklist لا يمكن عرضه مباشرة للزوار العاديين.
- المحتوى غير الموجود في أي قائمة يُفحص بعد ذلك حسب access mode.

إذا كانت الصورة مقيّدة بالتقييم العمري وقواعد القوائم معًا، فلن يستطيع الزوار العاديون عرض الملف الأصلي مباشرة.

## الإعدادات الموصى بها

للمواقع العامة:

- فعّل المراجعة.
- اختر access mode يناسب جمهور الموقع.
- استخدم Child mode أو Teen mode للزوار من كل الأعمار.
- تجنب Adult mode إذا لم ترغب في عرض محتوى للبالغين للعامة.
- راجع تقييمات الملفات في لوحة الإدارة وعدّلها يدويًا عند الحاجة.

للمواقع الخاصة أو الشخصية:

- Adult mode مناسب غالبًا.
- فعّل المراجعة إذا كانت مفيدة.
- راجع وعدّل التقييمات من لوحة الإدارة عند الحاجة.

## FAQ

### هل تختفي الملفات من لوحة الإدارة بعد تغيير Access Mode؟

لا.

Access mode يؤثر على الوصول العام العادي فقط ولا يؤثر على لوحة الإدارة.

### لماذا عرضت Public Gallery صورًا أقل بعد التحويل إلى Child Mode؟

Child mode يسمح فقط بعرض ملفات General للعامة. تُفلتر ملفات R12 و R16 و R18.

### هل يمكن للروابط العامة فتح صور للبالغين؟

إذا كان access mode الحالي لا يسمح بذلك التقييم، فلن تعيد الروابط العامة العادية الصورة الأصلية.

### هل يمكن لـ Random Image API إعادة صور مقيّدة؟

لا.

يفلتر random image API المرشحين حسب access mode الحالي.

### ماذا يحدث للصور القديمة غير المصنفة؟

لا تُخفى الصور غير المصنفة تلقائيًا لمجرد عدم وجود نتيجة مراجعة. يمكنك تعديل تقييمها لاحقًا من لوحة الإدارة.
