# Cloudflare API Token

تتيح بيانات اعتماد Cloudflare API لـ ImgBed مسح ذاكرة Cloudflare CDN المؤقتة بعد تغيّر الملفات.

![إعدادات Cloudflare API Token](../../image/Safety/cloudflare%20api%20token截图.png)

## مكان الإعداد

افتح لوحة الإدارة، ثم انتقل إلى:

```text
System Settings -> Security Settings -> Cloudflare API Token
```

يجب تعبئة الحقول التالية:

- Zone ID
- البريد الإلكتروني للحساب
- API Key

## وظيفة هذا الإعداد

قد يخزّن Cloudflare روابط الصور العامة في ذاكرة التخزين المؤقت.

تجعل ذاكرة التخزين المؤقت تحميل الصور أسرع، لكنها قد تُبقي محتوى قديمًا ظاهرًا لبعض الوقت بعد حذف ملف أو حجبه أو استبداله أو نقله.

بعد إعداد بيانات اعتماد Cloudflare API، يحاول ImgBed مسح ذاكرة Cloudflare المؤقتة المرتبطة بهذه العمليات عند اكتمالها.

يكون هذا مفيدًا عندما:

- تحذف صورة وتريد إيقاف الرابط العام عن العمل في أسرع وقت ممكن.
- تحجب صورة وتريد منع الزوار من رؤية الملف الأصلي.
- تستبدل ملفًا بالاسم نفسه وتريد أن يرى الزوار النسخة الجديدة بسرعة أكبر.
- تنقل ملفات أو تعيد تسميتها وتريد تحديث ذاكرة المسارات القديمة سريعًا.
- تغيّر قواعد الوصول العام وتريد تحديث ذاكرة المعرض العام أو الصور العشوائية بسرعة أكبر.

## ماذا يحدث إذا تركته فارغًا

يعمل ImgBed بشكل طبيعي حتى من دون هذا الإعداد.

الفرق الوحيد هو أن ImgBed لن يمسح ذاكرة Cloudflare CDN المؤقتة تلقائيًا. وقد يستمر الزوار في رؤية المحتوى القديم إلى أن تنتهي صلاحية ذاكرة Cloudflare المؤقتة تلقائيًا.

## كيفية العثور على Zone ID

يمثّل Zone ID معرّف منطقة Cloudflare للموقع الذي يستخدمه نطاق ImgBed لديك.

1. سجّل الدخول إلى لوحة تحكم Cloudflare.
2. افتح الموقع الذي يحتوي على نطاق ImgBed.
3. ابحث عن `Zone ID` في صفحة نظرة عامة على الموقع.
4. انسخه إلى حقل `Zone ID` في ImgBed.

هذا هو Zone ID الخاص بالموقع، وليس معرّف الحساب.

## البريد الإلكتروني للحساب

أدخل عنوان البريد الإلكتروني الذي تستخدمه لتسجيل الدخول إلى Cloudflare.

يجب أن يتوافق هذا البريد مع API Key الذي ستدخله أدناه.

## API Key

أدخل Cloudflare Global API Key.

1. سجّل الدخول إلى لوحة تحكم Cloudflare.
2. افتح ملفك الشخصي.
3. انتقل إلى صفحة API Tokens.
4. ابحث عن `Global API Key`.
5. اعرضه وانسخه.
6. الصقه في حقل `API Key` في ImgBed.

![عرض Global API Key](../../image/Safety/查看全局令牌.png)

## متى يصبح الإعداد نافذًا

بعد تعبئة الحقول، احفظ الإعدادات.

ستحاول تغييرات الملفات اللاحقة مسح ذاكرة Cloudflare المؤقتة تلقائيًا. ولا تُمسح العمليات السابقة بأثر رجعي. إذا كنت قد حذفت أو استبدلت ملفًا قبل إعداد هذه البيانات، فانتظر انتهاء صلاحية ذاكرة Cloudflare المؤقتة أو امسحها يدويًا من Cloudflare.

## الأسئلة الشائعة

### هل هذا الإعداد مطلوب؟

لا.

إذا كان نطاقك لا يستخدم Cloudflare، أو لم تكن مهتمًا بتأخير ذاكرة CDN المؤقتة، فيمكنك تركه فارغًا.

### هل تؤدي بيانات الاعتماد الخاطئة إلى تعطيل الرفع؟

عادةً لا.

تمنع بيانات الاعتماد الخاطئة ImgBed فقط من مسح ذاكرة Cloudflare المؤقتة. يفترض أن يستمر الرفع والوصول العادي إلى الملفات في العمل.

### لماذا ما زالت الصورة المحذوفة تفتح؟

السبب الأكثر شيوعًا هو أن Cloudflare ما زال يحتفظ بالملف القديم في ذاكرة التخزين المؤقت.

عند إعداد بيانات اعتماد Cloudflare API بشكل صحيح، يمسح ImgBed ذاكرة التخزين المؤقت لعنوان URL المرتبط بالملف عند حذفه.

### لماذا ما زلت أرى الصورة القديمة بعد استبدال ملف؟

يحدث ذلك عادةً بسبب ذاكرة CDN المؤقتة أيضًا.

بعد إعداد هذا الخيار، يحاول ImgBed مسح ذاكرة التخزين المؤقت لعنوان URL القديم عند الكتابة فوق ملف بالاسم نفسه.
