# Cloudflare API Token

تسمح بيانات Cloudflare API لـ ImgBed بتنظيف كاش Cloudflare CDN بعد تغيّر الملفات.

![إعدادات Cloudflare API Token](../../image/Safety/cloudflare%20api%20token截图.png)

## أين تضبطه؟

افتح لوحة الإدارة، ثم انتقل إلى:

```text
System Settings -> Security Settings -> Cloudflare API Token
```

تحتاج إلى ملء:

- Zone ID
- Account email
- API Key

## ماذا يفعل هذا الإعداد؟

قد يخزّن Cloudflare روابط الصور العامة في الكاش.

الكاش يجعل عرض الصور أسرع، لكنه قد يُبقي المحتوى القديم ظاهرًا لبعض الوقت بعد حذف ملف أو حجبه أو استبداله أو نقله.

بعد ضبط بيانات Cloudflare API، يحاول ImgBed تنظيف كاش Cloudflare المرتبط عند اكتمال هذه العمليات.

هذا مفيد عندما:

- تحذف صورة وتريد أن يتوقف الرابط العام عن العمل بأسرع وقت.
- تحجب صورة وتريد منع الزوار من رؤية الملف الأصلي.
- تستبدل ملفًا بالاسم نفسه وتريد أن يرى الزوار النسخة الجديدة بسرعة.
- تنقل أو تعيد تسمية ملفات وتريد تحديث كاش المسارات القديمة بسرعة.
- تغيّر قواعد الوصول العام وتريد تحديث كاش public gallery أو random image بسرعة.

## ماذا يحدث إذا تركته فارغًا؟

سيستمر ImgBed في العمل بشكل طبيعي دون هذا الإعداد.

الفرق الوحيد أن ImgBed لن ينظف كاش Cloudflare CDN بشكل نشط. قد يواصل الزوار رؤية المحتوى القديم إلى أن تنتهي صلاحية كاش Cloudflare طبيعيًا.

## كيف تجد Zone ID؟

Zone ID هو معرّف Cloudflare Zone للموقع الذي يستخدمه نطاق ImgBed.

1. سجّل الدخول إلى Cloudflare dashboard.
2. افتح الموقع الذي يحتوي نطاق ImgBed.
3. ابحث عن `Zone ID` في صفحة overview.
4. انسخه إلى حقل `Zone ID` في ImgBed.

هذا هو Zone ID الخاص بالموقع، وليس account ID.

## Account Email

أدخل البريد الذي تستخدمه للدخول إلى Cloudflare.

يجب أن يطابق API Key الذي ستدخله أدناه.

## API Key

أدخل Cloudflare Global API Key.

1. سجّل الدخول إلى Cloudflare dashboard.
2. افتح ملفك الشخصي.
3. انتقل إلى صفحة API Tokens.
4. ابحث عن `Global API Key`.
5. اعرضه وانسخه.
6. الصقه في حقل `API Key` في ImgBed.

![عرض global API key](../../image/Safety/查看全局令牌.png)

## متى يبدأ العمل؟

بعد ملء الحقول احفظ الإعدادات.

ستحاول تغييرات الملفات المستقبلية تنظيف كاش Cloudflare تلقائيًا. العمليات السابقة لا تُنظف بأثر رجعي. إذا حذفت أو استبدلت ملفًا قبل ضبط هذا الإعداد، فانتظر انتهاء كاش Cloudflare أو نظفه يدويًا من Cloudflare.

## FAQ

### هل هو مطلوب؟

لا.

إذا كان نطاقك لا يستخدم Cloudflare، أو لا يهمك تأخير كاش CDN، يمكنك تركه فارغًا.

### هل تكسر البيانات الخاطئة الرفع؟

غالبًا لا.

البيانات الخاطئة تمنع ImgBed فقط من تنظيف كاش Cloudflare. الرفع والوصول العادي للملفات يجب أن يستمرا.

### لماذا ما زالت الصورة المحذوفة تفتح؟

السبب الأكثر شيوعًا أن Cloudflare ما زال يحتفظ بالملف القديم في الكاش.

مع بيانات Cloudflare API الصحيحة، ينظف ImgBed كاش الرابط المرتبط عند حذف الملف.

### لماذا أرى الصورة القديمة بعد استبدال ملف؟

هذا غالبًا بسبب كاش CDN أيضًا.

بعد ضبط هذا الإعداد، يحاول ImgBed تنظيف كاش الرابط القديم عند الكتابة فوق ملف بالاسم نفسه.
