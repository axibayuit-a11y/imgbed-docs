# حدود معدل الرفع للمستخدمين

تتحكم حدود معدل الرفع للمستخدمين في عدد مرات رفع الملفات من الصفحة الرئيسية بواسطة المستخدمين العاديين أو الزوار. يساعد ذلك على منع إساءة استخدام صفحات الرفع العامة.

تؤثر هذه الميزة في الرفع من الصفحة الرئيسية فقط. ولا تشمل رفع المسؤولين أو عمليات الرفع التي تتم باستخدام API Tokens.

## مكان الإعداد

افتح لوحة الإدارة، ثم انتقل إلى:

```text
System Settings -> Security Settings -> Upload Management -> User Rate Limits
```

![إعدادات حدود معدل الرفع للمستخدمين](../../image/other/用户频控截图.png)

## تفعيل حدود المعدل

بعد تشغيل "تفعيل حدود المعدل"، يتتبع ImgBed عمليات الرفع الحديثة حسب عنوان IP الخاص بالرافع.

القيم الافتراضية:

| الإعداد | القيمة الافتراضية | الوصف |
| --- | --- | --- |
| نافذة الفحص | 1.5 ساعة | المدة السابقة التي تُحتسب ضمن سجلات الرفع. |
| الحد الأقصى لعدد الملفات | 20 | أكبر عدد ملفات مسموح به داخل نافذة الفحص. |
| حد حجم الملف الواحد | 20 MB | الحد الأقصى لحجم ملف واحد. |
| حد إجمالي حجم الرفع | 200 MB | الحد الأقصى لإجمالي حجم الرفع داخل نافذة الفحص. |

على سبيل المثال، مع نافذة مدتها 1.5 ساعة، و20 ملفًا، و20 MB لكل ملف، و200 MB إجمالًا، تُحظر عمليات الرفع من عنوان IP نفسه عند تجاوز أي حد من الحدود المضبوطة.

## استبعاد أنواع الملفات

يمنع خيار "أنواع ملفات الرفع المستبعدة" المستخدمين العاديين أو الزوار من رفع فئات الملفات المحددة.

الفئات المتاحة:

| النوع | الوصف |
| --- | --- |
| الصور | ملفات الصور مثل jpg وpng وwebp وgif وما شابهها |
| الفيديو | ملفات الفيديو مثل mp4 وwebm وmov وما شابهها |
| الصوت | ملفات الصوت مثل mp3 وflac وwav وما شابهها |
| المستندات | ملفات المستندات مثل pdf وtxt وmd وdocx وما شابهها |
| أخرى | الملفات التي لا تندرج ضمن الفئات السابقة، مثل zip وrar وexe وapk |

افتراضيًا، لا يكون أي نوع محددًا، وهذا يعني أنه مسموح.

عند النقر على نوع معيّن وتمييزه، يصبح هذا النوع محظورًا.

إذا تم تحديد "أخرى"، فسيُمنع الزوار من رفع ملفات zip أو rar، وتظهر لهم رسالة تفيد بأن نوع الملف غير مدعوم.

## رسائل الحظر

عند تفعيل أحد الحدود، يرى المستخدمون رسالة مناسبة:

![رسالة رفع متكرر جدًا](../../image/other/频繁报错提示.png)

| الحالة | معنى الرسالة |
| --- | --- |
| الملف الواحد كبير جدًا | الملف كبير جدًا ويجب ضغطه قبل الرفع. |
| نوع الملف محظور | نوع الملف غير مدعوم. أزله ثم حاول مرة أخرى. |
| عمليات الرفع متكررة جدًا | عمليات الرفع الحديثة متكررة جدًا، مع عرض وقت إعادة المحاولة. |
| إجمالي الحجم مرتفع جدًا | إجمالي حجم الرفع الحديث مرتفع جدًا، مع عرض وقت إعادة المحاولة. |

## متى يجب تفعيلها

فعّل حدود معدل الرفع للمستخدمين إذا كانت صفحة الرفع الرئيسية متاحة للعامة.

أسباب شائعة:

- أنت قلق من عمليات رفع جماعية عبر السكربتات.
- تريد تقييد الملفات الكبيرة التي يرفعها الزوار.
- تريد السماح للمستخدمين العاديين برفع الصور فقط، لا الأرشيفات أو ملفات التثبيت.
- تريد إبقاء الرفع العام متاحًا مع التحكم في استهلاك الموارد.

إذا كان الموقع لاستخدامك الشخصي فقط، أو كان الرفع متاحًا للمسؤولين فقط، فيمكنك ترك هذا الخيار معطلًا.
