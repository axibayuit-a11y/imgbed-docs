# Koffie sponsoren

Hallo, ik ben Linjiang, de ontwikkelaar van ImgBed.  

In het begin wilde ik alleen een eenvoudige willekeurige image host voor mijn eigen blog maken. Toevallig vond ik het open-sourceproject CloudFlare-ImgBed. De functies waren toen nog vrij beperkt, dus ik stuurde een paar PRs om ontbrekende dingen aan te vullen.  
Terwijl ik eraan bleef werken, kreeg ik een groter idee: zou dit een complete persoonlijke contenthostingtool kunnen worden die iedereen makkelijk kan gebruiken?  

Ik heb maandenlang stap voor stap aan dit project gebouwd en elke functie verfijnd tot wat het nu is. Eerlijk gezegd ben ik geen professionele ontwikkelaar, maar iemand die nog veel leert. Zonder AI-tools zoals Codex en Claude had ik zo’n complexe projectstructuur niet kunnen afmaken.  

Als ImgBed je echt heeft geholpen, hoop ik oprecht dat je de ontwikkeling wilt steunen. Zelfs de prijs van een kop koffie betekent veel voor mij.

✅ Nieuwe functies als eerste proberen
