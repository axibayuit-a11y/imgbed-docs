# Telegram चैनल जोड़ना

## शुरू करने से पहले क्या चाहिए

| आवश्यकता | उद्देश्य |
| --- | --- |
| Telegram खाता | बॉट और स्टोरेज चैनल बनाने के लिए। |
| `@BotFather` | Telegram बॉट बनाने के लिए। |
| Telegram चैनल | फ़ाइलों का अंतिम स्टोरेज गंतव्य। |
| `@userinfobot` | चैनल का `Chat ID` पूछने के लिए। |

## कहाँ जोड़ें

1. सिस्टम सेटिंग्स खोलें।
2. अपलोड सेटिंग्स पर जाएँ।
3. ऊपर दाएँ कोने में चैनल जोड़ें पर क्लिक करें।
4. `Telegram` चुनें।

## फ़ील्ड संदर्भ

| फ़ील्ड | कार्य | आवश्यक |
| --- | --- | --- |
| चैनल नाम | इस चैनल का पहचानने योग्य नाम, जैसे `Telegram Primary`। | आवश्यक |
| सक्रिय | इस चैनल को सक्षम या अक्षम करता है। | सुझाया गया |
| Bot Token | आपके Telegram बॉट का टोकन। | आवश्यक |
| Session ID (Chat ID) | Telegram चैनल का ID। | आवश्यक |
| रिले प्रॉक्सी URL (वैकल्पिक) | केवल तब उपयोग करें जब Telegram एक्सेस अस्थिर हो। `https://` सहित पूरा प्रॉक्सी URL दर्ज करें। | वैकल्पिक |
| टिप्पणी | आगे के रखरखाव के लिए नोट्स। | वैकल्पिक |

## कॉन्फ़िगरेशन चरण

### 1. Telegram बॉट बनाएँ

1. Telegram खोलें और `@BotFather` खोजें।
2. बातचीत खोलें और `Start` पर क्लिक करें।
3. `/newbot` भेजें।
4. निर्देशों के अनुसार बॉट का प्रदर्शित नाम दर्ज करें।
5. निर्देशों के अनुसार बॉट का उपयोगकर्ता नाम दर्ज करें। आम तौर पर इसका अंत `bot` से होना चाहिए।
6. बॉट बन जाने के बाद `@BotFather` बॉट टोकन लौटाएगा।

यही टोकन ImgBed में `Bot Token` के रूप में दर्ज करना है।

![बॉट टोकन सहेजें](../../image/upload/telegram/保存机器人令牌.png)

### 2. चैनल बनाएँ

1. Telegram में नया चैनल पर क्लिक करें।
2. चैनल नाम दर्ज करें।
3. चैनल बनाना पूरा करें।

सार्वजनिक और निजी दोनों चैनल उपयोग किए जा सकते हैं।

![चैनल बनाएँ](../../image/upload/telegram/新建频道.png)

### 3. बॉट को चैनल में जोड़ें

1. अभी बनाया गया चैनल खोलें।
2. चैनल सेटिंग्स खोलें।
3. सदस्य या व्यवस्थापक जोड़ें।
4. बनाए गए बॉट उपयोगकर्ता नाम को खोजें।
5. बॉट को चैनल में जोड़ें।

अधिक भरोसेमंद अपलोड के लिए बॉट को व्यवस्थापक अनुमति दें।

![बॉट को चैनल में आमंत्रित करें](../../image/upload/telegram/邀请机器人进频道里.png)

### 4. User Info - Get ID - IDbot से Channel ID प्राप्त करें

1. Telegram में `@userinfobot` खोजें। इसका प्रदर्शित नाम आम तौर पर `User Info - Get ID - IDbot` होता है।
2. बातचीत खोलें और `Start` पर क्लिक करें।
3. बॉट के विकल्पों में से `Channel` चुनें।
4. संदेश चयनकर्ता में लक्षित चैनल चुनें और उसे `@userinfobot` को भेजें।
5. जब `@userinfobot` परिणाम लौटाए, तो `Id: -100...` के रूप में दिखने वाला नंबर कॉपी करें।

`-100` से शुरू होने वाला नंबर ImgBed के लिए आवश्यक `Session ID (Chat ID)` है।

![Channel ID प्राप्त करें](../../image/upload/telegram/获取频道id.png)

### 5. ImgBed में Telegram चैनल भरें

चैनल कॉन्फ़िगरेशन संवाद पर लौटें और फ़ील्ड इस तरह भरें:

| UI फ़ील्ड | मान |
| --- | --- |
| चैनल पहचानकर्ता | कस्टम चैनल नाम, जैसे `TelegramPrimary`। |
| सक्रिय | सुझाया गया। |
| Bot Token | `@BotFather` से मिला बॉट टोकन। |
| Session ID (Chat ID) | `@userinfobot` द्वारा लौटाया गया `-100...` नंबर। |
| रिले प्रॉक्सी URL (वैकल्पिक) | केवल आवश्यकता होने पर, जैसे `https://your-tg-proxy.example.com`. |
| टिप्पणी | वैकल्पिक नोट्स। |

पूरा होने पर सहेजें पर क्लिक करें।

![कॉन्फ़िगरेशन संपादित करें](../../image/upload/telegram/编辑配置.png)

## सत्यापन

| जाँच | जाँचने का तरीका |
| --- | --- |
| चैनल कार्ड दिखाई देता है | सहेजने के बाद अपलोड सेटिंग्स पेज पर Telegram चैनल कार्ड दिखना चाहिए। |
| चैनल सक्षम हो सकता है | Active स्विच सक्षम रहना चाहिए। |
| कॉन्फ़िगरेशन सहेजा गया है | विवरण दृश्य में Bot Token और Chat ID सहेजे हुए दिखने चाहिए। |
| अपलोड काम करता है | परीक्षण छवि अपलोड करें और पुष्टि करें कि वह लक्षित Telegram चैनल में दिखाई देती है। |

## त्वरित सूची

```text
Create a bot with @BotFather
-> Save the Bot Token
-> Create a Telegram channel
-> Add the bot to the channel and grant administrator permissions
-> Search for @userinfobot and choose Channel
-> Forward any message from the channel to @userinfobot
-> Copy the returned Id: -100...
-> Enter the Bot Token and Chat ID in ImgBed
-> Save and upload a test image
```
