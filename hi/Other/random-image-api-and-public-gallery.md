# यादृच्छिक छवि API और सार्वजनिक गैलरी

दोनों सुविधाएँ यहाँ कॉन्फ़िगर की जाती हैं:

```text
System Settings -> Other Settings
```

## यादृच्छिक छवि API

यादृच्छिक छवि API चुनी गई डायरेक्टरी से एक यादृच्छिक फ़ाइल लौटाता है। यह साइट पृष्ठभूमि, अवतार रोटेशन या बाहरी पेजों से यादृच्छिक छवि कॉल के लिए उपयोगी है।

इसे सक्षम करने के बाद उपयोग करें:

```text
https://your-domain.com/random
```

## यादृच्छिक छवि API सेटिंग्स

| विकल्प | उद्देश्य |
| --- | --- |
| सक्षम करें | `/random` एंडपॉइंट को चालू या बंद करता है। बंद होने पर पहुँच निषिद्ध होती है। |
| डायरेक्टरी | सीमित करता है कि यादृच्छिक छवि API किन डायरेक्टरी का उपयोग कर सकता है। यहाँ शामिल न की गई डायरेक्टरी API द्वारा उपयोग नहीं की जा सकतीं। |
| कॉल डेमो | यादृच्छिक छवि API लिंक जनरेट करता है जिन्हें आप सीधे कॉपी कर सकते हैं। |

आप कई डायरेक्टरी चुन सकते हैं। उदाहरण के लिए, यदि केवल `/landscape/` और `/portrait/` की अनुमति है, तो यादृच्छिक छवि API केवल उन्हीं डायरेक्टरी और उनकी उपडायरेक्टरी से फ़ाइलें चुन सकता है।

## यादृच्छिक छवि API पैरामीटर

| पैरामीटर | उदाहरण | उद्देश्य |
| --- | --- | --- |
| `dir` | `/landscape/` | यादृच्छिक डायरेक्टरी निर्दिष्ट करता है। |
| `content` | `image` | मीडिया प्रकार निर्दिष्ट करता है। `image`, `video`, `audio` या कॉमा से अलग किए गए संयोजन उपयोग करें। |
| `orientation` | `auto` | छवि अभिविन्यास फ़िल्टर करता है। `portrait`, `landscape` या `auto` उपयोग करें। |
| `type` | `url` | लौटाए जाने वाला फ़ॉर्मैट। खाली होने पर रीडायरेक्ट होता है, `url` सादा टेक्स्ट URL लौटाता है, `json` JSON लौटाता है। |
| `origin` | `1` | `type=url` के साथ पूर्ण URL लौटाने के लिए उपयोग होता है। |
| `age` | `all-ages,r12` | आयु रेटिंग के आधार पर फ़िल्टर करता है। |
| `tag` | `wallpaper,sky` | केवल वे फ़ाइलें लौटाता है जिनमें ये टैग हों। |
| `ex` | `private` | इन टैग वाली फ़ाइलों को बाहर करता है। |

## लौटाए जाने वाले फ़ॉर्मैट

`type` के बिना API सीधे यादृच्छिक फ़ाइल URL पर रीडायरेक्ट करता है।

`type=url` के साथ यह टेक्स्ट URL लौटाता है।

`type=json` के साथ यह फ़ाइल जानकारी लौटाता है, जिसमें फ़ाइल URL, फ़ाइल ID, फ़ाइल नाम, फ़ाइल प्रकार, टैग, रेटिंग और संबंधित मेटाडेटा शामिल होते हैं।

## पहुँच नियम

यादृच्छिक छवि API सार्वजनिक पहुँच नियमों का पालन करता है:

| नियम | प्रभाव |
| --- | --- |
| डायरेक्टरी प्रतिबंध | केवल अनुमति प्राप्त डायरेक्टरी की फ़ाइलें चुनी जा सकती हैं। |
| ब्लॉकलिस्ट | ब्लॉकलिस्ट में मौजूद फ़ाइलें यादृच्छिक पूल से बाहर रहती हैं। |
| अनुमति सूची मोड | सक्षम होने पर केवल सार्वजनिक पहुँच के लिए अनुमति प्राप्त फ़ाइलें लौटाई जाती हैं। |
| आयु रेटिंग | R12, R16, R18 और समान सामग्री मौजूदा पहुँच मोड के अनुसार फ़िल्टर होती है। |

यदि फ़िल्टरिंग के बाद कोई फ़ाइल मेल नहीं खाती, तो API कोई मिलान परिणाम नहीं लौटाता।

## कैश

यादृच्छिक छवि API गति सुधारने के लिए डायरेक्टरी के उम्मीदवार समूहों को कैश करता है।

फ़ाइलें बदलने के बाद ImgBed डायरेक्टरी कैश संस्करण अपडेट करता है, और बाद के अनुरोध उम्मीदवार समूह को फिर से बनाते हैं। बार-बार क्वेरी से बचने के लिए खाली डायरेक्टरी को थोड़े समय के लिए कैश किया जाता है।

## सार्वजनिक गैलरी

सार्वजनिक गैलरी उन डायरेक्टरी के लिए केवल-पढ़ने योग्य सार्वजनिक ब्राउज़िंग पेज देती है जिन्हें आप आगंतुकों को दिखाने की अनुमति देते हैं।

इसे सक्षम करने के बाद, आगंतुक यह खोल सकते हैं:

```text
https://your-domain.com/browse/directory-name
```

## सार्वजनिक गैलरी सेटिंग्स

| विकल्प | उद्देश्य |
| --- | --- |
| सक्षम करें | सार्वजनिक गैलरी को चालू या बंद करता है। बंद होने पर आगंतुक इसे ब्राउज़ नहीं कर सकते। |
| छवि लोडिंग मोड | नियंत्रित करता है कि पूर्वावलोकन मूल छवियों का उपयोग करेंगे या थंबनेल का। |
| खुली डायरेक्टरी | सेट करता है कि आगंतुक किन डायरेक्टरी तक पहुँच सकते हैं। |

## छवि लोडिंग मोड

| मोड | उद्देश्य |
| --- | --- |
| मूल | आगंतुक पेज सीधे मूल फ़ाइलें लोड करता है। |
| थंबनेल | आगंतुक पेज तेज़ लोडिंग के लिए थंबनेल को प्राथमिकता देता है। |

## खुली डायरेक्टरी

खुली डायरेक्टरी तय करती हैं कि आगंतुक क्या देख सकते हैं।

उदाहरण:

```text
/1/,/2/,/landscape/,/portrait/
```

फिर आगंतुक इन तक पहुँच सकते हैं:

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

उपडायरेक्टरी भी खोली जा सकती हैं, जैसे `/2026/lucky/`। जिन डायरेक्टरी को खुला नहीं किया गया है, उनसे आगंतुकों को रोका जाता है।

## सार्वजनिक गैलरी सुविधाएँ

| सुविधा | विवरण |
| --- | --- |
| डायरेक्टरी ब्राउज़ करना | खुली डायरेक्टरी में फ़ाइलें और उपडायरेक्टरी देखें। |
| खोज | फ़ाइल नाम, फ़ाइल ID या टैग से खोजें। |
| प्रकार फ़िल्टर | छवियाँ, वीडियो, ऑडियो या अन्य फ़ाइलें फ़िल्टर करें। |
| टैग फ़िल्टर | चुने हुए टैग शामिल करें या बाहर करें। |
| अभिविन्यास फ़िल्टर | लैंडस्केप या पोर्ट्रेट छवियाँ फ़िल्टर करें। |
| समय फ़िल्टर | अपलोड समय सीमा के आधार पर फ़िल्टर करें। |
| एक्सटेंशन फ़िल्टर | फ़ाइल एक्सटेंशन के आधार पर फ़िल्टर करें। |
| लिंक कॉपी करना | फ़ाइल पहुँच लिंक कॉपी करें। |
| मीडिया पूर्वावलोकन | आगंतुक पेज पर छवियाँ, वीडियो और ऑडियो देखें या चलाएँ। |

## सार्वजनिक गैलरी पहुँच नियम

सार्वजनिक गैलरी भी सार्वजनिक पहुँच नियमों का पालन करती है:

| नियम | प्रभाव |
| --- | --- |
| खुली डायरेक्टरी | केवल अनुमति प्राप्त डायरेक्टरी दिखाई जाती हैं। |
| पहुँच मोड | सामग्री मौजूदा आयु-रेटिंग पहुँच मोड के अनुसार फ़िल्टर होती है। |
| अनुमति सूची मोड | सक्षम होने पर केवल सार्वजनिक पहुँच के लिए अनुमति प्राप्त फ़ाइलें दिखाई जाती हैं। |
| ब्लॉकलिस्ट | ब्लॉकलिस्ट में मौजूद फ़ाइलें छिपाई जाती हैं। |
