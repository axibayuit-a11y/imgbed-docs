# Cloudflare API Token

Cloudflare API प्रमाण-पत्र ImgBed को फ़ाइल बदलने के बाद Cloudflare CDN कैश साफ़ करने देते हैं।

![Cloudflare API Token सेटिंग](../../image/Safety/cloudflare%20api%20token截图.png)

## कहाँ सेट करें

प्रशासन पैनल खोलें, फिर यहाँ जाएँ:

```text
System Settings -> Security Settings -> Cloudflare API Token
```

आपको ये भरना होगा:

- Zone ID
- खाता ईमेल
- API Key

## यह सेटिंग क्या करती है

Cloudflare सार्वजनिक छवि URL को कैश कर सकता है।

कैश छवियों को तेज़ी से पहुँचाने में मदद करता है, लेकिन किसी फ़ाइल को हटाने, रोकने, बदलने या स्थानांतरित करने के बाद पुरानी सामग्री कुछ समय तक दिखाई दे सकती है।

Cloudflare API प्रमाण-पत्र कॉन्फ़िगर होने के बाद, ऐसे काम पूरे होने पर ImgBed संबंधित Cloudflare कैश को साफ़ करने की कोशिश करता है।

यह इन स्थितियों में उपयोगी है:

- आप कोई छवि हटाते हैं और चाहते हैं कि सार्वजनिक लिंक जल्द से जल्द काम करना बंद करे।
- आप कोई छवि रोकते हैं और चाहते हैं कि आगंतुक मूल फ़ाइल न देखें।
- आप उसी नाम वाली फ़ाइल बदलते हैं और चाहते हैं कि आगंतुक नया संस्करण जल्दी देखें।
- आप फ़ाइलों को स्थानांतरित या नामांतरित करते हैं और चाहते हैं कि पुराने पथ का कैश जल्दी ताज़ा हो।
- आप सार्वजनिक पहुँच नियम बदलते हैं और चाहते हैं कि सार्वजनिक गैलरी या यादृच्छिक छवि कैश जल्दी अपडेट हो।

## खाली छोड़ने पर क्या होता है

इस सेटिंग के बिना भी ImgBed सामान्य रूप से काम करता है।

केवल अंतर यह है कि ImgBed Cloudflare CDN कैश को सक्रिय रूप से साफ़ नहीं करेगा। आगंतुक Cloudflare कैश के स्वाभाविक रूप से समाप्त होने तक पुरानी सामग्री देख सकते हैं।

## Zone ID कैसे खोजें

Zone ID उस Cloudflare साइट का Zone ID है जिसका उपयोग आपका ImgBed डोमेन करता है।

1. Cloudflare डैशबोर्ड में साइन इन करें।
2. वह साइट खोलें जिसमें आपका ImgBed डोमेन है।
3. साइट अवलोकन पेज पर `Zone ID` खोजें।
4. इसे ImgBed में `Zone ID` फ़ील्ड में कॉपी करें।

यह साइट का Zone ID है, खाता ID नहीं।

## खाता ईमेल

वह ईमेल पता दर्ज करें जिससे आप Cloudflare में साइन इन करते हैं।

यह नीचे दिए जाने वाले API Key से मेल खाना चाहिए।

## API Key

अपना Cloudflare Global API Key दर्ज करें।

1. Cloudflare डैशबोर्ड में साइन इन करें।
2. अपनी प्रोफ़ाइल खोलें।
3. API Tokens पेज पर जाएँ।
4. `Global API Key` खोजें।
5. उसे देखें और कॉपी करें।
6. उसे ImgBed में `API Key` फ़ील्ड में चिपकाएँ।

![Global API Key देखें](../../image/Safety/查看全局令牌.png)

## यह कब प्रभावी होता है

फ़ील्ड भरने के बाद सेटिंग सहेजें।

भविष्य में होने वाले फ़ाइल बदलाव स्वतः Cloudflare कैश साफ़ करने की कोशिश करेंगे। पहले हो चुके काम पीछे से साफ़ नहीं किए जाते। यदि आपने यह सेटिंग करने से पहले कोई फ़ाइल हटाई या बदली थी, तो Cloudflare कैश के समाप्त होने की प्रतीक्षा करें या Cloudflare में कैश को हाथ से साफ़ करें।

## सामान्य प्रश्न

### क्या यह आवश्यक है?

नहीं।

यदि आपका डोमेन Cloudflare का उपयोग नहीं करता, या CDN कैश की देरी से आपको फर्क नहीं पड़ता, तो इसे खाली छोड़ा जा सकता है।

### गलत प्रमाण-पत्र से अपलोड टूटेंगे?

आम तौर पर नहीं।

गलत प्रमाण-पत्र केवल ImgBed को Cloudflare कैश साफ़ करने से रोकते हैं। अपलोड और सामान्य फ़ाइल पहुँच चलते रहने चाहिए।

### हटाई गई छवि अभी भी क्यों खुलती है?

सबसे सामान्य कारण यह है कि Cloudflare के पास पुरानी फ़ाइल अभी भी कैश में है।

सही Cloudflare API प्रमाण-पत्र होने पर ImgBed फ़ाइल हटाते समय संबंधित URL कैश साफ़ करता है।

### फ़ाइल बदलने के बाद भी पुरानी छवि क्यों दिखती है?

यह भी आम तौर पर CDN कैश के कारण होता है।

यह सेटिंग कॉन्फ़िगर होने के बाद, उसी नाम वाली फ़ाइल overwrite होने पर ImgBed पुराने URL कैश को साफ़ करने की कोशिश करता है।
