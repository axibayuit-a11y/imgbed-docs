# ImgBed

د Cloudflare پر چاپېریال ولاړ، د شخصي منځپانګې د کوربه‌توب بشپړ ابزار دی.

ImgBed د انځور پورته کولو او د تړونونو له مديريت څخه پيل شو. اوس دا د شخصي منځپانګې د کوربه‌توب لپاره يو بشپړ حل ګرځېدلی دی. تاسې يې کارولای شئ چې:

- فایلونه خپل د خوښې هر زېرمه چینل ته پورته کړئ
- انځورونه او د هر ډول بڼې فایلونه په عامه ډول شریک کړئ
- په يوه دقيقه کې شخصي بلاګ ژر جوړ کړئ
- له مقناطیسي تړونو څخه فایلونه په يو کلیک کې خوندي کړئ
- د ImgBed د نوډونو فدراسیون سره يوځای شئ او پرانيستي فایل شاخصونه شریک کړئ

زموږ لومړنی هدف دا دی چې هر څوک وکولای شي د انځورونو د کوربه‌توب او فایل شریکولو خپل ځانګړی سایټ ولري. مهمه نه ده چې تاسې د کوم هېواد یاست یا کومه ژبه کاروئ؛ که یوازې انځورونه ساتل، البومونه منظمول او له ملګرو سره فایلونه شریکول غواړئ، یا د بلاګ، تخلیقي منځپانګې او عامه سرچینو لپاره یو ثابت کوربه‌توب ځای جوړول غواړئ، ImgBed هڅه کوي د پیل خنډ تر وسه کم کړي او دا کار لا ازاد، روښانه او اسانه کړي.

## چمتو او پياوړي امکانات

- 🌍 د 28 انٹرفېس ژبو ملاتړ، څو د نړۍ کاروونکي يې اسانه وکاروي
- 📦 13 جوړ دننه اصلي پورته کولو چینلونه: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud
- 🧩 بشپړ ماډلر ډیزاین، هره ځانګړنه د اړتیا له مخې فعالېږي او سرچینې نه ضایع کېږي
- 🔐 د API Token، لاسرسي حالت او لېست قواعدو ملاتړ کوي، څو د منځپانګې اجازې په ډېر دقت اداره شي

دا پروژه لا هم ښه کېږي. زموږ توان محدود دی، خو هڅه کوو هره ځانګړنه روښانه، باثباته او د کارولو لپاره اسانه کړو. که ImgBed تاسې سره مرسته وکړي چې خپل د انځور کوربه‌توب ځای، بلاګ يا د فایل شریکولو فضا په اسانه ولرئ، موږ به ډېر خوشحاله شو.
