# ImgBed

د Cloudflare ecosystem پر بنسټ يو all-in-one personal content hosting tool دی.

ImgBed د image upload او link management څخه پيل شو. اوس دا د personal content hosting لپاره يو بشپړ solution ګرځېدلی دی. تاسې يې کارولای شئ چې:

- files هر هغه storage channel ته upload کړئ چې تاسې يې خوښوئ
- images او د هر format files په public ډول share کړئ
- په يوه دقيقه کې personal blog ژر جوړ کړئ
- له magnet links څخه files په one-click کې save کړئ
- د ImgBed node federation سره يوځای شئ او open file indexes share کړئ

موږ په "one-size-fits-all" solution باور نه لرو. که تاسې يوازې images ساتل غواړئ، albums په bulk ډول manage کول غواړئ، څو notes ليکل غواړئ، يا يو lightweight public resource site جوړول غواړئ، ImgBed ستاسې انتخاب ته درناوی کوي او maximum flexibility درکوي.

## چمتو او پياوړي امکانات

- 🌍 د 28 interface languages ملاتړ، څو global users يې اسانه وکاروي
- 📦 13 built-in mainstream upload channels: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud
- 🧩 pure modular design، هر feature د اړتيا له مخې enable کېږي او resources نه ضايع کېږي
- 🔒 data بشپړ ستاسې په control کې دی. هېڅ third party ستاسې content نه شي ليدلای يا delete کولای

دا project لا هم ښه کېږي. زموږ توان محدود دی، خو هڅه کوو هر feature clear, stable او easy to use کړو. که ImgBed تاسې سره مرسته وکړي چې خپل image host, blog يا file sharing space په اسانه ولرئ، موږ به ډېر خوشحاله شو.
