# د ImgBed کارن لاسوندونه

ImgBed د Cloudflare ecosystem پر بنسټ جوړ شوی د image hosting او file sharing project دی. په پیل کې دا د image upload او link management لپاره و، خو اوس د personal content hosting لپاره یو پراخ tool شوی دی: files بېلابېلو storage channels ته upload کولای شئ، public gallery خپرولای شئ، خپل blog چلولای شئ، له magnet links څخه files transfer کولای شئ، او له نورو ImgBed nodes سره د federated file index له لارې open files share کولای شئ.

موږ غواړو ImgBed یوازې د "image upload او link اخیستلو" tool پاتې نه شي. موخه دا ده چې هر څوک په اسانه خپل image host، blog او file sharing space ولري. که images ساتئ، albums manage کوئ، څو posts لیکئ، یا lightweight public resource page جوړوئ، ImgBed هڅه کوي ستاسو د خپلې طریقې لپاره ځای پرېږدي.

اوس ImgBed د 28 localized languages ملاتړ کوي او Telegram، Cloudflare R2، S3، WebDAV، Discord، Hugging Face، GitHub Releases، GitLab Packages، OneDrive، Google Drive، Dropbox، Yandex، pCloud او نور upload channels لري. د خپل region، accounts او usage habits له مخې مناسب storage ټاکلای شئ.

دا project لا هم ښه کېږي. زموږ توان محدود دی، خو هڅه کوو هر feature روښانه، stable او ګټور کړو. که ImgBed ستاسې د خپل image host، blog یا file sharing space په جوړولو کې مرسته وکړي، موږ به ډېر خوشحاله شو.
