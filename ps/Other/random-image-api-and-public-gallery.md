# تصادفي انځورونه او عامه ګالري

دواړه ځانګړنې دلته تنظیمېږي:

```text
System Settings -> Other Settings
```

## Random Image API

Random Image API له ټاکلو ډایرکټریو څخه یو تصادفي فایل راګرځوي. دا د سایټ شالیدونو، د اواتار څرخېدو یا له بهرنیو پاڼو څخه د تصادفي انځور غوښتنو لپاره ګټور دی.

له فعالېدو وروسته، دا وکاروئ:

```text
https://your-domain.com/random
```

## د Random Image API تنظیمات

| اختیار | موخه |
| --- | --- |
| فعالول | د `/random` پای ټکی فعال یا غیر فعالوي. کله چې غیر فعال وي، لاسرسی منع کېږي. |
| ډایرکټرۍ | محدودوي چې Random Image API کومې ډایرکټرۍ کارولای شي. هغه ډایرکټرۍ چې دلته شاملې نه وي، API یې نه شي کارولای. |
| د غوښتنې بېلګه | د Random Image API لینکونه جوړوي چې مستقیم یې کاپي کولای شئ. |

تاسو کولای شئ څو ډایرکټرۍ وټاکئ. د بېلګې په توګه، که یوازې `/landscape/` او `/portrait/` اجازه ولري، Random Image API یوازې له همدغو ډایرکټریو او د هغوی له فرعي ډایرکټریو څخه فایلونه ټاکلای شي.

## د Random Image API پارامترونه

| پارامتر | بېلګه | موخه |
| --- | --- | --- |
| `dir` | `/landscape/` | تصادفي ډایرکټري ټاکي. |
| `content` | `image` | د رسنۍ ډول ټاکي. `image`، `video`، `audio` یا په کامو جلا شوي ترکیبونه وکاروئ. |
| `orientation` | `auto` | د انځور لوری فلټر کوي. `portrait`، `landscape` یا `auto` وکاروئ. |
| `type` | `url` | د راستنېدو بڼه. تش ارزښت مخامخ لېږدوي، `url` ساده متني URL راګرځوي، `json` JSON راګرځوي. |
| `origin` | `1` | له `type=url` سره کارېږي څو بشپړ URL راستانه کړي. |
| `age` | `all-ages,r12` | د عمر درجې له مخې فلټر کوي. |
| `tag` | `wallpaper,sky` | یوازې هغه فایلونه راګرځوي چې دا ټګونه ولري. |
| `ex` | `private` | هغه فایلونه لرې کوي چې دا ټګونه ولري. |

## د راستنېدو بڼې

له `type` پرته، API مستقیم د تصادفي فایل URL ته لېږدوي.

له `type=url` سره، متني URL راګرځوي.

له `type=json` سره، د فایل معلومات راګرځوي، لکه د فایل URL، د فایل ID، د فایل نوم، د فایل ډول، ټګونه، درجه او اړوند مېټاډاټا.

## د لاسرسي قواعد

Random Image API د عامه لاسرسي قواعد تعقیبوي:

| قاعده | اغېز |
| --- | --- |
| د ډایرکټري محدودیت | یوازې په اجازه لرونکو ډایرکټریو کې فایلونه ټاکل کېدای شي. |
| د بند لست | د بند لست فایلونه له تصادفي ټولګې څخه لرې کېږي. |
| د اجازې لست حالت | کله چې فعال وي، یوازې هغه فایلونه راګرځي چې د عامه لاسرسي اجازه لري. |
| د عمر درجه | R12، R16، R18 او ورته منځپانګه د اوسني لاسرسي حالت له مخې فلټر کېږي. |

که له فلټر کولو وروسته هېڅ فایل برابر نه شي، API د نه سمون پایله راګرځوي.

## کېش

Random Image API د سرعت د ښه کولو لپاره د ډایرکټریو د نوماندو فایلونو ټولګې کېش کوي.

کله چې فایلونه بدل شي، ImgBed د ډایرکټري د کېش نسخه تازه کوي، او وروسته غوښتنې د نوماندو فایلونو ټولګه بیا جوړوي. تشې ډایرکټرۍ د لنډ وخت لپاره په کېش کې ساتل کېږي، څو تکراري پوښتنې کمې شي.

## عامه ګالري

عامه ګالري د هغو ډایرکټریو لپاره د یوازې لوستلو عامه کتنې پاڼه برابروي چې تاسو یې لیدونکو ته د کتلو اجازه ورکوئ.

له فعالېدو وروسته، لیدونکي دا پرانیستلای شي:

```text
https://your-domain.com/browse/directory-name
```

## د عامه ګالرۍ تنظیمات

| اختیار | موخه |
| --- | --- |
| فعالول | عامه ګالري فعال یا غیر فعالوي. کله چې غیر فعاله وي، لیدونکي یې نه شي کتلای. |
| د انځور بارولو حالت | کنټرولوي چې مخکتنې اصلي انځورونه کاروي که کوچني انځورونه. |
| پرانیستې ډایرکټرۍ | ټاکي چې لیدونکي کومو ډایرکټریو ته لاسرسی لري. |

## د انځور بارولو حالت

| حالت | موخه |
| --- | --- |
| اصلي | د لیدونکي پاڼه اصلي فایلونه مستقیم باروي. |
| کوچنی انځور | د لیدونکي پاڼه د چټک بارولو لپاره کوچنیو انځورونو ته لومړیتوب ورکوي. |

## پرانیستې ډایرکټرۍ

پرانیستې ډایرکټرۍ ټاکي چې لیدونکي څه لیدلای شي.

د بېلګې په توګه:

```text
/1/,/2/,/landscape/,/portrait/
```

بیا لیدونکي دې ته لاسرسی لرلای شي:

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

فرعي ډایرکټرۍ هم پرانیستل کېدای شي، لکه `/2026/lucky/`. لیدونکي له هغو ډایرکټریو څخه بندېږي چې پرانیستې نه وي.

## د عامه ګالرۍ ځانګړنې

| ځانګړنه | تشریح |
| --- | --- |
| د ډایرکټریو کتل | په پرانیستو ډایرکټریو کې فایلونه او فرعي ډایرکټرۍ کتل. |
| لټون | د فایل نوم، فایل ID یا ټګونو له مخې لټون. |
| د ډول فلټر | انځورونه، ویډیوګانې، غږ یا نور فایلونه فلټر کول. |
| د ټګ فلټر | ټاکلي ټګونه شاملول یا لرې کول. |
| د لوري فلټر | افقي یا عمودي انځورونه فلټر کول. |
| د وخت فلټر | د اپلوډ د وخت محدودې له مخې فلټر کول. |
| د غځونې فلټر | د فایل غځونې له مخې فلټر کول. |
| لینک کاپي کول | فایل ته د لاسرسي لینکونه کاپي کول. |
| د رسنیو مخکتنه | د لیدونکي په پاڼه کې انځورونه کتل یا ویډیوګانې او غږ غږول. |

## د عامه ګالرۍ د لاسرسي قواعد

عامه ګالري هم د عامه لاسرسي قواعد تعقیبوي:

| قاعده | اغېز |
| --- | --- |
| پرانیستې ډایرکټرۍ | یوازې اجازه لرونکې ډایرکټرۍ ښودل کېږي. |
| د لاسرسي حالت | منځپانګه د اوسني عمر درجې لاسرسي حالت له مخې فلټر کېږي. |
| د اجازې لست حالت | کله چې فعال وي، یوازې هغه فایلونه ښودل کېږي چې د عامه لاسرسي اجازه لري. |
| د بند لست | د بند لست فایلونه پټېږي. |
