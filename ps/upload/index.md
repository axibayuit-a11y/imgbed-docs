# د Upload Settings

دا برخه د System Settings پاڼې د channel ترتیب تعقیبوي، څو تاسې هغه storage provider ته ژر ورشئ چې configure کوئ.

## دا برخه څنګه جوړه شوې

- هر provider خپله جلا پاڼه لري. د 13 upload channels جلا ساتل د setup steps تعقیبول اسانه کوي.
- Screenshots د `../../image/upload/provider-name/` لاندې ساتل شوي او له هرې پاڼې څخه local reference شوي دي.
- په پښتو نسخه کې د UI field names او تخنیکي نومونه، چې د محصول له UI سره باید برابر پاتې شي، په انګلیسي ساتل شوي دي.

## په دې فصل کې څه راځي

- هر upload channel څنګه اضافه شي.
- د channel تر اضافه کولو مخکې کوم credentials، keys یا IDs ته اړتیا ده.
- څنګه وګورئ چې channel د uploads لپاره چمتو دی.

Troubleshooting، quota strategy او best practices په اړوندو پاڼو کې جلا تشریح شوي دي.
