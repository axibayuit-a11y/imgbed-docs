# د Cloudflare API Token

د Cloudflare API معلومات ImgBed ته اجازه ورکوي چې د فایلونو له بدلون وروسته د Cloudflare CDN زیرمه پاکه کړي.

![د Cloudflare API Token تنظیمات](../../image/Safety/cloudflare%20api%20token截图.png)

## چېرته یې تنظیم کړئ

د ادارې پینل پرانیزئ، بیا دې ځای ته لاړ شئ:

```text
System Settings -> Security Settings -> Cloudflare API Token
```

دا فیلډونه باید ډک کړئ:

- Zone ID
- Account email
- API Key

## دا تنظیم څه کوي

Cloudflare کېدای شي د عامه انځور URLونه زیرمه کړي.

زیرمه کول د انځورونو سپارل چټکوي، خو دا هم کېدای شي چې د فایل له حذف، بندولو، بدلولو یا لېږدولو وروسته زوړ منځپانګه د لږ وخت لپاره ښکاره پاتې شي.

کله چې د Cloudflare API معلومات تنظیم شي، ImgBed د دغو عملیاتو له بشپړېدو وروسته د اړوند Cloudflare زیرمې د پاکولو هڅه کوي.

دا ګټور دی کله چې:

- انځور حذف کوئ او غواړئ عامه لینک ژر تر ژره کار پرېږدي.
- انځور بندوئ او غواړئ لیدونکي نور اصلي فایل ونه ویني.
- په همدې نوم فایل بدلوئ او غواړئ لیدونکي نوې نسخه ژر وویني.
- فایلونه لېږدوئ یا نومونه یې بدلوئ او غواړئ د زړې لارې زیرمه ژر تازه شي.
- د عامه لاسرسي قاعدې بدلوئ او غواړئ عامه ګالري یا د تصادفي انځور زیرمه ژر تازه شي.

## که تش یې پرېږدئ څه کېږي

ImgBed له دې تنظیم پرته هم عادي کار کوي.

یوازې توپیر دا دی چې ImgBed به د Cloudflare CDN زیرمه په فعاله توګه نه پاکوي. لیدونکي ښايي زوړ منځپانګه تر هغه وویني چې د Cloudflare زیرمه په طبیعي ډول پای ته ورسېږي.

## د Zone ID موندل

Zone ID د هغه سایټ Cloudflare Zone ID دی چې ستاسو ImgBed ډومېن پکې کارېږي.

1. د Cloudflare داشبورډ ته ننوتل وکړئ.
2. هغه سایټ پرانیزئ چې ستاسو ImgBed ډومېن پکې دی.
3. د سایټ د عمومي کتنې په پاڼه کې `Zone ID` پیدا کړئ.
4. دا د ImgBed د `Zone ID` فیلډ ته کاپي کړئ.

دا د سایټ Zone ID دی، د حساب ID نه دی.

## د حساب برېښنالیک

هغه برېښنالیک پته ولیکئ چې Cloudflare ته د ننوتلو لپاره یې کاروئ.

دا باید له هغه API Key سره سمون ولري چې لاندې یې ورکوئ.

## API Key

خپل Cloudflare Global API Key ولیکئ.

1. د Cloudflare داشبورډ ته ننوتل وکړئ.
2. خپل پروفایل پرانیزئ.
3. د API Tokens پاڼې ته لاړ شئ.
4. `Global API Key` پیدا کړئ.
5. ویې ګورئ او کاپي یې کړئ.
6. دا د ImgBed د `API Key` فیلډ ته ولګوئ.

![د Global API Key کتل](../../image/Safety/查看全局令牌.png)

## کله اغېزمنېږي

د فیلډونو له ډکولو وروسته، تنظیمات خوندي کړئ.

راتلونکي فایل بدلونونه به په اتومات ډول د Cloudflare زیرمه پاکولو هڅه وکړي. پخواني عملیات شاته نه پاکېږي. که فایل مو د دې تنظیم له برابرولو مخکې حذف یا بدل کړی وي، د Cloudflare زیرمې پای ته رسېدو ته انتظار وکړئ یا یې په Cloudflare کې په لاسي ډول پاکه کړئ.

## FAQ

### ایا دا اړین دی؟

نه.

که ستاسو ډومېن Cloudflare نه کاروي، یا د CDN زیرمې ځنډ درته مهم نه وي، دا تش پرېښودلای شئ.

### ایا ناسم معلومات به پورته کول خراب کړي؟

عموما نه.

ناسم معلومات یوازې ImgBed د Cloudflare زیرمې له پاکولو منع کوي. پورته کول او عادي فایل لاسرسی باید دوام وکړي.

### ولې حذف شوی انځور لا هم پرانیستل کېږي؟

تر ټولو عام لامل دا دی چې Cloudflare لا هم زوړ فایل زیرمه کړی وي.

د سمو Cloudflare API معلوماتو په شتون کې، ImgBed د فایل له حذفېدو سره اړوند URL زیرمه پاکوي.

### ولې د فایل له بدلولو وروسته لا هم زوړ انځور وینم؟

دا هم عموما د CDN زیرمې له امله وي.

کله چې دا تنظیم برابر شي، ImgBed هڅه کوي د همدې نوم فایل د بیا لیکل کېدو پر مهال د زوړ URL زیرمه پاکه کړي.
