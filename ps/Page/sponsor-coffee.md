# قهوه ملاتړ کړئ

سلام، زه Linjiang يم، د ImgBed جوړوونکی.  

په پیل کې مې یوازې غوښتل چې د خپل بلاګ لپاره يو ساده random image host جوړ کړم. تصادفاً مې د CloudFlare-ImgBed په نوم open-source پروژه وموندله. هغه مهال يې ځانګړنې ډېرې بنسټيزې وې، نو ما څو PR ولېږل او هغه څه مې ورزیات کړل چې راته پکار وو.  
ورو ورو چې مې پرې کار کاوه، يوه لويه مفکوره راته پيدا شوه: ايا دا کېدای شي داسې بشپړ personal content hosting tool شي چې هر څوک يې په اسانۍ وکاروي؟  

ما څو مياشتې دا پروژه لږ لږ بيا جوړه کړه او هره ځانګړنه مې تر اوسنۍ بڼې ورسوله. رښتيا ووايم، زه مسلکي developer نه يم؛ لا هم زده کړه کوم. د Codex او Claude په څېر AI وسايلو له مرستې پرته به مې دومره پېچلی project structure بشپړ کړی نه وای.  

که ImgBed په رښتيا درسره مرسته کړې وي، هيله لرم د دې د پرمختګ ملاتړ په پام کې ونيسئ. د يوې قهوې په اندازه مرسته هم زما لپاره ډېر ارزښت لري.
