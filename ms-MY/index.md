# ImgBed

Alat serba lengkap untuk pengehosan kandungan peribadi, dibina atas ekosistem Cloudflare.

ImgBed bermula dengan muat naik imej dan pengurusan pautan. Kini ia sudah berkembang menjadi penyelesaian lengkap untuk pengehosan kandungan peribadi. Anda boleh menggunakannya untuk:

- Memuat naik fail ke mana-mana saluran storan pilihan anda
- Berkongsi imej dan fail dalam apa-apa format secara terbuka
- Bina blog peribadi dengan pantas dalam satu minit
- Menyimpan fail daripada pautan magnet dengan satu klik
- Menyertai persekutuan nod ImgBed dan berkongsi indeks fail terbuka

Matlamat asal kami ialah membantu setiap orang memiliki tapak pengehosan gambar dan perkongsian fail mereka sendiri. Tidak kira anda datang dari negara mana atau menggunakan bahasa apa; sama ada anda hanya mahu menyimpan gambar, menyusun album, berkongsi fail dengan rakan, atau membina ruang pengehosan yang stabil untuk blog, kandungan kreatif atau sumber awam, ImgBed berusaha merendahkan halangan permulaan sebanyak mungkin supaya pengalaman ini lebih bebas, lebih jelas dan lebih mudah dimulakan.

## Kuasa sedia guna

- 🌍 Menyokong 28 bahasa antara muka untuk pengguna di seluruh dunia
- 📦 13 saluran muat naik utama terbina dalam: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud
- 🧩 Reka bentuk modular sepenuhnya, semua ciri boleh diaktifkan apabila perlu sahaja, tanpa membazir sumber
- 🔐 Menyokong API Token, mod akses dan peraturan senarai untuk mengurus kebenaran kandungan dengan lebih terperinci

Projek ini masih terus diperbaiki. Keupayaan kami terhad, tetapi kami akan cuba sebaik mungkin untuk menjadikan setiap ciri jelas, stabil dan mudah digunakan. Jika ImgBed membantu anda memiliki ruang pengehosan imej, blog atau perkongsian fail sendiri dengan lebih mudah, kami sangat gembira.
