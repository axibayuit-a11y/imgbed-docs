# Taja kopi

Hai, saya Linjiang, pembangun ImgBed.  

Pada mulanya saya cuma mahu membuat hos imej rawak yang ringkas untuk blog sendiri. Secara kebetulan saya menemui projek sumber terbuka CloudFlare-ImgBed. Ketika itu fungsinya masih agak asas, jadi saya mula menghantar beberapa PR untuk menambah perkara yang saya perlukan.  
Semakin saya mengubahnya, muncul satu idea: bolehkah ini menjadi alat pengehosan kandungan peribadi yang lengkap dan mudah digunakan oleh semua orang?  

Saya mengambil masa beberapa bulan membina semula projek ini sedikit demi sedikit dan memperkemas setiap fungsi hingga menjadi seperti sekarang. Sejujurnya, saya bukan pembangun profesional; saya masih belajar. Tanpa bantuan alat AI seperti Codex dan Claude, saya tidak mungkin dapat menyiapkan struktur projek yang begitu kompleks.  

Jika ImgBed benar-benar membantu anda, saya amat berharap anda sudi mempertimbangkan untuk menyokong pembangunannya. Walaupun sekadar harga secawan kopi, ia sangat bermakna bagi saya.
