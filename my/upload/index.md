# အပ်လုဒ် ဆက်တင်များ

အပ်လုဒ် ဆက်တင်များသည် ImgBed ကို သင်ကိုယ်တိုင်အသုံးပြုမည့် သိုလှောင်မှု ချန်နယ်များနှင့် ချိတ်ဆက်ပေးသည်။ ချန်နယ်တစ်ခုကို ပြင်ဆင်ပြီးပါက အပ်လုဒ်လုပ်ထားသော ပုံများနှင့် ဖိုင်များကို သင်ရွေးချယ်ထားသော ဝန်ဆောင်မှုတွင် သိမ်းဆည်းမည်ဖြစ်သည်။ ထို့နောက် ImgBed သည် ဖိုင်မှတ်တမ်းများ၊ ဝင်ရောက်သုံးစွဲရန် လင့်ခ်များ၊ preview များ၊ public gallery လုပ်ဆောင်ချက်များ၊ random image API အသုံးပြုခွင့်၊ WebDAV အသုံးပြုခွင့်နှင့် ဆက်စပ် workflow များကို စီမံပေးသည်။

အသုံးပြုသူအမျိုးမျိုးအတွက် သင့်တော်သည့် ချန်နယ်လည်း မတူနိုင်သည်။ ရိုးရှင်းသော ပြင်ဆင်မှုဖြင့် စတင်လိုပါက Telegram၊ Discord သို့မဟုတ် GitHub Releases ကို စတင်ရွေးချယ်နိုင်သည်။ သိုလှောင်နိုင်စွမ်း၊ မြန်နှုန်းနှင့် ရေရှည်တည်ငြိမ်မှုကို ပိုအရေးကြီးထားပါက Cloudflare R2၊ S3၊ OneDrive၊ Google Drive၊ Dropbox၊ Yandex၊ pCloud သို့မဟုတ် သင်၏ ကိုယ်ပိုင် WebDAV ဝန်ဆောင်မှုကို စဉ်းစားပါ။

## မစတင်မီ

- အသုံးပြုလိုသော သိုလှောင်မှု အကောင့် သို့မဟုတ် API credential များကို ပြင်ဆင်ထားပါ။
- OAuth အခြေပြု ချန်နယ်များသည် callback URL လိုအပ်သောကြောင့် သင်၏ ImgBed ဒိုမိန်းကို ဝင်ရောက်နိုင်ကြောင်း သေချာပါစေ။
- ချန်နယ်ထည့်ပြီးပါက ဖိုင်များကို မှန်ကန်စွာ သိမ်းဆည်းနိုင်၊ ဖွင့်နိုင်ကြောင်း စစ်ဆေးရန် စမ်းသပ်ပုံတစ်ပုံကို ဦးစွာ အပ်လုဒ်လုပ်ပါ။

## ဤအခန်းတွင် ပါဝင်သော အကြောင်းအရာများ

- အပ်လုဒ် ချန်နယ်တစ်ခုစီကို ပြင်ဆင်မတိုင်မီ လိုအပ်သည့်အရာများ။
- ပြင်ပ platform များတွင် app ဖန်တီးခြင်း၊ key ကူးယူခြင်း သို့မဟုတ် token ခွင့်ပြုခြင်းနည်းလမ်း။
- ချန်နယ် configuration ကို ImgBed ထဲသို့ ဖြည့်သွင်းပြီး အပ်လုဒ် အလုပ်လုပ်ကြောင်း အတည်ပြုနည်း။
