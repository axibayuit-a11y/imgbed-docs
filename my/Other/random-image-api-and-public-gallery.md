# ကျပန်းပုံနှင့် အများသုံးပြခန်း

လုပ်ဆောင်ချက်နှစ်ခုလုံးကို အောက်ပါနေရာတွင် သတ်မှတ်နိုင်သည်။

```text
System Settings -> Other Settings
```

## ကျပန်းပုံ API

ကျပန်းပုံ API သည် ရွေးထားသောဖိုင်တွဲများထဲမှ ဖိုင်တစ်ခုကို ကျပန်းရွေးပြီး ပြန်ပေးသည်။ ဆိုက်နောက်ခံပုံများ၊ ကိုယ်စားပြုပုံလှည့်ပြသခြင်း သို့မဟုတ် ပြင်ပစာမျက်နှာများမှ ကျပန်းပုံခေါ်ယူမှုများအတွက် အသုံးဝင်သည်။

ဖွင့်ပြီးနောက် အောက်ပါလိပ်စာကို အသုံးပြုပါ။

```text
https://your-domain.com/random
```

## ကျပန်းပုံ API ဆက်တင်များ

| ရွေးချယ်စရာ | ရည်ရွယ်ချက် |
| --- | --- |
| ဖွင့်ခြင်း | `/random` အဆုံးမှတ်ကို ဖွင့် သို့မဟုတ် ပိတ်သည်။ ပိတ်ထားပါက ဝင်ရောက်ခွင့်ကို ငြင်းပယ်သည်။ |
| ဖိုင်တွဲများ | ကျပန်း API အသုံးပြုနိုင်သော ဖိုင်တွဲများကို ကန့်သတ်သည်။ ဤနေရာတွင် မပါဝင်သော ဖိုင်တွဲများကို API က အသုံးမပြုနိုင်ပါ။ |
| ခေါ်ယူမှုနမူနာ | တိုက်ရိုက်ကူးယူနိုင်သော ကျပန်း API လင့်ခ်များကို ဖန်တီးပေးသည်။ |

ဖိုင်တွဲအများအပြားကို ရွေးချယ်နိုင်သည်။ ဥပမာ `/landscape/` နှင့် `/portrait/` ကိုသာ ခွင့်ပြုထားပါက ကျပန်း API သည် ထိုဖိုင်တွဲနှစ်ခုနှင့် ၎င်းတို့၏ ဖိုင်တွဲခွဲများထဲမှ ဖိုင်များကိုသာ ရွေးနိုင်သည်။

## ကျပန်းပုံ API ပါရာမီတာများ

| ပါရာမီတာ | ဥပမာ | ရည်ရွယ်ချက် |
| --- | --- | --- |
| `dir` | `/landscape/` | ကျပန်းရွေးမည့် ဖိုင်တွဲကို သတ်မှတ်သည်။ |
| `content` | `image` | မီဒီယာအမျိုးအစားကို သတ်မှတ်သည်။ `image`၊ `video`၊ `audio` သို့မဟုတ် ကော်မာဖြင့် ခွဲထားသော ပေါင်းစပ်တန်ဖိုးများကို အသုံးပြုပါ။ |
| `orientation` | `auto` | ပုံဦးတည်ချက်ကို စစ်ထုတ်သည်။ `portrait`၊ `landscape` သို့မဟုတ် `auto` ကို အသုံးပြုပါ။ |
| `type` | `url` | ပြန်ပေးမည့်ပုံစံ။ ဗလာထားပါက လမ်းကြောင်းပြောင်းသည်။ `url` သည် စာသား URL ပြန်ပေးပြီး `json` သည် JSON ပြန်ပေးသည်။ |
| `origin` | `1` | URL အပြည့် ပြန်ပေးရန် `type=url` နှင့်အတူ အသုံးပြုသည်။ |
| `age` | `all-ages,r12` | အသက်အဆင့်သတ်မှတ်ချက်ဖြင့် စစ်ထုတ်သည်။ |
| `tag` | `wallpaper,sky` | ဤတဂ်များပါဝင်သော ဖိုင်များကိုသာ ပြန်ပေးသည်။ |
| `ex` | `private` | ဤတဂ်များပါဝင်သော ဖိုင်များကို ဖယ်ထုတ်သည်။ |

## ပြန်ပေးသည့်ပုံစံများ

`type` မပါပါက API သည် ကျပန်းဖိုင် URL သို့ တိုက်ရိုက် လမ်းကြောင်းပြောင်းသည်။

`type=url` ပါပါက စာသား URL ကို ပြန်ပေးသည်။

`type=json` ပါပါက ဖိုင် URL၊ ဖိုင် ID၊ ဖိုင်အမည်၊ ဖိုင်အမျိုးအစား၊ တဂ်များ၊ အဆင့်သတ်မှတ်ချက်နှင့် ဆက်စပ်မက်တာဒေတာများပါဝင်သော ဖိုင်အချက်အလက်များကို ပြန်ပေးသည်။

## ဝင်ရောက်ခွင့်စည်းမျဉ်းများ

ကျပန်းပုံ API သည် အများသုံးဝင်ရောက်ခွင့်စည်းမျဉ်းများကို လိုက်နာသည်။

| စည်းမျဉ်း | သက်ရောက်မှု |
| --- | --- |
| ဖိုင်တွဲကန့်သတ်ချက် | ခွင့်ပြုထားသော ဖိုင်တွဲများထဲရှိ ဖိုင်များကိုသာ ရွေးချယ်နိုင်သည်။ |
| ပိတ်စာရင်း | ပိတ်စာရင်းထဲရှိ ဖိုင်များကို ကျပန်းရွေးချယ်စုမှ ဖယ်ထုတ်သည်။ |
| ခွင့်ပြုစာရင်းမုဒ် | ဖွင့်ထားပါက အများသုံးဝင်ရောက်ခွင့်အတွက် ခွင့်ပြုထားသော ဖိုင်များကိုသာ ပြန်ပေးသည်။ |
| အသက်အဆင့်သတ်မှတ်ချက် | R12၊ R16၊ R18 နှင့် အလားတူအကြောင်းအရာများကို လက်ရှိဝင်ရောက်ခွင့်မုဒ်အတိုင်း စစ်ထုတ်သည်။ |

စစ်ထုတ်ပြီးနောက် ကိုက်ညီသော ဖိုင်မရှိပါက API သည် ကိုက်ညီသောရလဒ်မရှိကြောင်း ပြန်ပေးသည်။

## ကက်ရှ်

ကျပန်းပုံ API သည် မြန်နှုန်းတိုးစေရန် ဖိုင်တွဲအလိုက် ကိုယ်စားလှယ်ဖိုင်စုများကို ကက်ရှ်လုပ်သည်။

ဖိုင်များ ပြောင်းလဲပြီးနောက် ImgBed သည် ဖိုင်တွဲကက်ရှ်ဗားရှင်းကို အပ်ဒိတ်လုပ်ပြီး နောက်လာမည့်တောင်းဆိုမှုများတွင် ကိုယ်စားလှယ်ဖိုင်စုကို ပြန်လည်တည်ဆောက်သည်။ မကြာခဏ မေးမြန်းမှု မဖြစ်စေရန် ဖိုင်မရှိသော ဖိုင်တွဲများကို ခဏသာ ကက်ရှ်လုပ်ထားသည်။

## အများသုံးပြခန်း

အများသုံးပြခန်းသည် ဧည့်သည်များကြည့်ခွင့်ရှိသော ဖိုင်တွဲများအတွက် ဖတ်ရန်သာဖြစ်သော အများသုံးကြည့်ရှုစာမျက်နှာကို ပေးသည်။

ဖွင့်ပြီးနောက် ဧည့်သည်များသည် အောက်ပါလိပ်စာကို ဖွင့်နိုင်သည်။

```text
https://your-domain.com/browse/directory-name
```

## အများသုံးပြခန်း ဆက်တင်များ

| ရွေးချယ်စရာ | ရည်ရွယ်ချက် |
| --- | --- |
| ဖွင့်ခြင်း | အများသုံးပြခန်းကို ဖွင့် သို့မဟုတ် ပိတ်သည်။ ပိတ်ထားပါက ဧည့်သည်များ မကြည့်ရှုနိုင်ပါ။ |
| ပုံတင်ပုံမုဒ် | အစမ်းကြည့်ပုံများတွင် မူရင်းပုံများ သို့မဟုတ် ပုံငယ်များကို အသုံးပြုမည်ကို ထိန်းချုပ်သည်။ |
| ဖွင့်ထားသောဖိုင်တွဲများ | ဧည့်သည်များ ဝင်ရောက်နိုင်သော ဖိုင်တွဲများကို သတ်မှတ်သည်။ |

## ပုံတင်ပုံမုဒ်

| မုဒ် | ရည်ရွယ်ချက် |
| --- | --- |
| မူရင်း | ဧည့်သည်စာမျက်နှာသည် မူရင်းဖိုင်များကို တိုက်ရိုက်တင်သည်။ |
| ပုံငယ် | ဧည့်သည်စာမျက်နှာသည် ပိုမြန်စွာတင်နိုင်ရန် ပုံငယ်များကို ဦးစားပေးအသုံးပြုသည်။ |

## ဖွင့်ထားသောဖိုင်တွဲများ

ဖွင့်ထားသောဖိုင်တွဲများသည် ဧည့်သည်များ မြင်နိုင်သည့်အရာကို ဆုံးဖြတ်သည်။

ဥပမာ:

```text
/1/,/2/,/landscape/,/portrait/
```

ထို့နောက် ဧည့်သည်များသည် အောက်ပါတို့ကို ဝင်ရောက်နိုင်သည်။

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

`/2026/lucky/` ကဲ့သို့သော ဖိုင်တွဲခွဲများကိုလည်း ဖွင့်ထားနိုင်သည်။ ဖွင့်မထားသော ဖိုင်တွဲများကို ဧည့်သည်များ ဝင်ရောက်ခွင့်မရှိပါ။

| ရှာဖွေခြင်း | ဖိုင်အမည်၊ ဖိုင် ID သို့မဟုတ် တဂ်များဖြင့် ရှာဖွေနိုင်သည်။ |
| အမျိုးအစားစစ်ထုတ်ခြင်း | ပုံများ၊ ဗီဒီယိုများ၊ အသံများ သို့မဟုတ် အခြားဖိုင်များကို စစ်ထုတ်နိုင်သည်။ |
| တဂ်စစ်ထုတ်ခြင်း | ရွေးထားသော တဂ်များကို ထည့်သွင်းခြင်း သို့မဟုတ် ဖယ်ထုတ်ခြင်း ပြုလုပ်နိုင်သည်။ |
| ဦးတည်ချက်စစ်ထုတ်ခြင်း | အလျားလိုက် သို့မဟုတ် ဒေါင်လိုက်ပုံများကို စစ်ထုတ်နိုင်သည်။ |
| အချိန်စစ်ထုတ်ခြင်း | အပ်လုဒ်တင်ချိန်အပိုင်းအခြားဖြင့် စစ်ထုတ်နိုင်သည်။ |
| ဖိုင်နောက်ဆက်တွဲစစ်ထုတ်ခြင်း | ဖိုင်နောက်ဆက်တွဲဖြင့် စစ်ထုတ်နိုင်သည်။ |
| လင့်ခ်ကူးယူခြင်း | ဖိုင်ဝင်ရောက်ရန် လင့်ခ်များကို ကူးယူနိုင်သည်။ |

အများသုံးပြခန်းသည်လည်း အများသုံးဝင်ရောက်ခွင့်စည်းမျဉ်းများကို လိုက်နာသည်။

| ဝင်ရောက်ခွင့်မုဒ် | အကြောင်းအရာကို လက်ရှိအသက်အဆင့်သတ်မှတ်ချက်ဝင်ရောက်ခွင့်မုဒ်အတိုင်း စစ်ထုတ်သည်။ |
