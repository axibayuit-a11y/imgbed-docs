# ImgBed

Универсальный инструмент для личного хостинга контента на базе экосистемы Cloudflare.

ImgBed начинался с загрузки изображений и управления ссылками. Теперь он вырос в полноценное решение для личного хостинга контента. С его помощью можно:

- Загружать файлы в любой предпочитаемый канал хранения
- Публично делиться изображениями и файлами любого формата
- Быстро развернуть личный блог за одну минуту
- Одним кликом сохранять файлы из magnet-ссылок
- Присоединяться к федерации узлов ImgBed и делиться открытыми файловыми индексами

Мы не верим в решения "один размер для всех". Хотите ли вы просто хранить изображения, массово управлять альбомами, писать небольшие заметки или создать легкий публичный сайт с ресурсами, ImgBed уважает ваш выбор и оставляет вам максимум гибкости.

## Мощные возможности сразу из коробки

- 🌍 Поддержка 28 языков интерфейса для пользователей по всему миру
- 📦 13 встроенных основных каналов загрузки: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex и pCloud
- 🧩 Полностью модульная архитектура: включайте функции только по необходимости и не тратьте ресурсы зря
- 🔐 Поддерживает API Token, режимы доступа и правила списков для точного управления правами на контент

Проект продолжает развиваться. Наши возможности ограничены, но мы сделаем все возможное, чтобы каждая функция была понятной, стабильной и удобной. Если ImgBed поможет вам проще получить собственный хостинг изображений, блог или пространство для обмена файлами, мы будем очень рады.

## Начать работу

Разверните ImgBed через [https://glow.ccwu.cc/](https://glow.ccwu.cc/) и быстро получите собственный сервис для хостинга изображений и файлов.
