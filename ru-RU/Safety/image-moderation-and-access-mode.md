# Модерация изображений и режим доступа

Модерация изображений присваивает загруженным файлам возрастные рейтинги. Режим доступа определяет, какие рейтинги можно показывать через публичный доступ.

Это влияет на публичную галерею, публичные URL файлов и API случайных изображений. На админ-панель ограничение не распространяется: администраторы все равно могут просматривать и управлять всеми файлами.

## Где это настраивается

Откройте админ-панель и перейдите в:

```text
System Settings -> Security Settings -> Upload Management -> Image Moderation
```

Основные настройки:

- Режим доступа
- Включить модерацию
- Провайдер модерации

## Что делает режим доступа

Режим доступа определяет, какие возрастные рейтинги могут быть видимы публично.

Текущие режимы:

| Режим доступа | Публично видимые рейтинги |
| --- | --- |
| Режим для взрослых | General, R12, R16, R18 |
| Молодежный режим | General, R12, R16 |
| Подростковый режим | General, R12 |
| Детский режим | Только General |

По умолчанию используется режим для взрослых.

Для приватных сайтов или сайтов с контентом для взрослых этот режим может быть уместен. Для более строгой публичной галереи выберите молодежный, подростковый или детский режим.

## Что дает включение модерации

Когда модерация включена, ImgBed во время загрузки вызывает выбранного провайдера модерации и сохраняет определенный возрастной рейтинг.

Основные рейтинги:

| Рейтинг | Значение |
| --- | --- |
| General | Безопасный публичный контент |
| R12 | Слегка чувствительный контент |
| R16 | Более чувствительный контент |
| R18 | Контент для взрослых |

Результат модерации используется при определении публичного доступа.

Если модерация не включена или старые файлы не имеют рейтинга, такие файлы считаются неоцененными. Неоцененные файлы не удаляются автоматически из публичной галереи или API случайных изображений только из-за отсутствия рейтинга.

## Выбор провайдера модерации

Доступные провайдеры:

- moderatecontent.com
- nsfwjs
- Sightengine

У каждого провайдера свои требования:

- moderatecontent.com обычно требует API Key.
- nsfwjs обычно требует URL конечной точки API.
- Sightengine требует API user и API secret.

Выбирайте по своему аккаунту, доступности и качеству распознавания. Если модерация включена и настроена правильно, ImgBed пытается записать рейтинг изображения во время загрузки.

## Влияние на публичную галерею

Публичная галерея фильтрует файлы по режиму доступа.

Примеры:

- Режим для взрослых: R18-изображения могут появляться.
- Молодежный режим: R18-изображения скрыты.
- Подростковый режим: R16 и R18 скрыты.
- Детский режим: показываются только изображения с рейтингом General.

Это влияет только на обычный публичный доступ. Админ-панель продолжает показывать все файлы.

## Влияние на публичные URL файлов

Публичные URL файлов — это прямые ссылки на изображения, которые открывают посетители.

Если рейтинг файла разрешен текущим режимом доступа, ImgBed возвращает исходное изображение.

Если рейтинг выше разрешенного уровня, обычный публичный доступ не возвращает исходное изображение. Вместо этого ImgBed возвращает настроенный результат блокировки или изображение-заглушку.

Пример:

- Текущий режим — детский.
- Изображение имеет рейтинг R18.
- Посетитель открывает публичный URL напрямую.
- ImgBed не возвращает этому посетителю исходное R18-изображение.

Администраторы, просматривающие файлы в админ-панели, этим ограничением не затрагиваются.

## Влияние на API случайных изображений

API случайных изображений также фильтрует пул кандидатов по режиму доступа.

В детском режиме случайные изображения выбираются только из файлов с рейтингом General.

В молодежном режиме случайные изображения могут быть из General, R12 и R16, но не из R18.

Так API случайных изображений не обходит ограничения публичной галереи.

## Связь с правилами списков

Режим доступа — не единственное правило публичного доступа. Он работает вместе с правилами разрешающих и блокирующих списков.

Проще говоря:

- контент из разрешающего списка сначала считается публичным;
- контент из блокирующего списка не может просматриваться напрямую обычными посетителями;
- контент, которого нет ни в одном списке, затем проверяется по режиму доступа.

Если изображение ограничено и возрастным рейтингом, и правилами списков, обычные посетители все равно не смогут напрямую увидеть исходный файл.

## Рекомендуемые настройки

Для публичных сайтов:

- Включите модерацию.
- Выберите режим доступа по аудитории сайта.
- Для посетителей любого возраста используйте детский или подростковый режим.
- Избегайте режима для взрослых, если не хотите показывать взрослый контент публично.
- Просматривайте рейтинги файлов в админ-панели и при необходимости меняйте их вручную.

Для приватных или личных сайтов:

- Режим для взрослых обычно подходит.
- Включите модерацию, если она полезна.
- Просматривайте и корректируйте рейтинги в админ-панели по необходимости.

## FAQ

### Исчезнут ли файлы из админ-панели после смены режима доступа?

Нет.

Режим доступа влияет только на обычный публичный доступ. На админ-панель он не влияет.

### Почему после перехода на детский режим в публичной галерее стало меньше изображений?

Детский режим позволяет публично показывать только файлы с рейтингом General. R12, R16 и R18 фильтруются.

### Могут ли публичные URL открывать изображения для взрослых?

Если текущий режим доступа не разрешает этот рейтинг, обычные публичные URL не возвращают исходное изображение.

### Может ли API случайных изображений вернуть ограниченные изображения?

Нет.

API случайных изображений фильтрует кандидатов по текущему режиму доступа.

### Что будет со старыми изображениями без рейтинга?

Изображения без рейтинга не скрываются автоматически только из-за отсутствия результатов модерации. Их рейтинги можно позже изменить в админ-панели.
