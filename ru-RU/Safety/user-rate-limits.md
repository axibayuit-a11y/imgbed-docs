# Ограничения частоты загрузок для пользователей

Ограничения частоты загрузок управляют тем, как часто обычные пользователи или посетители могут загружать файлы с главной страницы. Это помогает защитить публичную страницу загрузки от злоупотреблений.

Функция влияет только на загрузки с главной страницы. Загрузки администратора и загрузки через API Tokens не ограничиваются этими правилами частоты.

## Где это настраивается

Откройте админ-панель и перейдите в:

```text
System Settings -> Security Settings -> Upload Management -> User Rate Limits
```

![Настройки ограничения частоты загрузок](../../image/other/用户频控截图.png)

## Включение ограничений частоты

После включения `Включить ограничения частоты` ImgBed отслеживает недавние загрузки по IP-адресу отправителя.

Значения по умолчанию:

| Настройка | По умолчанию | Описание |
| --- | --- | --- |
| Окно обнаружения | 1.5 часа | Насколько далеко назад считаются записи загрузок. |
| Максимальное число файлов | 20 | Максимальное число файлов в окне обнаружения. |
| Лимит размера одного файла | 20 MB | Максимальный размер одного файла. |
| Лимит общего размера загрузок | 200 MB | Максимальный общий размер загрузок в окне обнаружения. |

Например, с окном 1,5 часа, 20 файлами, 20 MB на файл и 200 MB всего загрузки с того же IP блокируются, как только превышен любой настроенный лимит.

## Исключение типов файлов

`Исключенные типы загружаемых файлов` блокирует обычным пользователям или посетителям загрузку выбранных категорий файлов.

Доступные категории:

| Тип | Описание |
| --- | --- |
| Изображения | jpg, png, webp, gif и похожие файлы изображений |
| Видео | mp4, webm, mov и похожие видеофайлы |
| Аудио | mp3, flac, wav и похожие аудиофайлы |
| Документы | pdf, txt, md, docx и похожие документы |
| Другое | Файлы вне этих категорий, например zip, rar, exe, apk |

По умолчанию категория не выбрана, то есть она разрешена.

Нажатие на категорию подсвечивает ее, а это означает, что тип заблокирован.

Если выбрать `Другое`, посетители, загружающие zip или rar, будут заблокированы и увидят сообщение, что этот тип файла не поддерживается.

## Сообщения о блокировке

Когда срабатывает лимит, пользователи видят соответствующее сообщение:

![Сообщение о слишком частой загрузке](../../image/other/频繁报错提示.png)

| Сценарий | Значение сообщения |
| --- | --- |
| Один файл слишком большой | Файл слишком большой, его стоит сжать перед загрузкой. |
| Тип файла заблокирован | Этот тип файла не поддерживается. Уберите его и попробуйте снова. |
| Загрузки слишком частые | Недавние загрузки слишком частые, показано время повторной попытки. |
| Общий размер слишком большой | Суммарный размер недавних загрузок слишком велик, показано время повторной попытки. |

## Когда включать

Включите ограничения частоты загрузок, если ваша главная страница загрузки доступна публично.

Типичные причины:

- вы опасаетесь массовых автоматизированных загрузок;
- хотите ограничить большие загрузки от посетителей;
- хотите разрешить обычным пользователям загружать только изображения, а не архивы или установщики;
- хотите оставить публичную загрузку доступной, но контролировать расход ресурсов.

Если сайт только для вас или загружать могут только администраторы, это можно оставить выключенным.
