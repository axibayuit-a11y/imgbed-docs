# Автоматическое тегирование

Автоматическое тегирование настраивается в разделе:

```text
System Settings -> Other Settings -> Auto Tagging
```

Оно автоматически создает теги для изображений. Эти теги полезны для поиска, фильтрации в API случайного изображения, фильтрации в публичной галерее и управления доступом по возрастному рейтингу.

## Что умеет автоматическое тегирование

| Возможность | Описание |
| --- | --- |
| Генерация тегов содержимого | Добавляет теги для людей, сцен, объектов, художественного стиля и похожего визуального содержимого. |
| Генерация тегов персонажей | Полезно для аниме-изображений и иллюстраций. |
| Добавление тегов ориентации | Добавляет `landscape`, `portrait` или `square`. |
| Добавление рейтинга изображения | Сохраняет результаты рейтинга `G/S/Q/E` для общего, чувствительного, сомнительного или откровенного контента. |
| Автотегирование при загрузке | Новые изображения автоматически попадают в процесс тегирования. |
| Пакетное тегирование | Добавляет теги к старым изображениям во всех папках или в выбранных папках. |

## Что нужно подготовить сначала

Подготовьте как минимум один доступный URL Hugging Face Space.

Рекомендуемый способ — дублировать Space SmilingWolf `wd-tagger` в свою учетную запись Hugging Face:

```text
https://huggingface.co/spaces/SmilingWolf/wd-tagger
```

Вы можете временно использовать публичный Space, но публичные Spaces используются многими пользователями и могут вставать в очередь, замедляться или становиться недоступными. Дублированный Space в вашей учетной записи стабильнее для долгосрочного автоматического тегирования.

## Дублирование Space SmilingWolf

1. Войдите в Hugging Face.
2. Откройте `https://huggingface.co/spaces/SmilingWolf/wd-tagger`.

![Публичный Space SmilingWolf](../../image/other/微笑狼的公开仓库.png)

3. Нажмите меню с тремя точками в правом верхнем углу.
4. Выберите `Duplicate this Space`.
5. Оставьте имя Space по умолчанию или задайте свое, например `wd-tagger`.
6. Установите видимость `Public`. Публичные Spaces проще вызывать из ImgBed.
7. Сначала оставьте бесплатную конфигурацию оборудования по умолчанию. Обновляйте ее позже только при заметных очередях.
8. Создайте Space и дождитесь завершения сборки.

После завершения сборки откройте страницу своего Space. URL обычно выглядит так:

```text
https://huggingface.co/spaces/your-name/wd-tagger
```

Скопируйте URL из браузера и вставьте его в `Space URLs` в ImgBed.

## Заполнение нескольких Space URLs

Вводите по одному URL Space в каждой строке.

Примеры:

| Значение | Описание |
| --- | --- |
| `https://huggingface.co/spaces/SmilingWolf/wd-tagger` | Публичный Space SmilingWolf. Подходит для временного тестирования. |
| `https://huggingface.co/spaces/lintonxue00/wd-tagger` | URL страницы скопированного Space. |
| `https://huggingface.co/spaces/your-name/wd-tagger` | Ваш собственный дублированный Space. |

Можно указать несколько URL. ImgBed использует несколько Spaces вместе, что может повысить скорость.

Если один Space временно недоступен, остальные могут продолжить обработку.

## Настройки

| Параметр | Рекомендация |
| --- | --- |
| `Space URLs` | Введите подготовленные URL Space. Укажите как минимум один. |
| Целевая папка | Оставьте пустым для всех папок. Выбирайте папку только тогда, когда нужно обработать конкретный каталог. |
| Модель распознавания | По умолчанию оставьте `wd-swinv2-tagger-v3`. |
| Порог общих тегов | Значение по умолчанию подходит большинству изображений. Более низкие значения дают больше тегов, более высокие — меньше. |
| Порог тегов персонажей | Значение по умолчанию консервативно и помогает избегать неправильных тегов персонажей. |
| Автоматический порог `MCut` | Сначала оставьте выключенным. Включайте, когда хотите, чтобы модель сама определяла количество тегов. |
| Автотегирование при загрузке | Включите, если новые изображения должны автоматически получать теги. |
| Запуск тегирования | Вручную запускает пакетное тегирование старых изображений. |

## Рекомендуемые начальные значения

| Параметр | Рекомендуемое значение |
| --- | --- |
| Модель распознавания | `wd-swinv2-tagger-v3` |
| Порог общих тегов | `0.35` |
| Порог тегов персонажей | `0.85` |
| `MCut` | Сначала выключено |
| Автотегирование при загрузке | Включите при необходимости |

Если тегов слишком много, немного увеличьте порог общих тегов.

Если тегов слишком мало, немного уменьшите порог общих тегов.

## Пакетное тегирование

1. Заполните `Space URLs`.
2. Выберите целевую папку.
3. Запустите тегирование.
4. Дождитесь завершения прогресса.

Если целевая папка пуста, ImgBed обрабатывает все папки.

Пакетное тегирование лучше всего подходит для старых изображений. Для новых изображений включите автотегирование при загрузке, чтобы не запускать его вручную каждый раз.

## Автотегирование при загрузке

После включения автотегирования при загрузке новые изображения автоматически вызывают настроенные `Space URLs`.

Это подходит для долгосрочного использования.

Если ваш Space стоит в очереди, сама загрузка все равно может завершиться первой, а тегирование продолжится после нее.

## Какие изображения обрабатываются

Автоматическое тегирование в основном обрабатывает файлы изображений.

Изображения, у которых уже есть полные теги, ориентация, рейтинг, ширина и высота, пропускаются, чтобы избежать лишних вызовов Space.

ImgBed по возможности заполняет только недостающую информацию. Например, если не хватает только ориентации, он пытается добавить ориентацию без запуска полного процесса генерации тегов содержимого.

## Частые вопросы

### Зачем дублировать собственный Space?

Публичные Spaces используются многими пользователями. Ваш собственный дублированный Space в основном используется вашим сайтом ImgBed, поэтому обычно он быстрее и надежнее.

### Space постоянно запускается

После первого создания или после долгого простоя Space может требовать времени на запуск.

Сначала откройте страницу своего Space. Когда он сможет нормально распознать изображение, вернитесь в ImgBed и запустите тегирование.

### Как скопировать URL Space?

Откройте страницу своего Hugging Face Space и скопируйте адрес из браузера.

Примеры:

```text
https://huggingface.co/spaces/lintonxue00/wd-tagger
https://huggingface.co/spaces/SmilingWolf/wd-tagger
```

### Можно ли добавить несколько Spaces?

Да. Вводите по одному URL Space в каждой строке.

Несколько Spaces обрабатывают изображения вместе и полезны, когда изображений много.

### Почему теги на английском?

Модели SmilingWolf выводят английские теги. Это ожидаемо.

Теги в основном используются для поиска, фильтрации, API случайного изображения и фильтров публичной галереи.

### Для чего нужны рейтинговые теги?

Результаты рейтинга работают вместе с режимом доступа в настройках безопасности.

Например, когда доступ посетителей ограничен возрастным рейтингом, публичный просмотр и функции случайного изображения фильтруют изображения по этим правилам.

## Краткий порядок действий

```text
Sign in to Hugging Face
-> Open SmilingWolf/wd-tagger
-> Duplicate this Space
-> Wait for the Space to build
-> Copy your Space URL
-> Fill Space URLs in ImgBed
-> Choose model and thresholds
-> Start tagging or enable auto-tag on upload
```
