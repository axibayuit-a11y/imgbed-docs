# Magnet Transfer

Magnet transfer скачивает файлы по magnet link и автоматически загружает их в выбранный cloud storage channel.

Это удобно для переноса anime-серий, видео, архивов и похожих файлов. Вставьте magnet link, и ImgBed создаст фоновую задачу загрузки. После завершения файл будет загружен в ImgBed, а финальная ссылка появится в списке загрузок.

![Magnet transfer](../../image/other/磁力链接/磁力链接.png)

## Где использовать

Вход для magnet transfer находится в зоне загрузки на главной странице.

Вставьте magnet link в поле ввода, выберите `Transfer`, затем загрузите.

![Загрузка anime](../../image/other/磁力链接/上传番剧.png)

## Перед первым использованием

Сначала настройте magnet transfer в админ-панели.

Обычно нужны:

1. GitHub-аккаунт для запуска download task.
2. Cloud upload channel, например Google Drive или OneDrive.
3. Целевая upload directory.
4. Task timeout.

Когда настройки готовы, вернитесь на главную страницу и вставьте magnet link, чтобы начать transfer.

## Загрузка magnet link

1. Вставьте magnet link в поле загрузки на главной странице.
2. Убедитесь, что режим установлен как `Transfer`.
3. Нажмите upload.
4. Дождитесь, пока ImgBed создаст magnet task.
5. После запуска задачи используйте плавающую панель `Magnet Tasks` в правом нижнем углу, чтобы следить за прогрессом.

Download и upload могут занять время. Скорость зависит от magnet-ресурса, среды GitHub runtime и выбранного cloud storage channel.

![Magnet скачивается](../../image/other/磁力链接/磁力链接下载中.png)

## После завершения

После завершения задачи в списке загрузок появятся имя файла и ссылка.

Для видео показывается video preview, для изображений — image preview, а для остальных файлов — обычная иконка файла.

![Скачанное видео](../../image/other/磁力链接/下载好后的视频.png)

Можно скопировать:

| Тип ссылки | Для чего |
| --- | --- |
| Original link | Прямой доступ к файлу |
| Markdown | Markdown-посты или заметки |
| HTML | Код веб-страницы |
| BBCode | Форумы с поддержкой BBCode |

## Панель Magnet Task

Панель magnet task в правом нижнем углу показывает количество задач, название, прогресс и финальный статус.

Типичные состояния:

| Статус | Значение |
| --- | --- |
| Waiting | Задача создана и ждет запуска. |
| Downloading | Magnet-ресурс скачивается. |
| Uploading | Файл скачан и передается в cloud storage. |
| Completed | Upload успешен, ссылку можно копировать. |
| Failed | Задача не завершилась успешно. Проверьте сообщение и попробуйте снова. |

## Советы

- Если magnet link содержит несколько файлов, ImgBed старается показать основной завершенный файл.
- Большие файлы требуют больше времени. Дождитесь завершения задачи перед обновлением страницы.
- Если у magnet-ресурса нет доступных peers, он может быть очень медленным или не скачаться.
- Если в cloud account закончилась квота, authorization истекла или upload directory указана неверно, задача может завершиться ошибкой.
- Video preview может появиться через несколько секунд после завершения upload.

## FAQ

### После вставки Magnet Link ничего не запускается

Убедитесь, что magnet transfer включен в админ-панели, а также выбран пригодный GitHub-аккаунт и cloud channel.

### Скачивание всегда медленное

Скорость magnet зависит от самого ресурса. Если нет доступных peers, скачивание может быть очень медленным или невозможным.

### После upload нет preview

Сначала проверьте, открывается ли ссылка на файл. Видеофайлам может понадобиться немного времени для загрузки в браузере; также можно открыть ссылку напрямую.

### Что проверить, если task failed?

Проверьте, действителен ли magnet link, работает ли cloud channel и правильна ли upload directory. Затем отправьте задачу еще раз.
