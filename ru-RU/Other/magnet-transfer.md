# Передача по magnet-ссылке

Передача по magnet-ссылке скачивает файлы из magnet-ссылки и автоматически загружает их в выбранный канал облачного хранилища.

Она полезна для переноса серий аниме, видео, архивов и похожих файлов. Вставьте magnet-ссылку, и ImgBed создаст фоновую задачу скачивания. После завершения скачивания файл будет загружен в ImgBed, а итоговая ссылка появится в списке загрузок.

![Передача по magnet-ссылке](../../image/other/磁力链接/磁力链接.png)

## Где использовать

Точка входа для передачи по magnet-ссылке находится в области загрузки на главной странице.

Вставьте magnet-ссылку в поле ввода, выберите `Transfer`, затем выполните загрузку.

![Загрузка аниме](../../image/other/磁力链接/上传番剧.png)

## Перед первым использованием

Сначала настройте передачу по magnet-ссылке в панели администратора.

Обычно нужны:

1. Учетная запись GitHub для запуска задачи скачивания.
2. Канал загрузки в облако, например Google Drive или OneDrive.
3. Целевой каталог загрузки.
4. Тайм-аут задачи.

Когда настройки будут готовы, вернитесь на главную страницу и вставьте magnet-ссылку, чтобы начать передачу.

## Загрузка magnet-ссылки

1. Вставьте magnet-ссылку в поле загрузки на главной странице.
2. Убедитесь, что режим установлен на `Transfer`.
3. Нажмите кнопку загрузки.
4. Дождитесь, пока ImgBed создаст magnet-задачу.
5. После запуска задачи используйте плавающую панель `Magnet Tasks` в правом нижнем углу, чтобы проверить прогресс.

Скачивание и загрузка могут занять время. Скорость зависит от magnet-ресурса, среды выполнения GitHub и выбранного канала облачного хранилища.

![Скачивание magnet-ссылки](../../image/other/磁力链接/磁力链接下载中.png)

## После завершения

После завершения задачи список загрузок показывает имя файла и ссылку.

Для видео отображается предпросмотр видео, для изображений — предпросмотр изображения, а для остальных файлов — обычный значок файла.

![Скачанное видео](../../image/other/磁力链接/下载好后的视频.png)

Можно скопировать:

| Тип ссылки | Сценарий использования |
| --- | --- |
| Исходная ссылка | Прямой доступ к файлу |
| Markdown | Посты или заметки в Markdown |
| HTML | Код веб-страницы |
| BBCode | Форумы с поддержкой BBCode |

## Панель magnet-задач

Панель magnet-задач в правом нижнем углу показывает количество задач, имя задачи, прогресс и итоговый статус.

Типичные состояния:

| Статус | Значение |
| --- | --- |
| Ожидание | Задача создана и ожидает запуска. |
| Скачивание | Magnet-ресурс скачивается. |
| Загрузка | Файл скачан и загружается в облачное хранилище. |
| Завершено | Загрузка успешно завершена, ссылку можно скопировать. |
| Ошибка | Задача не завершилась успешно. Проверьте сообщение и попробуйте снова. |

## Советы

- Если magnet-ссылка содержит несколько файлов, ImgBed в первую очередь показывает основной завершенный файл.
- Большие файлы требуют больше времени. Дождитесь завершения задачи перед обновлением страницы.
- Если у magnet-ресурса нет доступных пиров, он может скачиваться очень медленно или завершиться ошибкой.
- Если в облачной учетной записи закончилась квота, авторизация истекла или каталог загрузки указан неверно, задача может завершиться ошибкой.
- Предпросмотр видео может появиться через несколько секунд после завершения загрузки.

## FAQ

### Ничего не запускается после вставки magnet-ссылки

Убедитесь, что передача по magnet-ссылке включена в панели администратора и выбраны доступная учетная запись GitHub и облачный канал.

### Скачивание всегда медленное

Скорость magnet зависит от самого ресурса. Если доступных пиров нет, скачивание может быть очень медленным или невозможным.

### После загрузки нет предпросмотра

Сначала проверьте, открывается ли ссылка на файл. Видеофайлам может потребоваться немного времени для загрузки в браузере, либо вы можете открыть ссылку напрямую.

### Что проверить, если задача завершилась ошибкой?

Проверьте, действительна ли magnet-ссылка, работает ли облачный канал и правильно ли указан каталог загрузки. Затем отправьте задачу снова.
