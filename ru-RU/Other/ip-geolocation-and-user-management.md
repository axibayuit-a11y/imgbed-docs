# IP-геолокация и управление пользователями

IP-геолокация преобразует IP-адреса в записях об отправителях файлов, устройствах входа и похожих журналах в приблизительные местоположения.

После настройки панель администратора может понятнее показывать источники загрузок и обращений. Управление пользователями также позволяет блокировать или восстанавливать доступ к загрузке для подозрительных IP-адресов.

## Где это настроить

Откройте:

```text
System Settings -> Other Settings -> IP Geolocation
```

![IP-геолокация](../../image/other/ip定位/ip定位.png)

## Доступные настройки

Новый процесс IP-геолокации поддерживает несколько источников, а не зависит от одного картографического сервиса.

| Настройка | Назначение |
| --- | --- |
| Язык IP-геолокации | Выбирает язык отображения, например английский, упрощенный китайский, японский, французский и другие. |
| MaxMind Account ID | ID учетной записи MaxMind для MaxMind GeoLite Web Service. |
| MaxMind License Key | License Key MaxMind. |
| Tencent Map Key | Ключ Tencent Location Service. Полезен для китайских адресов и IP-адресов материкового Китая. |
| ipapi Key | Ключ APILayer ipapi. Поддерживает многоязычную IP-геолокацию. |

Заполняйте только те сервисы, которые вам нужны. Не обязательно настраивать каждое поле.

Если ключ не указан, ImgBed все равно попробует встроенные бесплатные источники, но стабильность, языковая поддержка и точность могут быть ниже, чем у сервиса, настроенного вами самостоятельно.

## Рекомендуемый выбор

Если в основном нужны китайские адреса:

1. Установите язык IP-геолокации на упрощенный китайский.
2. Настройте Tencent Map Key.
3. При желании добавьте MaxMind или ipapi как резервные источники.

Если в основном нужны английские или многоязычные адреса:

1. Выберите нужный язык.
2. Настройте MaxMind Account ID и License Key.
3. Добавьте ipapi Key, если нужны более качественные многоязычные результаты.

## Настройка MaxMind

Для MaxMind нужны:

```text
MaxMind Account ID
MaxMind License Key
```

Найдите ID учетной записи в панели MaxMind и создайте License Key на странице License Keys.

![Настройка ключа MaxMind](../../image/other/ip定位/maxmind的key配置.png)

После создания вставьте Account ID и License Key в ImgBed и сохраните.

Бесплатный план MaxMind подходит для повседневного использования, но у него есть лимиты запросов. Если квота превышена, ImgBed продолжит пробовать другие доступные источники.

## Настройка ipapi

ipapi использует APILayer API Key.

Откройте консоль ipapi и скопируйте показанный там API Key.

![Настройка ipapi](../../image/other/ip定位/ipapi配置.png)

Вставьте его в поле `ipapi Key` в ImgBed и сохраните.

ipapi поддерживает многоязычную IP-геолокацию и полезен, когда нужно показывать адреса на выбранном языке. У его бесплатного плана также есть лимиты запросов. Если квота закончится, ImgBed продолжит пробовать другие доступные источники.

## Настройка Tencent Map Key

Tencent Map Key полезен для китайских адресов, особенно для IP-адресов материкового Китая.

При создании ключа в Tencent Location Service включите:

```text
WebServiceAPI
```

После создания вставьте ключ в `Tencent Map Key` и сохраните.

Если нужна только базовая китайская IP-геолокация, Tencent Map Key достаточно для начала.

## Что проверять в управлении пользователями

Управление пользователями доступно в верхней части панели администратора.

![Управление пользователями](../../image/other/用户管理显示.png)

Управление пользователями показывает активность загрузок по IP:

| Поле | Описание |
| --- | --- |
| Источник IP | IP-адрес источника загрузки. |
| Адрес | Приблизительное местоположение, определенное по IP. |
| Общий размер загрузок | Общий размер файлов, загруженных с этого IP. |
| Количество загрузок | Количество загрузок с этого IP. |
| Загрузка разрешена | Включено означает, что загрузки разрешены. Выключено означает, что загрузки заблокированы. |

Нажмите стрелку слева, чтобы развернуть список файлов, загруженных с этого IP.

В списке файлов отображаются имя файла, предварительный просмотр, размер файла, результат модерации, статус файла и время загрузки. Если загрузки выглядят подозрительно, сначала разверните IP, просмотрите файлы, а затем решите, нужно ли блокировать дальнейшие загрузки.

Если IP подозрителен, выключите `Upload allowed`. Будущие загрузки с этого IP будут заблокированы.

## Поиск, сортировка и расширенные фильтры

В верхней части управления пользователями можно искать по источнику IP или адресу.

Сортируйте по времени, количеству загрузок или общему размеру загрузок, чтобы найти недавние загрузки, IP с высокой частотой загрузок или IP с большим объемом использования.

Для более глубокой проверки откройте расширенные фильтры.

![Расширенные фильтры](../../image/other/用户管理高级筛选.png)

Расширенные фильтры поддерживают:

| Фильтр | Использование |
| --- | --- |
| Диапазон времени | Показывает IP, которые загружали файлы за выбранный период. |
| Статус доступа | Фильтрует по нормальному, заблокированному и похожим состояниям. |
| Список разрешения/блокировки | Фильтрует по списку разрешения, списку блокировки или отсутствию настройки. |
| Тип файла | Показывает IP, которые загружали изображения, видео, аудио, документы, код или другие файлы. |
| Размер файла | Фильтрует по диапазону размера загруженных файлов. |
| Возрастной рейтинг | Фильтрует по отсутствию настройки, General, R12+, R16+, R18 и похожим рейтингам. |
| Статус файла | Фильтрует по текущему статусу файла для расследования аномальных файлов. |

Нажмите `Apply Filters`, чтобы применить фильтры. Используйте `Reset`, чтобы вернуться ко всем данным.

## Мобильное представление

На мобильных устройствах управление пользователями переключается на карточный макет.

![Мобильное управление пользователями](../../image/other/手机端显示用户管理效果.png)

Каждая карточка показывает IP, адрес, общий размер загрузок, количество загрузок и переключатель разрешения загрузки. Вы можете управлять пользователями без горизонтальной прокрутки таблицы.

## Если местоположение выглядит неверным

IP-геолокация приблизительна. Это не точный адрес.

Если пользователь находится за прокси, в дата-центре, на облачном сервере или в трансграничной сети, отображаемое местоположение может отличаться от реального.

Используйте эту функцию, чтобы понимать примерный источник, находить аномальные загрузки и принимать решения о блокировке. Не воспринимайте ее как точное отслеживание.

## Распространенные случаи

| Случай | Значение |
| --- | --- |
| Адрес пустой | IP мог еще не быть определен, или текущий источник временно недоступен. |
| Язык адреса неверный | Проверьте язык IP-геолокации и наличие настроенного источника, поддерживающего этот язык. |
| Адрес показывает дата-центр | Многие прокси, облачные серверы и сканеры отображаются как адреса дата-центров или провайдеров. |
| Количество загрузок высокое | Внимательно проверьте этот IP и при необходимости заблокируйте загрузки. |
| Общий размер загрузок большой | Отсортируйте или отфильтруйте данные, разверните IP и проверьте конкретные файлы. |
| Нужно восстановить после блокировки | Снова включите `Upload allowed`. |

## Краткий порядок действий

```text
Open IP Geolocation in Other Settings
-> Choose IP geolocation language
-> Fill MaxMind, Tencent Map, or ipapi credentials as needed
-> Save settings
-> Open User Management
-> Review IP source, address, total upload size, and upload count
-> Use search, sort, or advanced filters to find abnormal IPs
-> Allow or block uploads as needed
```
