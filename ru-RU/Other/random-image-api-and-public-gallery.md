# API случайного изображения и публичная галерея

Обе функции настраиваются в разделе:

```text
System Settings -> Other Settings
```

## API случайного изображения

API случайного изображения возвращает один случайный файл из выбранных каталогов. Это полезно для фонов сайта, ротации аватаров или вызовов случайного изображения с внешних страниц.

После включения используйте:

```text
https://your-domain.com/random
```

## Настройки API случайного изображения

| Параметр | Назначение |
| --- | --- |
| Включить | Включает или выключает конечную точку `/random`. Когда она выключена, доступ запрещен. |
| Каталоги | Ограничивает, какие каталоги может использовать API случайного изображения. Каталоги, не включенные сюда, не могут использоваться API. |
| Демонстрация вызова | Генерирует ссылки API случайного изображения, которые можно сразу скопировать. |

Можно выбрать несколько каталогов. Например, если разрешены только `/landscape/` и `/portrait/`, API случайного изображения сможет выбирать файлы только из этих каталогов и их подкаталогов.

## Параметры API случайного изображения

| Параметр | Пример | Назначение |
| --- | --- | --- |
| `dir` | `/landscape/` | Указывает случайный каталог. |
| `content` | `image` | Указывает тип медиа. Используйте `image`, `video`, `audio` или комбинации через запятую. |
| `orientation` | `auto` | Фильтрует ориентацию изображения. Используйте `portrait`, `landscape` или `auto`. |
| `type` | `url` | Формат возврата. Пустое значение означает перенаправление, `url` возвращает URL простым текстом, `json` возвращает JSON. |
| `origin` | `1` | Используется с `type=url`, чтобы вернуть полный URL. |
| `age` | `all-ages,r12` | Фильтрует по возрастному рейтингу. |
| `tag` | `wallpaper,sky` | Возвращает только файлы, содержащие эти теги. |
| `ex` | `private` | Исключает файлы, содержащие эти теги. |

## Форматы возврата

Без `type` API напрямую перенаправляет на URL случайного файла.

С `type=url` он возвращает текстовый URL.

С `type=json` он возвращает сведения о файле, включая URL файла, ID файла, имя файла, тип файла, теги, рейтинг и связанные метаданные.

## Правила доступа

API случайного изображения соблюдает правила публичного доступа:

| Правило | Эффект |
| --- | --- |
| Ограничение каталогов | Можно выбрать только файлы в разрешенных каталогах. |
| Список блокировки | Файлы из списка блокировки исключаются из случайного набора. |
| Режим списка разрешений | Когда включен, возвращаются только файлы, разрешенные для публичного доступа. |
| Возрастной рейтинг | Контент R12, R16, R18 и похожий контент фильтруется текущим режимом доступа. |

Если после фильтрации не найдено ни одного файла, API не возвращает подходящий результат.

## Кэш

API случайного изображения кэширует наборы кандидатов по каталогам, чтобы повысить скорость.

После изменения файлов ImgBed обновляет версию кэша каталога, а последующие запросы заново строят набор кандидатов. Пустые каталоги ненадолго кэшируются, чтобы избежать повторных запросов.

## Публичная галерея

Публичная галерея предоставляет открытую страницу просмотра только для чтения для каталогов, которые вы разрешаете видеть посетителям.

После включения посетители могут открыть:

```text
https://your-domain.com/browse/directory-name
```

## Настройки публичной галереи

| Параметр | Назначение |
| --- | --- |
| Включить | Включает или выключает публичную галерею. Когда она выключена, посетители не могут ее просматривать. |
| Режим загрузки изображений | Определяет, используют ли предпросмотры оригинальные изображения или миниатюры. |
| Открытые каталоги | Задает, к каким каталогам посетители могут получить доступ. |

## Режим загрузки изображений

| Режим | Назначение |
| --- | --- |
| Оригинал | Страница посетителя загружает исходные файлы напрямую. |
| Миниатюра | Страница посетителя в первую очередь загружает миниатюры, чтобы открываться быстрее. |

## Открытые каталоги

Открытые каталоги определяют, что могут видеть посетители.

Например:

```text
/1/,/2/,/landscape/,/portrait/
```

После этого посетители смогут открыть:

```text
https://your-domain.com/browse/1
https://your-domain.com/browse/2
https://your-domain.com/browse/landscape
https://your-domain.com/browse/portrait
```

Подкаталоги также можно открыть, например `/2026/lucky/`. Доступ посетителей к неоткрытым каталогам блокируется.

## Возможности публичной галереи

| Возможность | Описание |
| --- | --- |
| Просмотр каталогов | Просмотр файлов и подкаталогов в открытых каталогах. |
| Поиск | Поиск по имени файла, ID файла или тегам. |
| Фильтр типа | Фильтрует изображения, видео, аудио или другие файлы. |
| Фильтр тегов | Включает или исключает выбранные теги. |
| Фильтр ориентации | Фильтрует изображения в альбомной или портретной ориентации. |
| Фильтр времени | Фильтрует по диапазону времени загрузки. |
| Фильтр расширения | Фильтрует по расширению файла. |
| Копирование ссылки | Копирует ссылки доступа к файлу. |
| Предпросмотр медиа | Показывает или воспроизводит изображения, видео и аудио на странице посетителя. |

## Правила доступа публичной галереи

Публичная галерея также соблюдает правила публичного доступа:

| Правило | Эффект |
| --- | --- |
| Открытые каталоги | Показываются только разрешенные каталоги. |
| Режим доступа | Контент фильтруется текущим режимом доступа по возрастному рейтингу. |
| Режим списка разрешений | Когда включен, показываются только файлы, разрешенные для публичного доступа. |
| Список блокировки | Файлы из списка блокировки скрыты. |
