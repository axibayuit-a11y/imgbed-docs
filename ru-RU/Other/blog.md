# Блог

Функция блога добавляет на сайт ImgBed отдельную страницу блога.

После включения посетители могут открыть:

```text
https://your-domain.com/blog/
```

![Главная страница блога](../../image/other/博客/博客首页.png)

Блог адаптирован из проекта с открытым исходным кодом [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki). ImgBed переписывает и интегрирует его с Vue, чтобы он работал как часть сайта хостинга изображений.

## Где это настраивается

Настройки блога находятся здесь:

```text
System Settings -> Other Settings -> Blog
```

![Настройки блога](../../image/other/博客/QQ20260611-221702.png)

## Первичная настройка

1. Включите `Enable`.
2. Выберите учетную запись GitHub, где будет храниться конфигурация блога.
3. Нажмите `Update Blog`.
4. Дождитесь сообщения об успехе.
5. Откройте `https://your-domain.com/blog/`, чтобы посмотреть блог.

При первом использовании ImgBed подготовит приватный GitHub-репозиторий в выбранном аккаунте:

```text
imgbed-blog-config
```

В этом репозитории хранятся настройки блога и тексты статей.

## Написание постов

Редактируйте посты блога в своем приватном GitHub-репозитории:

```text
imgbed-blog-config
```

Обычный процесс:

1. Откройте GitHub.
2. Перейдите в приватный репозиторий `imgbed-blog-config`.
3. Отредактируйте или добавьте файлы постов.
4. Закоммитьте изменения.
5. Вернитесь в админ-панель ImgBed и нажмите `Update Blog`, либо трижды нажмите логотип в левом верхнем углу главной страницы блога, чтобы запустить обновление.

`Update Blog` не перезаписывает написанный вами контент. Он инициализирует репозиторий при необходимости и обновляет кеш блога.

## Поддерживаемые возможности

Блог поддерживает типичные функции: список постов, категории, теги, архив, поиск, темный режим и переключение языка.

Также поддерживаются комментарии и статистика посещений.

![Комментарии блога](../../image/other/博客/支持留言.png)

Комментарии показываются под постами. Посетители могут указать аватар, псевдоним, адрес электронной почты и текст комментария.

Статистика посещений показывает просмотры постов и посещения сайта, помогая понимать трафик блога.

## Адрес доступа

Блог всегда доступен по пути `/blog/`.

Например, если домен ImgBed:

```text
https://image.example.com
```

Адрес блога будет:

```text
https://image.example.com/blog/
```

После выключения блога посетители больше не смогут открыть страницу блога.
