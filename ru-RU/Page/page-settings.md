# Настройки страницы

Настройки страницы управляют отображением сайта, значениями по умолчанию на странице загрузки, фоновыми изображениями и внешним видом панели администратора.

## Общие настройки

| Параметр | Назначение |
| --- | --- |
| Заголовок сайта | Заголовок, отображаемый на вкладке браузера. |
| Значок сайта | Небольшой значок, отображаемый на вкладке браузера. |
| Название ImgBed | Название, отображаемое на страницах публичного интерфейса. |
| Логотип ImgBed | Изображение логотипа, отображаемое на страницах публичного интерфейса. |
| Ссылка логотипа | URL, открываемый при нажатии на логотип или аватар. |
| Интервал смены фона | Интервал ротации нескольких фоновых изображений в миллисекундах. `60000` означает 60 секунд. |
| Непрозрачность фона | Непрозрачность фонового изображения от `0` до `1`. Чем меньше значение, тем светлее изображение. |
| Префикс URL по умолчанию | Префикс, используемый при создании ссылок на изображения. Пустое значение означает использование текущего домена сайта. |

## Настройки клиента

| Параметр | Назначение |
| --- | --- |
| Объявление | Объявление, отображаемое в верхней части страницы загрузки. Поддерживается HTML. |
| Канал загрузки по умолчанию | Канал загрузки, выбранный по умолчанию на странице загрузки. Также можно выбрать Smart Dispatch. |
| Каталог загрузки по умолчанию | Каталог загрузки по умолчанию, например `/user/`. Пустое значение или `/` означает корневой каталог. |
| Способ именования по умолчанию | Стратегия создания имени файла после загрузки по умолчанию. См. ниже. |
| Преобразовывать в WebP по умолчанию | Преобразует изображения в WebP перед загрузкой. |
| Включать сжатие по умолчанию | Сжимает изображения локально в браузере перед загрузкой. |
| Порог сжатия по умолчанию | Автоматически сжимает изображение, когда его размер превышает это значение, в MB. |
| Целевой размер по умолчанию | Целевой размер файла после сжатия, в MB. |
| Фон страницы входа | Фоновое изображение для страницы входа пользователя. |
| Фон страницы загрузки | Фоновое изображение для страницы загрузки. |
| Ссылка портала в нижнем колонтитуле | URL, открываемый кнопкой портала в нижнем колонтитуле. |
| Скрыть нижний колонтитул | Скрывает нижний колонтитул публичного интерфейса, если включено. |

## Настройки администратора

| Параметр | Назначение |
| --- | --- |
| Фон входа администратора | Фоновое изображение для страницы входа администратора. |
| Фон администратора | Фоновое изображение для страниц администратора. Можно использовать один URL изображения или несколько URL. |
| Режим загрузки изображений | Режим загрузки предпросмотра в списке файлов администратора. Режим исходного изображения загружает оригинальные изображения. Умная загрузка предпочитает миниатюры для публичных изображений и оригиналы для изображений с ограниченным доступом. |
| Источник миниатюр | Сервис для создания миниатюр: wsrv.nl, Cloudflare Image Resizing или WordPress Photon. Cloudflare Image Resizing должен быть включен в Cloudflare перед выбором. |
| Виджет Live2D | Показывает персонажа Live2D в панели администратора. |
| Эффект фейерверка при клике | Показывает эффект фейерверка при нажатии на страницу. |
| Звездный след курсора | Показывает звездный след при движении мыши. |

## Форматы фоновых изображений

Фон страницы входа, фон страницы загрузки и фон входа администратора поддерживают следующие форматы:

| Значение | Эффект |
| --- | --- |
| `bing` | Использует ротацию обоев Bing. |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | Ротирует несколько изображений. |
| `["https://example.com/1.jpg"]` | Использует одно фоновое изображение. |
| `["https://your-domain.com/random?..."]` | Использует ссылку API случайного изображения. Вы можете настроить собственный API случайного изображения в других настройках, а затем вставить сюда созданную ссылку случайного изображения как запись одного фона. |

Фон администратора поддерживает URL изображений. Несколько URL можно разделять английскими запятыми, как указано на странице. Пустое значение означает использование фона по умолчанию.

## Способ именования по умолчанию

| Способ | Результат |
| --- | --- |
| По умолчанию | Случайный префикс на основе времени + исходное имя файла, например `1760000000000_cat.png`. |
| Только префикс | Только случайный префикс на основе времени и расширение, например `1760000000000.png`. |
| Только исходное имя | Сохраняет исходное имя файла, например `cat.png`. При совпадении имен ImgBed добавляет `(1)`, `(2)` и так далее. |
| Короткая ссылка | Использует короткий ID из 8 символов и расширение, например `a1b2c3d4.png`. |
