# Налаштування сторінки

Налаштування сторінки керують відображенням сайту, типовими значеннями сторінки завантаження, фоновими зображеннями та виглядом панелі адміністратора.

## Загальні налаштування

| Параметр | Призначення |
| --- | --- |
| Заголовок сайту | Заголовок, який відображається у вкладці браузера. |
| Піктограма сайту | Невелика піктограма, яка відображається у вкладці браузера. |
| Назва ImgBed | Назва, що відображається на сторінках публічного інтерфейсу. |
| Логотип ImgBed | Зображення логотипа, що відображається на сторінках публічного інтерфейсу. |
| Посилання логотипа | URL, який відкривається після натискання на логотип або аватар. |
| Інтервал зміни фону | Інтервал ротації кількох фонових зображень у мілісекундах. `60000` означає 60 секунд. |
| Непрозорість фону | Непрозорість фонового зображення від `0` до `1`. Менші значення роблять зображення світлішим. |
| Типовий префікс URL | Префікс, який використовується під час створення посилань на зображення. Порожнє значення означає використання поточного домену сайту. |

## Налаштування клієнта

| Параметр | Призначення |
| --- | --- |
| Оголошення | Оголошення, що відображається вгорі сторінки завантаження. HTML підтримується. |
| Типовий канал завантаження | Канал завантаження, вибраний типово на сторінці завантаження. Також можна вибрати Smart Dispatch. |
| Типовий каталог завантаження | Типовий каталог завантаження, наприклад `/user/`. Порожнє значення або `/` означає кореневий каталог. |
| Типовий спосіб іменування | Типова стратегія створення імені файлу після завантаження. Див. нижче. |
| Типово перетворювати на WebP | Перетворює зображення на WebP перед завантаженням. |
| Типово вмикати стиснення | Стискає зображення локально в браузері перед завантаженням. |
| Типовий поріг стиснення | Автоматично стискає зображення, коли його розмір перевищує це значення, у MB. |
| Типовий цільовий розмір | Цільовий розмір файлу після стиснення, у MB. |
| Фон сторінки входу | Фонове зображення для сторінки входу користувача. |
| Фон сторінки завантаження | Фонове зображення для сторінки завантаження. |
| Посилання порталу в нижньому колонтитулі | URL, який відкриває кнопка порталу в нижньому колонтитулі. |
| Приховати нижній колонтитул | Приховує нижній колонтитул публічного інтерфейсу, якщо ввімкнено. |

## Налаштування адміністратора

| Параметр | Призначення |
| --- | --- |
| Фон входу адміністратора | Фонове зображення для сторінки входу адміністратора. |
| Фон адміністратора | Фонове зображення для сторінок адміністратора. Можна використати один URL зображення або кілька URL. |
| Режим завантаження зображень | Режим завантаження попередніх переглядів у списку файлів адміністратора. Режим оригінального зображення завантажує оригінальні зображення. Розумне завантаження віддає перевагу мініатюрам для публічних зображень і оригіналам для зображень з обмеженим доступом. |
| Джерело мініатюр | Сервіс для створення мініатюр: wsrv.nl, Cloudflare Image Resizing або WordPress Photon. Cloudflare Image Resizing потрібно ввімкнути в Cloudflare перед вибором. |
| Віджет Live2D | Показує персонажа Live2D у панелі адміністратора. |
| Ефект феєрверка під час кліку | Показує ефект феєрверка під час натискання на сторінку. |
| Зоряний слід курсора | Показує зоряний слід під час руху миші. |

## Формати фонових зображень

Фон сторінки входу, фон сторінки завантаження та фон входу адміністратора підтримують такі формати:

| Значення | Ефект |
| --- | --- |
| `bing` | Використовує ротацію шпалер Bing. |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | Ротує кілька зображень. |
| `["https://example.com/1.jpg"]` | Використовує одне фонове зображення. |
| `["https://your-domain.com/random?..."]` | Використовує посилання API випадкового зображення. Ви можете налаштувати власний API випадкового зображення в інших налаштуваннях, а потім вставити сюди створене посилання випадкового зображення як запис одного фону. |

Фон адміністратора підтримує URL зображень. Кілька URL можна розділяти англійськими комами, як указано на сторінці. Порожнє значення означає використання типового фону.

## Типовий спосіб іменування

| Спосіб | Результат |
| --- | --- |
| Типово | Випадковий префікс на основі часу + початкове ім'я файлу, наприклад `1760000000000_cat.png`. |
| Лише префікс | Лише випадковий префікс на основі часу та розширення, наприклад `1760000000000.png`. |
| Лише початкове ім'я | Зберігає початкове ім'я файлу, наприклад `cat.png`. Якщо ім'я повторюється, ImgBed додає `(1)`, `(2)` тощо. |
| Коротке посилання | Використовує короткий ID із 8 символів і розширення, наприклад `a1b2c3d4.png`. |
