# ImgBed

Універсальний інструмент для особистого хостингу контенту на базі екосистеми Cloudflare.

ImgBed починався із завантаження зображень і керування посиланнями. Тепер він виріс у повноцінне рішення для особистого хостингу контенту. Ви можете використовувати його, щоб:

- Завантажувати файли в будь-який бажаний канал зберігання
- Публічно ділитися зображеннями та файлами будь-якого формату
- Швидко розгорнути особистий блог за одну хвилину
- Одним кліком зберігати файли з magnet-посилань
- Приєднатися до федерації вузлів ImgBed і спільно використовувати відкриті індекси файлів

Ми не віримо в рішення "один розмір для всіх". Якщо ви хочете просто зберігати зображення, масово керувати альбомами, писати кілька нотаток або створити легкий публічний ресурсний сайт, ImgBed поважає ваш вибір і залишає вам максимальну гнучкість.

## Потужні можливості одразу з коробки

- 🌍 Підтримка 28 мов інтерфейсу для користувачів у всьому світі
- 📦 13 вбудованих основних каналів завантаження: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex і pCloud
- 🧩 Повністю модульний дизайн: кожну функцію можна вмикати за потреби, не витрачаючи ресурси даремно
- 🔐 Підтримує API Token, режими доступу та правила списків для точного керування правами на контент

Проєкт і далі вдосконалюється. Наші можливості обмежені, але ми зробимо все можливе, щоб кожна функція була зрозумілою, стабільною та зручною. Якщо ImgBed допоможе вам легше мати власний хостинг зображень, блог або простір для обміну файлами, ми будемо дуже раді.

## Почати користування

Розгорніть ImgBed через [https://glow.ccwu.cc/](https://glow.ccwu.cc/) і швидко отримайте власний сервіс для хостингу зображень і файлів.
