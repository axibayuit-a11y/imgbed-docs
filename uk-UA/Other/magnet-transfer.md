# Передавання через magnet

Передавання через magnet завантажує файли з magnet-посилання та автоматично вивантажує їх у вибраний вами канал хмарного сховища.

Це корисно для перенесення серій аніме, відео, архівів і подібних файлів. Вставте magnet-посилання, і ImgBed створить фонове завдання завантаження. Після завершення завантаження файл буде вивантажено в ImgBed, а кінцеве посилання з'явиться у списку завантажень.

![Передавання через magnet](../../image/other/磁力链接/磁力链接.png)

## Де використовувати

Вхід для передавання через magnet розміщений в області завантаження на головній сторінці.

Вставте magnet-посилання в поле введення, виберіть `Transfer`, а потім завантажте.

![Завантаження аніме](../../image/other/磁力链接/上传番剧.png)

## Перед першим використанням

Спочатку налаштуйте передавання через magnet у панелі адміністратора.

Зазвичай потрібні:

1. Обліковий запис GitHub для запуску завдання завантаження.
2. Канал хмарного завантаження, наприклад Google Drive або OneDrive.
3. Цільовий каталог завантаження.
4. Тайм-аут завдання.

Коли налаштування готові, поверніться на головну сторінку та вставте magnet-посилання, щоб почати передавання.

## Завантаження magnet-посилання

1. Вставте magnet-посилання в поле завантаження на головній сторінці.
2. Переконайтеся, що режим установлено на `Transfer`.
3. Натисніть кнопку завантаження.
4. Дочекайтеся, поки ImgBed створить magnet-завдання.
5. Після запуску завдання використовуйте плаваючу панель `Magnet Tasks` у правому нижньому куті, щоб перевіряти прогрес.

Завантаження та вивантаження можуть зайняти час. Швидкість залежить від magnet-ресурсу, середовища виконання GitHub і вибраного каналу хмарного сховища.

![Magnet завантажується](../../image/other/磁力链接/磁力链接下载中.png)

## Після завершення

Після завершення завдання список завантажень показує назву файлу та посилання.

Для відео показується попередній перегляд відео, для зображень — попередній перегляд зображення, а для інших файлів — звичайна піктограма файлу.

![Завантажене відео](../../image/other/磁力链接/下载好后的视频.png)

Можна скопіювати:

| Тип посилання | Сценарій використання |
| --- | --- |
| Оригінальне посилання | Прямий доступ до файлу |
| Markdown | Дописи або нотатки Markdown |
| HTML | Код вебсторінки |
| BBCode | Форуми з підтримкою BBCode |

## Панель magnet-завдань

Панель magnet-завдань у правому нижньому куті показує кількість завдань, назву завдання, прогрес і кінцевий стан.

Поширені стани:

| Стан | Значення |
| --- | --- |
| Очікування | Завдання створено й очікує запуску. |
| Завантаження | Magnet-ресурс завантажується. |
| Вивантаження | Файл завантажено й він вивантажується у хмарне сховище. |
| Завершено | Вивантаження успішне, посилання можна скопіювати. |
| Помилка | Завдання не завершилося успішно. Перевірте повідомлення та спробуйте ще раз. |

## Поради

- Якщо magnet-посилання містить кілька файлів, ImgBed надає пріоритет головному завершеному файлу для показу.
- Великі файли потребують більше часу. Дочекайтеся завершення завдання, перш ніж оновлювати сторінку.
- Якщо magnet-ресурс не має доступних пірів, він може бути дуже повільним або завершитися помилкою.
- Якщо в хмарному обліковому записі закінчилася квота, авторизація минула або каталог завантаження неправильний, завдання може завершитися помилкою.
- Попередній перегляд відео може з'явитися через кілька секунд після завершення вивантаження.

## Поширені запитання

### Нічого не запускається після вставлення magnet-посилання

Переконайтеся, що передавання через magnet увімкнено в панелі адміністратора, а також вибрано придатний обліковий запис GitHub і хмарний канал.

### Завантаження завжди повільне

Швидкість magnet залежить від самого ресурсу. Якщо доступних пірів немає, завантаження може бути дуже повільним або неможливим.

### Після вивантаження немає попереднього перегляду

Спершу перевірте, чи відкривається посилання на файл. Відеофайлам може знадобитися трохи часу для завантаження в браузері, або ви можете відкрити посилання напряму.

### Що перевірити, якщо завдання завершилося помилкою?

Перевірте, чи magnet-посилання чинне, чи працює хмарний канал і чи правильний каталог завантаження. Потім надішліть завдання ще раз.
