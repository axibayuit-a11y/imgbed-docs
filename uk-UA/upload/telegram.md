# Додавання каналу Telegram

## Що підготувати заздалегідь

| Потрібно | Навіщо це потрібно |
| --- | --- |
| Обліковий запис Telegram | Для створення бота та каналу, у якому зберігатимуться файли. |
| `@BotFather` | Через нього створюється Telegram-бот. |
| Канал Telegram | Кінцеве місце зберігання файлів. |
| `@userinfobot` | Допомагає отримати `Chat ID` каналу. |

## Де додати канал

1. Відкрийте системні налаштування.
2. Перейдіть до налаштувань завантаження.
3. Натисніть Add Channel у правому верхньому куті.
4. Виберіть `Telegram`.

## Поля налаштування

| Поле | Що означає | Обов'язково |
| --- | --- | --- |
| Назва каналу | Зручна назва, наприклад "Telegram Primary". | Так |
| Активний | Увімкнення або вимкнення цього каналу. | Рекомендовано |
| Bot Token | Токен вашого Telegram-бота. | Так |
| Session ID (Chat ID) | Ідентифікатор Telegram-каналу. | Так |
| Relay Proxy URL (необов'язково) | Використовуйте лише тоді, коли доступ до Telegram нестабільний. Вкажіть повну адресу проксі, з `https://`. | Ні |
| Примітка | Нотатки для подальшого обслуговування. | Ні |

## Налаштування

### 1. Створіть Telegram-бота

1. Відкрийте Telegram і знайдіть `@BotFather`.
2. Відкрийте чат і натисніть `Start`.
3. Надішліть `/newbot`.
4. За підказками введіть відображувану назву бота.
5. За підказками введіть username бота. Зазвичай він має закінчуватися на `bot`.
6. Після створення бота `@BotFather` поверне токен.

Саме цей токен потрібно ввести в ImgBed у полі `Bot Token`.

![Збережіть токен бота](../../image/upload/telegram/保存机器人令牌.png)

### 2. Створіть канал

1. У Telegram натисніть New Channel.
2. Введіть назву каналу.
3. Завершіть створення каналу.

Можна використовувати і публічний, і приватний канал.

![Створення каналу](../../image/upload/telegram/新建频道.png)

### 3. Додайте бота до каналу

1. Відкрийте щойно створений канал.
2. Перейдіть до налаштувань каналу.
3. Додайте учасника або адміністратора.
4. Знайдіть username створеного бота.
5. Додайте бота до каналу.

Щоб завантаження працювали стабільно, краще надати боту права адміністратора.

![Запросіть бота до каналу](../../image/upload/telegram/邀请机器人进频道里.png)

### 4. Отримайте ID каналу через User Info - Get ID - IDbot

1. У Telegram знайдіть `@userinfobot`. Зазвичай він відображається як `User Info - Get ID - IDbot`.
2. Відкрийте чат і натисніть `Start`.
3. У варіантах бота виберіть `Channel`.
4. У вікні вибору повідомлення виберіть потрібний канал і надішліть його до `@userinfobot`.
5. Коли `@userinfobot` поверне результат, скопіюйте число в полі `Id: -100...`.

Число, що починається з `-100`, і є значенням `Session ID (Chat ID)`, яке потрібне ImgBed.

![Отримання ID каналу](../../image/upload/telegram/获取频道id.png)

### 5. Заповніть канал Telegram в ImgBed

Поверніться до діалогу налаштування каналу й заповніть поля так:

| Поле в інтерфейсі | Значення |
| --- | --- |
| Channel Identifier | Власна назва каналу, наприклад `TelegramPrimary`. |
| Active | Рекомендовано увімкнути. |
| Bot Token | Токен бота з `@BotFather`. |
| Session ID (Chat ID) | Число `-100...`, яке повернув `@userinfobot`. |
| Relay Proxy URL (необов'язково) | Лише за потреби, наприклад `https://your-tg-proxy.example.com`. |
| Remark | Необов'язкові нотатки. |

Коли все заповнено, натисніть Save.

![Редагування конфігурації](../../image/upload/telegram/编辑配置.png)

## Як перевірити

| Перевірка | Як перевірити |
| --- | --- |
| Картка каналу з'явилася | Після збереження на сторінці налаштувань завантаження має бути картка Telegram-каналу. |
| Канал можна ввімкнути | Перемикач Active має залишатися увімкненим. |
| Конфігурацію збережено | У деталях має бути видно, що Bot Token і Chat ID збережені. |
| Завантаження працює | Завантажте тестове зображення й переконайтеся, що воно з'явилося в цільовому Telegram-каналі. |

## Короткий чекліст

```text
Створити бота через @BotFather
-> Зберегти Bot Token
-> Створити Telegram-канал
-> Додати бота до каналу та надати права адміністратора
-> Знайти @userinfobot і вибрати Channel
-> Переслати будь-яке повідомлення з каналу до @userinfobot
-> Скопіювати повернений Id: -100...
-> Ввести Bot Token і Chat ID в ImgBed
-> Зберегти та завантажити тестове зображення
```
