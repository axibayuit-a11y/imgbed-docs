# Обмеження частоти завантажень для користувачів

Обмеження частоти завантажень для користувачів визначають, як часто звичайні користувачі або відвідувачі можуть завантажувати файли з головної сторінки. Це допомагає захистити публічну сторінку завантаження від зловживань.

Функція впливає лише на завантаження з головної сторінки. Завантаження адміністратора та завантаження через API Tokens не обмежуються цими правилами частоти.

## Де це налаштовується

Відкрийте адмін-панель і перейдіть до:

```text
System Settings -> Security Settings -> Upload Management -> User Rate Limits
```

![Налаштування обмеження частоти завантажень](../../image/other/用户频控截图.png)

## Увімкнення обмежень частоти

Після ввімкнення `Увімкнути обмеження частоти` ImgBed відстежує нещодавні завантаження за IP-адресою відправника.

Типові значення:

| Налаштування | Типово | Опис |
| --- | --- | --- |
| Вікно виявлення | 1.5 hours | Наскільки далеко назад рахуються записи завантажень. |
| Максимальна кількість файлів | 20 | Максимальна кількість файлів у вікні виявлення. |
| Обмеження розміру одного файлу | 20 MB | Максимальний розмір одного файлу. |
| Обмеження загального розміру завантажень | 200 MB | Максимальний загальний розмір завантажень у вікні виявлення. |

Наприклад, із вікном 1,5 години, 20 файлами, 20 MB на файл і 200 MB загалом завантаження з тієї самої IP-адреси блокуються, щойно перевищено будь-яке налаштоване обмеження.

## Виключення типів файлів

`Виключені типи файлів для завантаження` блокує звичайним користувачам або відвідувачам завантаження вибраних категорій файлів.

Доступні категорії:

| Тип | Опис |
| --- | --- |
| Зображення | jpg, png, webp, gif та подібні файли зображень |
| Відео | mp4, webm, mov та подібні відеофайли |
| Аудіо | mp3, flac, wav та подібні аудіофайли |
| Документи | pdf, txt, md, docx та подібні документи |
| Інше | Файли поза наведеними категоріями, наприклад zip, rar, exe, apk |

Типово категорія не вибрана, тобто вона дозволена.

Натискання на категорію підсвічує її, а це означає, що цей тип заблоковано.

Якщо вибрати `Інше`, відвідувачі, які завантажують zip або rar, будуть заблоковані й побачать повідомлення, що цей тип файлу не підтримується.

## Повідомлення про блокування

Коли спрацьовує обмеження, користувачі бачать відповідне повідомлення:

![Повідомлення про занадто часте завантаження](../../image/other/频繁报错提示.png)

| Сценарій | Значення повідомлення |
| --- | --- |
| Один файл завеликий | Файл завеликий, його варто стиснути перед завантаженням. |
| Тип файлу заблоковано | Цей тип файлу не підтримується. Приберіть його й спробуйте ще раз. |
| Завантаження занадто часті | Нещодавні завантаження занадто часті, показано час повторної спроби. |
| Загальний розмір завеликий | Сумарний розмір нещодавніх завантажень занадто великий, показано час повторної спроби. |

## Коли вмикати

Увімкніть обмеження частоти завантажень для користувачів, якщо ваша головна сторінка завантаження доступна публічно.

Типові причини:

- ви хвилюєтеся щодо масових автоматизованих завантажень;
- хочете обмежити великі завантаження від відвідувачів;
- хочете дозволити звичайним користувачам завантажувати лише зображення, а не архіви чи інсталятори;
- хочете залишити публічне завантаження доступним, але контролювати використання ресурсів.

Якщо сайт тільки для вас або завантажувати можуть лише адміністратори, це можна залишити вимкненим.
