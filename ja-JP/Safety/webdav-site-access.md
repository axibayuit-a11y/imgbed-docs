# WebDAV 設定とアクセス

ImgBed は WebDAV を通じてファイルへアクセスできます。PC のファイルマネージャーから接続すると、対応するファイルを通常のドライブのように扱えます。

## 使いどころ

- Windows や macOS からファイルを確認したい。
- WebDAV 対応クライアントでファイルを整理したい。
- 管理画面以外から保存済みファイルへアクセスしたい。

## Windows 11 で接続する

1. エクスプローラーを開きます。
2. 「PC」を右クリックします。
3. 「ネットワークの場所を追加」を選びます。
4. WebDAV の URL を入力します。
5. ユーザー名とパスワードを入力します。
6. 接続後、通常のフォルダのように開けることを確認します。

![Windows 11 WebDAV 設定](../../image/Safety/webdav在win11配置.png)

接続に成功すると、エクスプローラー上で WebDAV の内容を確認できます。

![Windows 表示](../../image/Safety/webdav在win显示效果.png)

## 認証情報

WebDAV のユーザー名とパスワードは、ImgBed 側で設定されている認証情報を使います。公開用のアカウントではなく、必要な範囲の権限だけを持つアカウントを使うと安全です。

## 接続できないとき

| 確認項目 | 内容 |
| --- | --- |
| URL | `https://` を含む正しい WebDAV URL か |
| 認証情報 | ユーザー名とパスワードが正しいか |
| 権限 | 対象ディレクトリへの読み書きが許可されているか |
| クライアント | Windows 側で WebDAV 接続が制限されていないか |

容量確認だけが失敗し、ファイル操作はできる場合もあります。その場合は WebDAV サーバー側が容量情報を返していない可能性があります。
