# 認証管理とログイン端末管理

認証管理では、管理画面やユーザー側ページへのログイン方法、ログイン済み端末、アクセス状態を確認できます。

## 管理画面ログイン

管理者は管理画面ログインから ImgBed の設定やファイル管理へ入ります。

![管理画面ログイン](../../image/Safety/管理端登录界面.png)

管理者アカウントは強い権限を持つため、パスワードや認証情報は安全に管理してください。

## ユーザー側ログイン

ユーザー向けのログイン画面も用意されています。

![ユーザー側ログイン](../../image/Safety/用户端登录界面.png)

公開アップロードや限定公開機能を使う場合、ユーザー側ログインを組み合わせることで、誰が操作したかを追いやすくなります。

## 認証管理画面

認証管理画面では、ログイン状態や端末情報を確認できます。

![認証管理](../../image/Safety/认证管理界面.png)

## 端末管理で見ること

| 項目 | 説明 |
| --- | --- |
| ログイン端末 | 現在ログインしている端末を確認します |
| IP / 位置 | どこからログインしたかの目安になります |
| 最終アクセス | 端末が最近使われているか確認できます |
| 解除 | 不要または不審な端末のログイン状態を無効化します |

## 不審なログインを見つけたら

1. 該当端末のログインを解除します。
2. 管理者パスワードや関連トークンを変更します。
3. Cloudflare、GitHub、保存先チャンネルなどの認証情報も確認します。
4. 必要に応じて API Token を再発行します。

## 運用のポイント

- 共有 PC ではログインしたままにしないでください。
- 管理者権限を持つアカウントは最小限にします。
- 定期的にログイン端末を見直します。
- 不要な端末や古いセッションは削除します。
