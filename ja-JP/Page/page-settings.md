# ページ設定

ページ設定では、サイト表示、アップロードページの既定値、背景画像、管理パネルの外観を制御します。

## 全般設定

| オプション | 目的 |
| --- | --- |
| サイトタイトル | ブラウザタブに表示されるタイトル。 |
| サイトアイコン | ブラウザタブに表示される小さなアイコン。 |
| ImgBed 名 | フロントエンドページに表示される名前。 |
| ImgBed ロゴ | フロントエンドページに表示されるロゴ画像。 |
| ロゴリンク | ロゴまたはアバターをクリックしたときに開く URL。 |
| 背景切り替え間隔 | 複数の背景をローテーションする間隔。単位はミリ秒です。`60000` は 60 秒を意味します。 |
| 背景の不透明度 | 背景画像の不透明度。範囲は `0` から `1` です。値が小さいほど薄く表示されます。 |
| 既定の URL プレフィックス | 画像リンクを生成するときに使用するプレフィックス。空の場合は現在のサイトドメインが使用されます。 |

## クライアント設定

| オプション | 目的 |
| --- | --- |
| お知らせ | アップロードページ上部に表示されるお知らせ。HTML を使用できます。 |
| 既定のアップロードチャネル | アップロードページで既定で選択されるアップロードチャネル。Smart Dispatch も選択できます。 |
| 既定のアップロードディレクトリ | 既定のアップロードディレクトリ。例: `/user/`。空または `/` はルートを意味します。 |
| 既定の命名方法 | アップロード後にファイル名を生成する既定の方式。下記を参照してください。 |
| 既定で WebP に変換 | アップロード前に画像を WebP に変換します。 |
| 既定で圧縮を有効化 | アップロード前にブラウザ内で画像をローカル圧縮します。 |
| 既定の圧縮しきい値 | 画像がこのサイズを超えた場合に自動で圧縮します。単位は MB です。 |
| 既定の目標サイズ | 圧縮後の目標ファイルサイズ。単位は MB です。 |
| ログインページの背景 | ユーザーログインページの背景画像。 |
| アップロードページの背景 | アップロードページの背景画像。 |
| フッターポータルリンク | フッターポータルボタンで開く URL。 |
| フッターを非表示 | 有効にするとフロントエンドのフッターを非表示にします。 |

## 管理設定

| オプション | 目的 |
| --- | --- |
| 管理ログイン背景 | 管理ログインページの背景画像。 |
| 管理背景 | 管理ページの背景画像。1 つの画像 URL または複数の URL を使用できます。 |
| 画像読み込みモード | 管理ファイル一覧でのプレビュー読み込みモード。元画像モードは元画像を読み込みます。スマート読み込みは公開画像ではサムネイルを優先し、制限付き画像では元画像を優先します。 |
| サムネイルソース | サムネイル生成に使用するサービス: wsrv.nl、Cloudflare Image Resizing、WordPress Photon。Cloudflare Image Resizing を選択する前に、Cloudflare で有効化しておく必要があります。 |
| Live2D ウィジェット | 管理パネルに Live2D キャラクターを表示します。 |
| クリック時の花火エフェクト | ページをクリックしたときに花火エフェクトを表示します。 |
| カーソルの星形トレイル | マウスを動かしたときに星形のトレイルを表示します。 |

## 背景画像の形式

ログインページの背景、アップロードページの背景、管理ログイン背景は、次の形式をサポートします。

| 値 | 効果 |
| --- | --- |
| `bing` | Bing 壁紙のローテーションを使用します。 |
| `["https://example.com/1.jpg","https://example.com/2.jpg"]` | 複数の画像をローテーションします。 |
| `["https://example.com/1.jpg"]` | 1 つの背景画像を使用します。 |
| `["https://your-domain.com/random?..."]` | ランダム画像 API のリンクを使用します。その他設定で独自のランダム画像 API を設定し、生成されたランダム画像リンクを単一背景の項目としてここに貼り付けることができます。 |

管理背景は画像 URL をサポートします。複数の URL は、ページ上の案内に従って英語のカンマで区切れます。空の場合は既定の背景が使用されます。

## 既定の命名方法

| 方法 | 結果 |
| --- | --- |
| 既定 | 時刻ベースのランダムプレフィックス + 元のファイル名。例: `1760000000000_cat.png`。 |
| プレフィックスのみ | 時刻ベースのランダムプレフィックスと拡張子のみ。例: `1760000000000.png`。 |
| 元の名前のみ | 元のファイル名を保持します。例: `cat.png`。重複する場合、ImgBed は `(1)`、`(2)` などを追加します。 |
| 短縮リンク | 8 文字の短縮 ID と拡張子を使用します。例: `a1b2c3d4.png`。 |
