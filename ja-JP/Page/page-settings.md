# ページ設定

ページ設定では、ImgBed の表示、公開ページ、アップロード体験に関わる基本項目を調整できます。

## 設定場所

```text
システム設定 -> ページ設定
```

## 主な設定

| 項目 | 説明 |
| --- | --- |
| サイト名 | ブラウザタイトルやページ上の表示名に使います |
| サイト説明 | サイトの説明文として表示します |
| ロゴ / アイコン | 画面やブラウザで使う画像を設定します |
| 背景画像 | トップページの背景に使います |
| フッター情報 | コピーライトや外部リンクなどを表示します |
| アップロード説明 | 利用者に見せるアップロード欄の補足文を設定します |

## 公開ページまわり

訪問者向けページを使う場合は、公開ディレクトリ、アクセスモード、画像読み込み方法と合わせて確認してください。

特に公開ギャラリーやランダム画像 API を使う場合、ページ設定だけでなく、その他設定側の公開範囲も一緒に見る必要があります。

## 変更後の確認

1. 設定を保存します。
2. トップページを再読み込みします。
3. PC とスマートフォン幅の両方で表示を確認します。
4. ロゴ、背景、説明文、フッターリンクが意図通り表示されるか確認します。

画像が反映されない場合は、画像 URL が開けるか、キャッシュが残っていないかを確認してください。
