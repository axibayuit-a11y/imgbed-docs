# 自動タグ付け

自動タグ付けは次の場所で設定します。

```text
システム設定 -> その他の設定 -> 自動タグ付け
```

画像にタグを自動で付け、検索、ランダム画像 API、公開ギャラリー、年齢レーティングによるアクセス制御で使いやすくします。

## できること

| 機能 | 説明 |
| --- | --- |
| コンテンツタグ | 人物、背景、物、画風などのタグを付けます |
| キャラクタータグ | アニメ画像やイラストのキャラクター判定に使えます |
| 向きタグ | `landscape`、`portrait`、`square` を補完します |
| 画像レーティング | `G/S/Q/E` の判定結果を保存します |
| アップロード時の自動処理 | 新規画像をアップロード後に自動で処理します |
| 一括処理 | 既存画像にまとめてタグを付けます |

## 事前準備

少なくとも 1 つ、アクセス可能な Hugging Face Space URL が必要です。

おすすめは SmilingWolf の `wd-tagger` Space を自分の Hugging Face アカウントへ複製する方法です。

```text
https://huggingface.co/spaces/SmilingWolf/wd-tagger
```

公開 Space をそのまま使うこともできますが、他の利用者と共有されるため、待ち時間が長くなったり一時的に使えなかったりします。長く使うなら、自分のアカウントに複製した Space の方が安定します。

## Space を複製する

1. Hugging Face にログインします。
2. `https://huggingface.co/spaces/SmilingWolf/wd-tagger` を開きます。

![SmilingWolf の公開 Space](../../image/other/微笑狼的公开仓库.png)

3. 右上のメニューから `Duplicate this Space` を選びます。
4. Space 名を確認します。必要なら `wd-tagger` など分かりやすい名前にします。
5. 公開範囲は `Public` のままにします。
6. まずは無料ハードウェアで作成します。
7. ビルド完了を待ちます。

作成後、ブラウザの URL を ImgBed の `Space URLs` に貼り付けます。

## 設定項目

| 項目 | おすすめ |
| --- | --- |
| `Space URLs` | 用意した Space URL を 1 行に 1 つ入力 |
| 対象フォルダ | 全体を処理するなら空欄、特定フォルダだけなら選択 |
| 認識モデル | まずは `wd-swinv2-tagger-v3` |
| 一般タグしきい値 | 迷ったら `0.35` 前後 |
| キャラクタータグしきい値 | 誤判定を抑えるなら `0.85` 前後 |
| `MCut` | 最初はオフで試す |
| アップロード時自動タグ付け | 新規画像にも使いたい場合だけオン |

タグが多すぎる場合は一般タグしきい値を少し上げます。少なすぎる場合は少し下げます。

## 一括タグ付け

1. `Space URLs` を入力します。
2. 対象フォルダを選びます。
3. タグ付けを開始します。
4. 進行状況が完了するまで待ちます。

対象フォルダを空にすると、全フォルダが処理対象になります。

## アップロード時の自動処理

有効にすると、新しくアップロードされた画像が自動でタグ付けキューに入ります。

Space が混雑していても、アップロード自体は先に完了し、タグ付けは後から進みます。

## よくある質問

### なぜ自分の Space を複製するのですか？

公開 Space は多くのユーザーが使います。自分用の Space なら待ち時間が短く、長期運用でも安定しやすくなります。

### タグが英語になるのは正常ですか？

正常です。SmilingWolf 系モデルは英語タグを返します。検索、フィルタ、ランダム画像 API で使うため、そのまま保存されます。

### レーティングは何に使いますか？

セキュリティ設定のアクセスモードと組み合わせ、訪問者に公開する画像を年齢レーティングで制御します。

## クイック手順

```text
Hugging Face にログイン
-> SmilingWolf/wd-tagger を開く
-> Duplicate this Space
-> ビルド完了を待つ
-> Space URL をコピー
-> ImgBed の Space URLs に貼り付け
-> モデルとしきい値を設定
-> 一括タグ付け、またはアップロード時自動タグ付けを有効化
```
