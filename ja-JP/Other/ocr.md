# OCR 文字認識

OCR 文字認識は、画像、PDF、Word などに含まれる文字を読み取り、検索や内容確認に使えるようにする機能です。

## 設定場所

```text
システム設定 -> その他の設定 -> OCR 文字認識
```

![OCR 設定](../../image/other/ip定位和ocr文字识别.png)

## できること

| 機能 | 説明 |
| --- | --- |
| 画像 OCR | スクリーンショットや写真内の文字を読み取ります |
| PDF OCR | PDF 内の文字を抽出します |
| Word OCR | Word ファイルの内容を読み取ります |
| 検索補助 | 読み取った文字をファイル検索に利用しやすくします |
| 一括処理 | 既存ファイルをまとめて認識できます |

![OCR 結果](../../image/other/ocr识别截图.png)

## 事前準備

利用する OCR サービスに応じて API キーやトークンを準備します。画面上の説明に沿って、必要なキーを入力してください。

PaddleOCR など外部サービスを使う場合は、認証情報が正しいこと、無料枠や利用制限に余裕があることを確認しておくと安心です。

![トークン取得](../../image/other/获取飞浆令牌.png)

## 使い方

1. OCR 機能を有効にします。
2. 利用する認識サービスのキーを入力します。
3. 保存します。
4. 対象ファイルをアップロード、または既存ファイルに対して OCR を実行します。
5. 認識結果をファイル詳細や検索で確認します。

![Word 認識結果](../../image/other/word识别结果.png)

![PDF 認識](../../image/other/pdf识别截图.png)

## 運用のポイント

- 画像が小さすぎる、ぼやけている、傾いている場合は認識精度が下がります。
- PDF や Word はファイルサイズが大きいと処理に時間がかかります。
- API キーの利用量制限に達すると、認識が失敗することがあります。
- OCR 結果は補助情報として扱い、重要な内容は元ファイルでも確認してください。
