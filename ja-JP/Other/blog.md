# ブログ機能の使い方

ブログ機能を有効にすると、ImgBed サイトに独立したブログページを追加できます。

```text
https://あなたのドメイン/blog/
```

![ブログトップ](../../image/other/博客/博客首页.png)

このブログ機能は [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki) を参考に、ImgBed 向けに Vue で組み込み直したものです。

## 設定場所

```text
システム設定 -> その他の設定 -> ブログ
```

![ブログ設定](../../image/other/博客/QQ20260611-221702.png)

## 初回設定

1. 「有効」をオンにします。
2. ブログ設定を保存する GitHub アカウントを選びます。
3. 「ブログを更新」をクリックします。
4. 更新完了のメッセージを待ちます。
5. `https://あなたのドメイン/blog/` を開いて確認します。

初回は、選択した GitHub アカウントに次のプライベートリポジトリが作成されます。

```text
imgbed-blog-config
```

このリポジトリにブログ設定と記事データが保存されます。

## 記事を書く場所

記事は GitHub の `imgbed-blog-config` リポジトリで編集します。

基本の流れは次のとおりです。

1. GitHub を開きます。
2. `imgbed-blog-config` に入ります。
3. 記事ファイルを追加または編集します。
4. 変更をコミットします。
5. ImgBed の管理画面で「ブログを更新」を押します。ブログトップ左上のロゴを 3 回クリックして更新することもできます。

「ブログを更新」は、すでに書いた記事を上書きする操作ではありません。初期化やキャッシュ更新のために使います。

## 利用できる機能

記事一覧、カテゴリ、タグ、アーカイブ、検索、ダークモード、多言語切り替えに対応しています。

コメントとアクセス統計も利用できます。

![ブログコメント](../../image/other/博客/支持留言.png)

コメントは記事ページの下に表示されます。訪問者はアバター、ニックネーム、メールアドレス、コメント内容を入力して投稿できます。

アクセス統計では記事閲覧数やサイト訪問数を確認できます。

## 公開 URL

ブログは常に `/blog/` 配下で公開されます。

```text
https://image.example.com/blog/
```

ブログを無効にすると、訪問者はブログページにアクセスできなくなります。
