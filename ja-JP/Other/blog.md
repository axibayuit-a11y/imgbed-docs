# ブログ

ブログ機能は、ImgBed サイトに独立したブログページを追加します。

有効にすると、訪問者は次の URL を開けます。

```text
https://your-domain.com/blog/
```

![ブログのホームページ](../../image/other/博客/博客首页.png)

このブログは、オープンソースの [LyraVoid/Mizuki](https://github.com/LyraVoid/Mizuki) プロジェクトをもとにしています。ImgBed はこれを Vue で書き直して統合し、画像ホスティングサイトの一部として動作できるようにしています。

## 設定場所

ブログ設定は次の場所にあります。

```text
System Settings -> Other Settings -> Blog
```

![ブログ設定](../../image/other/博客/QQ20260611-221702.png)

## 初回セットアップ

1. `Enable` をオンにします。
2. ブログ設定の保存に使用する GitHub アカウントを選択します。
3. `Update Blog` をクリックします。
4. 成功メッセージが表示されるまで待ちます。
5. `https://your-domain.com/blog/` を開いてブログを確認します。

初回利用時、ImgBed は選択したアカウントの下にプライベート GitHub リポジトリを用意します。

```text
imgbed-blog-config
```

このリポジトリには、ブログ設定と記事コンテンツが保存されます。

## 記事を書く

ブログ記事は、自分のプライベート GitHub リポジトリで編集します。

```text
imgbed-blog-config
```

一般的な流れ:

1. GitHub を開きます。
2. プライベートの `imgbed-blog-config` リポジトリに入ります。
3. 記事ファイルを編集または追加します。
4. 変更をコミットします。
5. ImgBed の管理パネルに戻って `Update Blog` をクリックします。または、ブログのホームページ左上にあるロゴを 3 回クリックしてブログ更新をトリガーします。

`Update Blog` は、すでに書いたコンテンツを上書きしません。必要に応じてリポジトリを初期化し、ブログキャッシュを更新します。

## 対応機能

ブログは、記事一覧、カテゴリ、タグ、アーカイブ、検索、ダークモード、言語切り替えなど、一般的なブログ機能に対応しています。

コメントと訪問統計にも対応しています。

![ブログコメント](../../image/other/博客/支持留言.png)

コメントは記事の下に表示されます。訪問者はアバター、ニックネーム、メールアドレス、コメント本文を送信できます。

訪問統計では、記事の閲覧数やサイト訪問数が表示され、ブログのトラフィックを把握するのに役立ちます。

## URL

ブログは常に `/blog/` 配下で配信されます。

たとえば、ImgBed のドメインが次の場合:

```text
https://image.example.com
```

ブログ URL は次のようになります。

```text
https://image.example.com/blog/
```

ブログを無効にすると、訪問者はブログページにアクセスできなくなります。
