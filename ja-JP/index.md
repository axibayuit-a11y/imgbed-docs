# ImgBed ユーザードキュメント

ImgBed は、Cloudflare エコシステムを基盤にした画像ホスティングとファイル共有のプロジェクトです。最初は画像アップロードとリンク管理から始まりましたが、現在はより幅広い個人向けコンテンツホスティングツールへと広がっています。複数のストレージにファイルをアップロードし、公開ギャラリーを作り、自分のブログを運営し、マグネットリンクからファイルを転送し、ほかの ImgBed ノードと連携して公開ファイル索引を共有できます。

私たちは、ImgBed を「画像をアップロードしてリンクを得るだけ」の道具にしたいわけではありません。誰でも自分の画像置き場、ブログ、ファイル共有スペースを持ちやすくすることが目標です。画像を保管したいだけの場合も、アルバムを整理したい場合も、少し文章を書きたい場合も、軽い公開リソースページを作りたい場合も、ImgBed は使い方を選べる余地を残します。

現在 ImgBed は 28 のローカライズ言語に対応し、Telegram、Cloudflare R2、S3、WebDAV、Discord、Hugging Face、GitHub Releases、GitLab Packages、OneDrive、Google Drive、Dropbox、Yandex、pCloud など多くのアップロードチャンネルを提供しています。地域、アカウント、普段の使い方に合わせて保存先を選べます。

このプロジェクトは今も改善を続けています。できることには限りがありますが、各機能をわかりやすく、安定して、使いやすいものにしていきます。ImgBed があなた自身の画像ホスト、ブログ、ファイル共有スペースづくりに役立つなら、とてもうれしく思います。
