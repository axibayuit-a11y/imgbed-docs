# ImgBed

Cloudflare エコシステムを基盤にした、オールインワンの個人向けコンテンツホスティングツールです。

ImgBed は画像アップロードとリンク管理から始まり、今では個人コンテンツを預けて共有するための総合的なソリューションへと成長しました。ImgBed では次のことができます。

- 好きな保存先チャネルへファイルをアップロード
- 画像や任意形式のファイルを公開共有
- 1 分で個人ブログをすばやく立ち上げ
- magnet リンクからワンクリックでファイルを保存
- ImgBed ノード連盟に参加し、公開ファイルインデックスを共有

私たちの出発点は、誰もが自分専用の画像ホスティングとファイル共有サイトを持てるようにすることです。どの国にいても、どの言語を使っていても、画像を保存したいだけの場合でも、アルバムを整理したい場合でも、友人にファイルを共有したい場合でも、ブログや創作コンテンツ、公開リソースのために安定したホスティング環境を用意したい場合でも、ImgBed はできるだけ導入のハードルを下げ、この体験をより自由で、わかりやすく、始めやすいものにしたいと考えています。

## すぐに使える強力な機能

- 🌍 28 種類のインターフェイス言語に対応し、世界中のユーザーが使いやすい
- 📦 13 種類の主要アップロードチャネルを内蔵：Telegram、Cloudflare R2、S3、WebDAV、Discord、Hugging Face、GitHub Releases、GitLab Packages、OneDrive、Google Drive、Dropbox、Yandex、pCloud
- 🧩 完全なモジュール設計で、必要な機能だけを有効化でき、リソースを無駄にしない
- 🔐 API Token、アクセスモード、リストルールに対応し、コンテンツ権限を細かく管理できます

このプロジェクトは今も継続的に改善されています。私たちの力には限りがありますが、すべての機能を分かりやすく、安定して、使いやすくするために全力を尽くします。ImgBed があなた自身の画像ホスト、ブログ、ファイル共有スペースをより簡単に持つ助けになれば、とても嬉しいです。
