# کافی اسپانسر کریں

السلام علیکم، میں Linjiang ہوں، ImgBed کا ڈویلپر۔  

شروع میں میں صرف اپنے بلاگ کے لیے ایک سادہ random image host بنانا چاہتا تھا۔ اتفاق سے مجھے open-source پروجیکٹ CloudFlare-ImgBed ملا۔ اس وقت اس کی خصوصیات کافی بنیادی تھیں، اس لیے میں نے چند PR بھیج کر اپنی ضرورت کی چیزیں شامل کرنا شروع کیں۔  
کام کرتے کرتے میرے ذہن میں ایک بڑا خیال آیا: کیا اسے ایسا مکمل personal content hosting tool بنایا جا سکتا ہے جسے ہر کوئی آسانی سے استعمال کر سکے؟  

میں نے کئی مہینوں تک اس پروجیکٹ کو تھوڑا تھوڑا کر کے دوبارہ بنایا اور ہر فیچر کو موجودہ شکل تک بہتر کیا۔ سچ کہوں تو میں کوئی پیشہ ور ڈویلپر نہیں، ابھی سیکھ رہا ہوں۔ Codex اور Claude جیسے AI ٹولز کی مدد کے بغیر میں اتنا پیچیدہ پروجیکٹ اسٹرکچر مکمل نہیں کر سکتا تھا۔  

اگر ImgBed واقعی آپ کے کام آیا ہے تو مجھے خوشی ہوگی اگر آپ اس کی ڈیولپمنٹ کو سپورٹ کرنے پر غور کریں۔ ایک کافی جتنی رقم بھی میرے لیے بہت معنی رکھتی ہے۔
