# IP جغرافیائی مقام اور صارف انتظام

IP جغرافیائی مقام اپ لوڈر ریکارڈز، لاگ ان ڈیوائسز اور اسی نوعیت کے لاگز میں موجود IP پتے تقریباً مقامات میں بدلتا ہے۔

ترتیب کے بعد ایڈمن پینل اپ لوڈ اور رسائی کے ذرائع زیادہ واضح طور پر دکھا سکتا ہے۔ صارف انتظام مشتبہ IP پتوں کے لیے اپ لوڈ رسائی کو بلاک یا بحال بھی کر سکتا ہے۔

## کہاں ترتیب دیں

کھولیں:

```text
System Settings -> Other Settings -> IP Geolocation
```

![IP جغرافیائی مقام](../../image/other/ip定位/ip定位.png)

## دستیاب ترتیبات

نیا IP جغرافیائی مقام عمل ایک نقشہ سروس پر انحصار کرنے کے بجائے کئی ذرائع کی حمایت کرتا ہے۔

| ترتیب | مقصد |
| --- | --- |
| IP جغرافیائی مقام کی زبان | دکھائی جانے والی زبان منتخب کرتی ہے، جیسے English، Simplified Chinese، Japanese، French وغیرہ۔ |
| MaxMind Account ID | MaxMind GeoLite Web Service کے لیے MaxMind اکاؤنٹ ID۔ |
| MaxMind License Key | MaxMind License Key۔ |
| Tencent Map Key | Tencent Location Service کی کلید۔ چینی پتوں اور مین لینڈ چین IPs کے لیے مفید ہے۔ |
| ipapi Key | APILayer ipapi کی کلید۔ کثیر لسانی IP جغرافیائی مقام کی حمایت کرتا ہے۔ |

صرف وہی خدمات بھریں جن کی ضرورت ہے۔ ہر فیلڈ ترتیب دینا ضروری نہیں۔

اگر کوئی کلید نہ دی جائے تو ImgBed پھر بھی اندرونی مفت ذرائع آزمانے کی کوشش کرتا ہے، مگر استحکام، زبان کی حمایت اور درستگی آپ کی اپنی ترتیب دی گئی خدمت سے کم ہو سکتی ہے۔

## تجویز کردہ انتخاب

اگر آپ کو بنیادی طور پر چینی پتے درکار ہیں:

1. IP جغرافیائی مقام کی زبان کو Simplified Chinese پر سیٹ کریں۔
2. Tencent Map Key ترتیب دیں۔
3. اختیاری طور پر MaxMind یا ipapi کو متبادل ذرائع کے طور پر شامل کریں۔

اگر آپ کو بنیادی طور پر انگریزی یا کثیر لسانی پتے درکار ہیں:

1. مطلوبہ زبان منتخب کریں۔
2. MaxMind Account ID اور License Key ترتیب دیں۔
3. بہتر کثیر لسانی نتائج کے لیے ipapi Key شامل کریں۔

## MaxMind ترتیب

MaxMind کو یہ درکار ہے:

```text
MaxMind Account ID
MaxMind License Key
```

MaxMind ڈیش بورڈ میں اکاؤنٹ ID تلاش کریں اور License Keys صفحے سے License Key بنائیں۔

![MaxMind کلید کی ترتیب](../../image/other/ip定位/maxmind的key配置.png)

بنانے کے بعد Account ID اور License Key کو ImgBed میں چسپاں کر کے محفوظ کریں۔

MaxMind کا مفت منصوبہ روزمرہ استعمال کے لیے مناسب ہے، مگر اس میں درخواست کی حدیں ہیں۔ کوٹا ختم ہونے پر ImgBed دیگر دستیاب ذرائع آزمانا جاری رکھتا ہے۔

## ipapi ترتیب

ipapi APILayer API Key استعمال کرتا ہے۔

ipapi کنسول کھولیں اور وہاں دکھائی گئی API Key کاپی کریں۔

![ipapi ترتیب](../../image/other/ip定位/ipapi配置.png)

اسے ImgBed کے `ipapi Key` فیلڈ میں چسپاں کر کے محفوظ کریں۔

ipapi کثیر لسانی IP جغرافیائی مقام کی حمایت کرتا ہے اور اس وقت مفید ہے جب آپ پتوں کو منتخب زبان میں دکھانا چاہتے ہیں۔ اس کے مفت منصوبے میں بھی درخواست کی حدیں ہیں۔ کوٹا ختم ہونے پر ImgBed دیگر دستیاب ذرائع آزمانا جاری رکھتا ہے۔

## Tencent Map Key ترتیب

Tencent Map Key چینی پتوں، خاص طور پر مین لینڈ چین IPs، کے لیے مفید ہے۔

Tencent Location Service میں کلید بناتے وقت فعال کریں:

```text
WebServiceAPI
```

بنانے کے بعد کلید کو `Tencent Map Key` میں چسپاں کر کے محفوظ کریں۔

اگر آپ کو صرف بنیادی چینی IP جغرافیائی مقام چاہیے تو Tencent Map Key شروع کرنے کے لیے کافی ہے۔

## صارف انتظام میں کیا دیکھیں

صارف انتظام ایڈمن پینل کے اوپر سے دستیاب ہے۔

![صارف انتظام](../../image/other/用户管理显示.png)

صارف انتظام IP کے لحاظ سے اپ لوڈ سرگرمی دکھاتا ہے:

| فیلڈ | وضاحت |
| --- | --- |
| IP ذریعہ | اپ لوڈر کا ماخذ IP۔ |
| پتہ | IP سے معلوم کیا گیا تقریباً مقام۔ |
| کل اپ لوڈ حجم | اس IP سے اپ لوڈ کی گئی فائلوں کا کل حجم۔ |
| اپ لوڈ تعداد | اس IP سے اپ لوڈز کی تعداد۔ |
| اپ لوڈ کی اجازت | فعال ہو تو اپ لوڈز کی اجازت ہوتی ہے۔ غیر فعال ہو تو اپ لوڈز بلاک ہوتے ہیں۔ |

اس IP سے اپ لوڈ کی گئی فائلوں کی فہرست پھیلانے کے لیے بائیں تیر پر کلک کریں۔

فائل فہرست فائل نام، پیش منظر، فائل حجم، جانچ نتیجہ، فائل حالت اور اپ لوڈ وقت دکھاتی ہے۔ اگر اپ لوڈز مشتبہ لگیں تو پہلے IP پھیلائیں، فائلیں دیکھیں، پھر فیصلہ کریں کہ مزید اپ لوڈز بلاک کرنے ہیں یا نہیں۔

اگر کوئی IP مشتبہ ہو تو `Upload allowed` بند کریں۔ اس IP سے آئندہ اپ لوڈز بلاک ہو جائیں گے۔

## تلاش، ترتیب اور اعلی فلٹرز

صارف انتظام کے اوپر IP ذریعہ یا پتے سے تلاش کریں۔

حالیہ اپ لوڈرز، زیادہ بار اپ لوڈ کرنے والے IPs، یا زیادہ استعمال والے IPs تلاش کرنے کے لیے وقت، اپ لوڈ تعداد یا کل اپ لوڈ حجم کے مطابق ترتیب دیں۔

مزید گہری جانچ کے لیے اعلی فلٹرز کھولیں۔

![اعلی فلٹرز](../../image/other/用户管理高级筛选.png)

اعلی فلٹرز یہ حمایت کرتے ہیں:

| فلٹر | استعمال |
| --- | --- |
| وقت کی حد | منتخب مدت میں فائلیں اپ لوڈ کرنے والے IPs دکھاتا ہے۔ |
| رسائی حالت | عام، بلاک شدہ اور اسی نوعیت کی حالتوں کے مطابق فلٹر کرتا ہے۔ |
| اجازت/بلاک فہرست | اجازت فہرست، بلاک فہرست یا غیر مقرر حالت کے مطابق فلٹر کرتا ہے۔ |
| فائل قسم | تصاویر، ویڈیوز، آڈیو، دستاویزات، کوڈ یا دوسری فائلیں اپ لوڈ کرنے والے IPs دکھاتا ہے۔ |
| فائل حجم | اپ لوڈ فائل حجم کی حد کے مطابق فلٹر کرتا ہے۔ |
| عمر درجہ بندی | غیر مقرر، General، R12+، R16+، R18 اور اسی نوعیت کی درجہ بندی کے مطابق فلٹر کرتا ہے۔ |
| فائل حالت | غیر معمولی فائلوں کی جانچ کے لیے موجودہ فائل حالت کے مطابق فلٹر کرتا ہے۔ |

لاگو کرنے کے لیے `Apply Filters` پر کلک کریں۔ تمام ڈیٹا پر واپس آنے کے لیے `Reset` استعمال کریں۔

## موبائل منظر

موبائل پر صارف انتظام کارڈ لے آؤٹ میں بدل جاتا ہے۔

![موبائل صارف انتظام](../../image/other/手机端显示用户管理效果.png)

ہر کارڈ IP، پتہ، کل اپ لوڈ حجم، اپ لوڈ تعداد اور اپ لوڈ اجازت سوئچ دکھاتا ہے۔ آپ افقی جدول اسکرول کیے بغیر صارفین کا انتظام کر سکتے ہیں۔

## اگر مقام غلط لگے

IP جغرافیائی مقام تقریباً ہوتا ہے۔ یہ درست گلی کا پتہ نہیں۔

اگر صارف پراکسی، ڈیٹا سینٹر، کلاؤڈ سرور یا سرحد پار نیٹ ورک کے پیچھے ہو تو دکھایا گیا مقام اصل مقام سے مختلف ہو سکتا ہے۔

اس خصوصیت کو عمومی ماخذ سمجھنے، غیر معمولی اپ لوڈز تلاش کرنے اور بلاک فیصلوں میں مدد کے لیے استعمال کریں۔ اسے درست ٹریکنگ نہ سمجھیں۔

## عام صورتیں

| صورت | مطلب |
| --- | --- |
| پتہ خالی ہے | IP ابھی حل نہ ہوا ہو، یا موجودہ ذریعہ عارضی طور پر دستیاب نہ ہو۔ |
| پتے کی زبان غلط ہے | IP جغرافیائی مقام کی زبان اور اس زبان کی حمایت کرنے والے ذریعہ کی ترتیب چیک کریں۔ |
| پتہ ڈیٹا سینٹر دکھاتا ہے | بہت سی پراکسیز، کلاؤڈ سرورز اور کرالرز ڈیٹا سینٹر یا ISP پتوں کے طور پر ظاہر ہوتے ہیں۔ |
| اپ لوڈ تعداد زیادہ ہے | اس IP کا احتیاط سے جائزہ لیں اور ضرورت ہو تو اپ لوڈز بلاک کریں۔ |
| کل اپ لوڈ حجم بڑا ہے | ترتیب دیں یا فلٹر کریں، IP پھیلائیں اور مخصوص فائلیں دیکھیں۔ |
| بلاک کے بعد بحال کرنا ہے | `Upload allowed` دوبارہ فعال کریں۔ |

## فوری عمل

```text
Open IP Geolocation in Other Settings
-> Choose IP geolocation language
-> Fill MaxMind, Tencent Map, or ipapi credentials as needed
-> Save settings
-> Open User Management
-> Review IP source, address, total upload size, and upload count
-> Use search, sort, or advanced filters to find abnormal IPs
-> Allow or block uploads as needed
```
