# Invitar un café

Hola, soy Linjiang, el desarrollador de ImgBed.  

Al principio solo quería crear un host de imágenes aleatorias sencillo para mi propio blog. Por casualidad encontré el proyecto open source CloudFlare-ImgBed. Sus funciones todavía eran bastante básicas, así que empecé enviando algunos PR para completar lo que faltaba.  
Mientras lo iba modificando, me surgió una idea: ¿y si pudiera convertirse en una herramienta completa de alojamiento de contenido personal que cualquiera pudiera usar con facilidad?  

He pasado varios meses reconstruyendo el proyecto poco a poco y puliendo cada función hasta llegar a lo que es hoy. Para ser sincero, no soy un desarrollador profesional; sigo aprendiendo. Sin herramientas de IA como Codex y Claude, no habría podido completar una arquitectura tan compleja.  

Si ImgBed realmente te ha ayudado, me encantaría que consideraras apoyar su desarrollo. Incluso el precio de un café significa muchísimo para mí.  

Beneficios de apoyo  
✅ Ocultar para siempre este aviso de apoyo  
✅ Probar antes las próximas funciones
