# Configuración de página

La configuración de página controla elementos básicos de presentación, páginas públicas y experiencia de subida en ImgBed.

## Dónde se configura

```text
Configuración del sistema -> Configuración de página
```

## Ajustes principales

| Ajuste | Descripción |
| --- | --- |
| Nombre del sitio | Se usa en el título del navegador y en la interfaz |
| Descripción del sitio | Texto descriptivo del sitio |
| Logotipo / icono | Imágenes usadas en la página y el navegador |
| Imagen de fondo | Fondo de la página principal |
| Pie de página | Copyright, enlaces externos u otra información |
| Texto de subida | Mensaje mostrado en el área de subida |

## Páginas públicas

Si usas páginas para visitantes, revisa también los directorios públicos, el modo de acceso y el modo de carga de imágenes.

Especialmente con la galería pública o la API de imagen aleatoria, la configuración de página debe ir acompañada de los permisos de publicación en Otros ajustes.

## Después de guardar

1. Guarda los cambios.
2. Recarga la página principal.
3. Comprueba la vista en escritorio y en móvil.
4. Verifica que logotipo, fondo, descripción y enlaces del pie se muestren como esperas.

Si una imagen no se actualiza, comprueba que su URL se pueda abrir y que el navegador no esté usando caché antigua.
