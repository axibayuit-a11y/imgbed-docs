# ImgBed

Alat serbaguna untuk hosting konten pribadi, dibangun di atas ekosistem Cloudflare.

ImgBed bermula dari unggahan gambar dan pengelolaan tautan. Kini ImgBed telah berkembang menjadi solusi lengkap untuk hosting konten pribadi. Anda bisa menggunakannya untuk:

- Mengunggah file ke kanal penyimpanan mana pun yang Anda sukai
- Membagikan gambar dan file dalam format apa pun secara publik
- Membangun blog pribadi dengan cepat dalam satu menit
- Menyimpan file dari link magnet dengan sekali klik
- Bergabung dengan federasi node ImgBed dan berbagi indeks file terbuka

Tujuan awal kami adalah membantu setiap orang memiliki situs hosting gambar dan berbagi file miliknya sendiri. Dari negara mana pun Anda berasal dan bahasa apa pun yang Anda gunakan, baik Anda hanya ingin menyimpan gambar, menata album, berbagi file dengan teman, maupun membangun ruang hosting yang stabil untuk blog, karya kreatif, atau sumber daya publik, ImgBed berusaha menurunkan hambatan semaksimal mungkin agar pengalaman ini lebih bebas, lebih jelas, dan lebih mudah dimulai.

## Kemampuan kuat yang siap dipakai

- 🌍 Mendukung 28 bahasa antarmuka agar pengguna global bisa memakainya dengan nyaman
- 📦 13 kanal unggahan utama bawaan: Telegram, Cloudflare R2, S3, WebDAV, Discord, Hugging Face, GitHub Releases, GitLab Packages, OneDrive, Google Drive, Dropbox, Yandex, pCloud
- 🧩 Desain modular murni, semua fitur bisa dinyalakan sesuai kebutuhan tanpa membuang sumber daya
- 🔐 Mendukung API Token, mode akses, dan aturan daftar untuk mengelola izin konten secara lebih rinci

Proyek ini masih terus disempurnakan. Kemampuan kami terbatas, tetapi kami akan berusaha semaksimal mungkin agar setiap fitur jelas, stabil, dan enak digunakan. Jika ImgBed membantu Anda lebih mudah memiliki tempat hosting gambar, blog, atau ruang berbagi file sendiri, kami akan sangat senang.
